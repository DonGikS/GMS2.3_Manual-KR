파일 메뉴

파일 메뉴에서 프로젝트를 변경, 저장 또는 가져 오거나 GameMaker Studio 2 IDE 환경 설정을 선택할 수 있습니다. 사용 가능한 옵션은 다음과 같습니다.

  • 새 프로젝트 -GameMaker Studio 2에서 처음부터 새 프로젝트를 만들기위한 것입니다. "저장"대화창이 열리고 새 프로젝트를 저장할 위치를 선택하라는 메시지가 나타납니다. 프로젝트 이름과 저장 위치를 ​​지정하면 비어있는 새 작업 공간으로 이동 하여 작업 을 시작할 수 있습니다. 프로젝트 파일은 * .yyp 파일 확장자로 저장됩니다.
  • 프로젝트 열기 -파일 탐색기를 열어로드 할 이전에 생성 된 * .yyz 프로젝트 파일을 선택할 수 있습니다. 가져 오기시 프로젝트가 2.3.0보다 낮은 버전을 사용하여 작성된 경우 다음 경고 메시지가 표시됩니다. 진행하기로 선택하면 파일 탐색기가 열리고 프로젝트에 파일 이름과 위치를 업데이트하여 저장할 수 있으므로 원본 프로젝트를 덮어 쓰지 않습니다. 프로젝트가 새로운 2.3.0+ 형식으로 변환되면 IDE에서 열리고 변환에 성공했다는 메시지가 표시됩니다. 변환 프로세스에서 생성 된 호환성 보고서를 확인하고 프로젝트를 실행하거나 컴파일하기 전에 수정해야 할 사항이 있는지 확인해야합니다. 2.3.0 이전 버전으로 작성된 프로젝트를 열려고 시도하는 동안 오류가 반복되면 YoYo Games ( 도움말 메뉴의 "버그보고"옵션 사용)에 연락하여 원래 * .yyp을 제공해야합니다. 변환하려는 파일입니다.
  • 프로젝트 가져 오기 -이전에 * .yyz 아카이브로 내 보낸 프로젝트를 가져 오거나 (아래 "내보내기 프로젝트"참조) 또는-YoYo Games의 GameMaker : Studio 1.4 소프트웨어로 만든 프로젝트가있는 경우 다음을 수행 할 수 있습니다. 해당 제품 라인에서 * .gmx 또는 * .gmz 파일을 가져 오려면이 옵션을 선택하십시오. 프로젝트를 열 때와 같이 * .yyz 가 2.3.0 업데이트 이전의 GameMaker Studio 2 버전으로 생성 된 경우 해당 업데이트로 프로젝트 파일 형식이 변경됨에 따라 변환해야합니다.

참고 : 이전 GameMaker 프로젝트를 GameMaker Studio 2로 가져 오려면 1.4 버전을 사용하여 생성해야하며 가져 오기에서 직접 작동하지 않을 수 있습니다. GameMaker Studio 2와 GameMaker : Studio 1.4는 많은 부분이 다르기 때문에 이전 프로젝트는 약간의 조정이 필요할 수 있습니다. YoYo Games 기술 자료에서 GM : S 1.4 Game To GameMaker Studio 2 포팅 기사와 더 이상 사용되지 않는 기능 및 호환성 스크립트 를 다루는 설명서 섹션을 참조하십시오.

중대한! 글꼴 리소스가 포함 된 * .yyz 프로젝트를 가져 오려면 글꼴을 제거 / 다시 추가하거나 글꼴 편집기에서 글꼴을 변경 한 다음 다시 설정하여 글꼴을 "재생성"해야 할 수 있습니다. 이는 기존 글꼴 정보가 * .yyz 가 내 보내질 때 유지되고 나중에 또는 다른 시스템에서 가져올 때 프로젝트에 필요한 글꼴이 설치되어 있지 않거나 동일한 이름의 다른 글꼴이있을 수 있기 때문입니다. ). 가져 오기 후 글꼴이 렌더링되는 방식에 문제가있는 경우에만 수행해야합니다.

  • 최근 프로젝트 -이 옵션을 사용하여 최근에 사용한 프로젝트를 볼 수 있습니다. 선택하면 시작 페이지 가 다시 열리고 (닫은 경우) 최근에 사용한 프로젝트를 목록에서 찾을 수 있습니다.
  • 프로젝트 저장 - 프로젝트 를 만들 때 (또는 프로젝트를 열 때) 정의한 것과 동일한 위치에 게임을 저장합니다.
  • 다른 이름으로 프로젝트 저장 -위와 동일하며 프로젝트 의 위치와 새 이름 만 지정하십시오.
  • 프로젝트 내보내기 -프로젝트를 단일 * .yyz 파일로 내 보냅니다. 사람에게 사본을 보내거나 백업을 저장하는 데 매우 유용합니다.
  • 환경 설정 -이것을 선택하면 GameMaker Studio 2의 모양과 느낌을 설정할 수있는 환경 설정 창으로 이동합니다.
  • 새 IDE- 이 옵션을 선택하면 사용할 수있는 새 IDE 인스턴스가 열립니다 (시작 페이지에서 열립니다).
  • 내 계정 -이 옵션은 기본 웹 브라우저를 열고 YoYo 계정 페이지로 이동합니다.
  • 로그 아웃 -이 옵션을 선택하면 GameMaker Studio 2에서 로그 아웃됩니다. 나중에 제품을 사용할 수 있으려면 동일한 라이센스 (또는 다른 라이센스)로 다시 로그인해야합니다.

중대한! 당신은 GameMaker 스튜디오 2의 평가판 라이센스를 사용하고 다른 라이센스 중 하나 (데스크톱, 모바일, UWP, 웹 또는 콘솔)를 구입 후, 당신은 IDE에서 로그 아웃해야합니다 다음 GameMaker 스튜디오이 다시 시작되었을 경우 다시 로그인 새 라이센스가 올바르게 적용되는지 다시 확인하십시오 (자동으로 발생하더라도).

  • 종료 -GameMaker Studio 2를 종료합니다.
