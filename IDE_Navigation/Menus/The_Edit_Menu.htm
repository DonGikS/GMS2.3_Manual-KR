편집 메뉴

편집 메뉴는 GameMaker Studio 2 인터페이스의 일반 판 도구를 사용할 수있는 곳입니다. 사용 가능한 옵션은 다음과 같습니다.

  • 실행 취소 -작업을 실행 취소합니다.
  • 다시 실행 -이전 작업을 다시 실행하십시오.
  • 검색 및 교체 -검색 창을 열어 주어진 입력에 대해 전체 코드 (방 생성 코드, 스크립트 및 이벤트 포함)를 검색 할 수 있습니다. 자세한 내용은 아래를 참조하십시오.

편집 메뉴에서 검색 및 바꾸기를 선택하면 다음 창이 열립니다.

이를 사용하여 룸 코드, 이벤트 코드, 스크립트 및 셰이더를 포함하여 게임의 모든 스크립트 된 자산을 검색 할 수 있습니다. 검색 할 때 상단의 상자에 새 검색어를 입력하거나 오른쪽에있는 화살표 아이콘을 클릭하여 이전에 검색 한 용어 목록을 열 수 있습니다. 초기 검색어를 대체하는 데 사용할 대체 용어를 제공하는 옵션도 있으며 오른쪽의 화살표 단추에서 이전에 사용한 대체 문자열을 사용하는 옵션도 있습니다.

검색은 일반적이거나 검색어 입력 아래의 옵션을 사용하여 세분화 할 수 있습니다.

  • 검색 대소 문자를 구분 하도록 선택할 수 있습니다.이 경우 검색 문자열과 정확히 일치하는 단어 만 포함됩니다 (예 :이 옵션없이 "My_Var"를 검색하면 "my_var", "mY_VAR", 또는 "My_Var"이지만이 옵션을 선택하면 "My_Var"만 검색 결과에 포함됩니다.
  • 전체 단어 만 포함하는 검색 결과 만 포함하도록 선택할 수 있습니다.이 경우 검색어를 포함하지만 정확히 일치하지 않는 결과는 무시됩니다. 즉,이 항목을 선택하지 않은 "게임"에 대한 검색이 표시됩니다. "game_end ()"또는 "game_state"와 같은 변수와 같은 것을 확인하고 검색하는 동안 "game"자체의 결과 만 표시됩니다.
  • 검색시 주석을 무시 하도록 선택할 수 있습니다. 이를 선택하면 검색어가 포함 된 코드의 주석이 무시되고 출력 결과에 표시되지 않습니다.
  • 자산 브라우저의 다양한 자산 유형을 기반으로 검색 결과에 필터 를 적용 할 수 있습니다. 기본적으로이 항목은 "Everything"으로 설정되어 있지만이 항목을 클릭하면 검색에 포함 할 자산을 확인하거나 선택 취소 할 수있는 다음 메뉴가 열립니다.

검색 (및 선택적으로 대체) 용어를 제공하고 검색 옵션을 설정 한 후 하단의 다른 단추를 사용하여 검색 수행 방법 및 수행 할 조치를 선택할 수 있습니다. 이 버튼들은 :

  • 모두 찾기 : 모든 검색어를 찾아서 출력 창에 표시합니다.
  • 이전 찾기 : 검색어의 이전 인스턴스를 찾습니다. 작업 공간에서 참조가 포함 된 자산을 열고 (아직 열려 있지 않은 경우) 이동합니다.
  • 다음 찾기 : 검색어의 다음 인스턴스를 찾습니다. 작업 공간에서 참조가 포함 된 자산을 열고 (아직 열려 있지 않은 경우) 이동합니다.
  • 모두 바꾸기 : 옵션 세트를 사용하여 지정된 용어를 검색하고 자산에서 지정한 새 바꾸기 용어로 대체합니다.
  • 바꾸기 : 현재 편집기에서 현재 선택된 용어를 선택한 대체 용어로 강조 표시합니다.
  • 검색 중지 : 현재 검색이 진행중인 경우 중지합니다.

All results from using the Search / Replace window will be shown in the Search Output Window at the bottom of the IDE. Each result can be 두 번 클릭하여 현재 작업 공간에서 용어가있는 자산을 열고 이동하여 편집 할 수 있습니다.
