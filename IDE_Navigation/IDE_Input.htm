IDE 입력 및 탐색

The GameMaker Studio 2 IDE accepts mouse and keyboard input, and many operations can be carried out using one or the other or both. In general you can click the left mouse button  무엇이든 선택하려면 / + 여러 항목을 선택하려면 길게 항목을 다른 도크 나 작업 공간으로 드래그하고 상황 별 메뉴를 엽니 다. macOS 시스템에서 GameMaker Studio 2를 실행 중이고 단일 버튼 마우스를 사용하는 경우 + 마우스 오른쪽 버튼 클릭 .

다양한 작업 공간 요소와 리소스 편집기를 탐색하는 데 사용할 수있는 많은 키보드 단축키가 있으며 가장 중요한 요소를 빠르게 살펴볼 것입니다 ( 여기 에서 전체 목록을 찾을 수 있음).

  • / + "Z": 이전 작업을 취소하고 대부분의 편집기에서 작동합니다. 여러 레벨의 실행 취소도 있으므로이 키를 여러 번 눌러 변경 사항을 "롤백"할 수 있습니다.
  • / + "Y": 실행 취소되고 대부분의 편집자에서 작동하는 이전 작업을 다시 실행합니다. 실행 취소와 마찬가지로 여러 레벨의 재실행도 있습니다.
  • / + "S": GameMaker Studio 2가 프로젝트를 저장하도록합니다.
  • / + "T": Goto Window 가 열리므로 게임 내 자산을 빠르게 검색하고 환경 설정 및 게임 옵션 을 검색 할 수 있습니다. 모든 항목의 목록이 표시되고 검색어를 입력 할 때이 목록이 적합하게 정리됩니다. 예를 들어, 모든 방에 "rm_"이라는 접두사가 붙은 경우 입력하면 목록을 닫아서 해당 자산 만 표시합니다. 그 이름으로 시작하십시오.

  • / + : 작업 공간 개요 창이 열리고 다른 작업 공간에서 열려있는 항목 사이를 빠르게 탐색 할 수 있습니다.

  • : 매뉴얼이 열립니다. DnD ™ 또는 GML 코드를 사용할 때 작업 또는 기능 (또는 키워드 등)을 선택한 다음 또는 마우스 가운데 버튼을 클릭하십시오 관련 페이지에서 매뉴얼을 엽니 다.
  • : IDE에서 다른 도킹 된 창을 모두 축소하거나 확장하는 데 사용할 수 있습니다.

Other ways to navigate the workspace include using the keyboard shortcut  / + + <화살표 키> : 누른 방향으로 열려있는 창 사이를 이동하고 마우스 가운데 버튼을 길게 누릅니다. 그런 다음 마우스를 드래그하여 작업 공간을 이동할 수도 있습니다.

When navigating around the various different workspaces and windows, you can use the right-mouse button 텍스트 필드에서 컨텍스트 메뉴를 열면 기본적으로 다음 필드가 있습니다.

경우에 따라서는 포커스가있는 편집기 나 창에 따라 이러한 옵션이 확장 될 수 있습니다.

이러한 입력 방법 외에도 펜 장치에 대한 지원이 제한적이며 랩톱을 사용하여 프로젝트를 수행하는 사람들을위한 특수 모드도 있습니다. 이 두 가지 모두 아래에 설명되어 있습니다.

노트북 모드 노트북 모드

랩톱에서 GameMaker Studio 2를 사용하는 경우 랩톱 모드 용 IDE 맨 위에 추가 옵션이 있습니다.

기본적으로 켜져 있지만이 버튼을 토글하여 원하는 경우 끌 수 있습니다. 랩탑 모드는 IDE의 일부 도구와 결합하여 터치 패드를 사용할 때 훨씬 더 나은 경험을 제공하여 3 가지 주요 마우스 상호 작용 ( 팬 , 확대 / 축소 및 스크롤)을 단순화합니다. 이를 위해 두 개의 수정 자 키를 사용합니다. 왼쪽 / 그리고 왼쪽 . 랩탑 모드가 켜져 있으면 왼쪽 / 그리고 왼쪽 왼쪽 방 편집기를 패닝 경우처럼 마우스 상호 작용에 대한 "예약"된다 / 줌입니다, 왼쪽 스크롤과 팬입니다. 랩톱 모드에서 스크롤하는 것은 패닝처럼 작동하므로 스크롤바를 사용할 필요가 없을 정도로 직관적이고 정확해야합니다. 이 버튼을 누른 상태에서 마우스를 움직 이기만하면됩니다 (클릭하거나 끌지 말고 움직 이기만하면 됨). 일반적인 단축키가있는 경우 일반적으로 / 또는 키 대신 키보드의 오른쪽 키를 대신 사용합니다.

터치 스크린 지원 터치 스크린 지원

여기서 마지막으로 언급 할 것은 GameMaker Studio 2 용 IDE는 터치 스크린을 최소로 지원한다는 것입니다. 모든 운영 체제에서 터치 스크린을 사용하여 기본 작업 공간에서 항목을 클릭하고 끌 수 있으며 두 번째 동시 포인터를 지원합니다. 두 번째 탭은 마우스 오른쪽 버튼 클릭을 수행합니다. Windows 8 이상에서 GameMaker Studio 2 IDE는 펜 장치도 지원합니다.
