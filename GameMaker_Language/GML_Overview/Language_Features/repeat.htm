반복

GameMaker Language에는 루프 를 수행 할 수있는 여러 가지 방법이 있습니다. 즉, 특정 횟수만큼 반복되는 문장이나 문장이 있습니다. 이들의 가장 간단한 형태를 가지고 반복 문이다 :

repeat (<expression>)
    {
    <statement>;
    <statement>;
    ...
    }

반복 하면 주어진 명령문 이 표현식 의 반올림 값으로 표시된 횟수만큼 반복됩니다. 예를 들어, 다음은 임의의 위치에 5 개의 공을 만듭니다.

repeat (5)
    {
    instance_create_layer(random(400), random(400), "Instances", obj_ball);
    }

이것은 같은 코드를 여러 번 입력하거나 배열을 사용하거나 여러 연산을 계산하는 것을 피하는 데 매우 유용 할 수 있습니다. 단일 명령문을 사용하는 것에 만 국한되지 않고 여러 개의 명령문을 묶어 반복 할 수 있습니다. 중괄호 {} 안에 있습니다. 예를 들면 다음과 같습니다.

var _x = 32;
repeat (global.p_lives)
    {
    draw_sprite(spr_heart, 0, _x, 32);
    _x += sprite_get_width(spr_heart);
    }

위의 예제는 "lives"전역 변수의 반복 이 많을 때 중괄호로 문장을 반복 하고 각 반복은 _x 위치에서 하트 스프라이트를 그린 다음 하트 스프라이트 너비에 따라 비트를 따라 위치를 이동합니다.

언제 반복 루프를 사용해야합니까? 반복 횟수를 유지할 필요없이 하나 이상의 명령문에 대해 고정 된 횟수만큼 반복하려는 경우

반복 루프 내에서 특별한 break 및 continue 문을 사용할 수 있다는 점은 주목할 가치가 있습니다. break 를 사용하면 루프를 즉시 종료하고 루프가 완료된 후 이벤트 나 함수에있는 코드로 이동합니다. 예 :

var i = 0;
var temp = 0;
repeat (10)
    {
    temp += array[i];
    if (temp > max_total)
        {
        break;
        }
    else
        {
        i += 1;
        }
    }

위의 코드는 10 개의 배열 값을 반복하여 지역 변수에 추가합니다. 로컬 변수의 총계가 max_total 의 지정된 값보다 큰 경우 루프를 break를 사용하여 종료합니다. 그렇지 않으면 루프가 계속됩니다.

반복 루프에서 continue 를 사용하는 예는 다음과 같습니다.

repeat(10)
    {   
    var _x = random(room_width);
    var _y = random(room_height);
    if (instance_position(_x, y, obj_Enemy)
        {
        continue;
        }
    instance_create_layer(_x, _y, "Instances", obj_Enemy);
    }

이 코드는 10 번 반복되어 임의의 룸 위치를 생성 한 다음 해당 위치에 obj_Enemy 객체의 인스턴스가 있는지 확인합니다. 그렇다면 현재 루프 반복이 계속을 사용하여 종료되고 새로운 반복이 시작되고, 그렇지 않으면 obj_Enemy 객체의 인스턴스가 임의의 위치에 작성됩니다.

루프 함수에 대한 자세한 예는 while , do ... until 및 for 섹션을 참조하십시오.
