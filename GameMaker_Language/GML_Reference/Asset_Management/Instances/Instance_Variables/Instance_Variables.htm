인스턴스 변수

새 객체를 만들면 이미 기본값으로 초기화 된 특정 변수가 제공됩니다. 이 변수는 게임 룸에서 오브젝트의 모든 인스턴스에 대해 생성되므로 인스턴스 변수 라고하며, 인스턴스가 생성되면 해당 변수의 값은 해당 인스턴스와 해당 인스턴스 에만 고유합니다. 이러한 변수 중 일부는 x 및 y 와 같이 많이 사용하지만 다른 변수는 수행하려는 작업에 따라 덜 필요하지만 일반적으로 매우 유용하며 가능한 경우 직접 작성하는 대신 변수를 사용하려고합니다 맞춤 변수.

대부분의 인스턴스 변수는 읽기뿐만 아니라 설정할 수 있으므로 특정 변수의 값을 조정하여 인스턴스의 속성과 동작을 변경할 수 있습니다. 예를 들어, 표시되는 내장 변수를 설정하여 인스턴스가 그려지지 않도록 할 수 있습니다 변수를 false로 설정하십시오 . 아래는 게임 내 모든 오브젝트의 모든 인스턴스에 대해 초기화 된 다양한 변수를 찾을 수 있습니다.

 

이 변수는 일반적인 인스턴스 속성을 처리합니다.

  • 고체
  • 지속성 있는
  • 깊이
  • 층
  • 경보

 

이 변수는 인스턴스 위치 및 이동을 처리합니다.

  • 중량
  • 중력 방향
  • hspeed

  • 속도

  • ystart
  • 엑스

  • 외설적 인

 

객체 속성

다음 변수는 인스턴스가 작성된 오브젝트의 고유 색인 ID를 보유합니다.

  • object_index

 

이 변수는 모두 인스턴스에 지정된 스프라이트와 관련이 있으며 그려지는 내용과 방법을 변경하는 데 사용할 수 있습니다.

  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • image_alpha
  • image_angle
  • image_blend
  • image_index
  • 이미지 _ 번호
  • image_speed
  • image_xscale
  • image_yscale

 

이 변수는 충돌 마스크를 처리합니다.

  • mask_index
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top

 

이러한 변수는 경로 및 인스턴스가 지정된 경우 경로와 상호 작용하는 방법과 관련이 있습니다.

  • path_index
  • path_position
  • path_position 이전
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

 

이 변수는 타임 라인을 사용하도록 인스턴스를 설정하기위한 것입니다.

  • timeline_index
  • timeline_running
  • timeline_speed
  • timeline_position
  • timeline_loop
