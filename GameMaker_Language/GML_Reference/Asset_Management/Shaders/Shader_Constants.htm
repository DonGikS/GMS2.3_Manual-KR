셰이더 상수

GLSL ES (OpenGL ES Shading Language) Reference Pages에 정의 된 셰이더 함수 및 상수 외에도 GameMaker Studio 2에 고유 한 다양한 셰이더 상수가 있습니다.

셰이더 배열 상수 gm_Matrices를 사용할 때 다음 디스플레이 행렬 상수를 배열 인덱스로 사용할 수 있습니다.

 

일정한 기술
MATRIX_VIEW 이 배열 인덱스 상수는 현재 뷰 매트릭스에 대한 인덱스를 유지합니다. 반환 된 인덱스는 셰이더 코드 내에서 gm_Matrices 상수를 호출 할 때 배열 값으로 사용됩니다.
MATRIX_PROJECTION 이 배열 인덱스 상수는 현재 투영 행렬에 대한 인덱스를 유지합니다. 반환 된 인덱스는 셰이더 코드 내에서 gm_Matrices 상수를 호출 할 때 배열 값으로 사용됩니다.
MATRIX_WORLD 이 배열 인덱스 상수는 현재 월드 매트릭스에 대한 인덱스를 유지합니다. 월드 공간에 조명 정보가있는 경우 조명과 같은 용도로 사용할 수 있습니다. 반환 된 인덱스는 셰이더 코드 내에서 gm_Matrices 상수를 호출 할 때 배열 값으로 사용됩니다.
MATRIX_WORLD_VIEW 이 배열 인덱스 상수는 월드와 뷰 행렬의 결과에 대한 인덱스를 유지합니다. 이것은 종종 안개와 같은 것들에 사용됩니다. 반환 된 인덱스는 셰이더 코드 내에서 gm_Matrices 상수를 호출 할 때 배열 값으로 사용됩니다.
MATRIX_WORLD_VIEW_PROJECTION 이 배열 인덱스 상수는 월드, 뷰 및 프로젝션 매트릭스를 곱한 결과에 대한 인덱스를 유지합니다. 정점 위치에 사용되는 정규 변환 행렬입니다. 반환 된 인덱스는 셰이더 코드 내에서 gm_Matrices 상수를 호출 할 때 배열 값으로 사용됩니다.
MATRIX_MAX 이것은 배열 인덱스가 아니라 정점 셰이더에서 행렬 배열의 크기를 반환합니다.
MAX_VS_LIGHTS 이것은 배열 인덱스가 아니라 정점 셰이더의 라이트 수를 반환합니다.

 

 

GameMaker Studio 2의 특정 값에 액세스하기 위해 다음과 같은 사전 정의 된 매트릭스 유니폼과 상수를 쉐이더에서 사용할 수 있습니다 :

 

일정한 기술
gm_ 행렬 [행렬] 이 배열 상수는 변환 매트릭스를 반환하며 GameMaker Studio 2가 쉐이더 코드 편집기 내에서 사용할 수 있도록 미리 정의 된 유니폼 중 하나입니다. 배열 인덱스는 위에 나열된 상수 중 하나에서 선택됩니다.
gm_BaseTexture GameMaker Studio 2에서 설정 한대로 현재 오브젝트의 텍스처를 반환하는 2D 샘플러 상수입니다. 따라서 셰이더를 호출하지 않고 그릴 때 일반적으로 사용되는 현재 스프라이트, 표면 또는 텍스처입니다.
gm_LightingEnabled 3D를 사용할 때 GameMaker Studio 2 조명을 가져 오거나 설정하는 데 사용할 수 있습니다.
gm_FogStart 폴리곤이 안개 색깔과 혼합되기 시작하는 거리를 얻는 데 사용할 수 있습니다.
gm_RcpFogRange 안개가 최대 인 거리를 구하는 데 사용할 수 있으며 더 이상 아무것도 볼 수 없습니다.
gm_PS_FogEnabled GPU에 픽셀 포그가 활성화되어 있으면 true 또는 false 를 반환 합니다 .
gm_FogColour GameMaker Studio 2에서 사용되는 안개 색상을 얻는 데 사용할 수 있습니다.
gm_VS_FogEnabled GPU에 버텍스 포그가 활성화되어 있는지 여부는 true 또는 false 를 반환 합니다 .
gm_AlphaTestEnabled 셰이더에서 알파 테스트를받는 데 사용할 수 있습니다. 알파 테스트에 대한 자세한 내용은 gpu_set_alphatestenable () 을 참조하십시오.
gm_AlphaRefValue 현재 알파 테스트 참조 값을 얻는 데 사용할 수 있습니다. 알파 테스트 참조에 대한 자세한 내용은 gpu_set_alphatestref () 를 참조하십시오.
