skeleton_attachment_create_colour

스켈 레탈 애니메이션 스프라이트를 만들 때 첨부 슬롯 과 첨부 파일 을 할당하여 첨부 할 수 있습니다. 이것들은 단순히 애니메이션과 분리 된 이미지 (스프라이트)이지만, 부착되면 부착 된 뼈와 함께 움직입니다. 일반적으로 애니메이션 프로그램 (Spine)에서 이러한 첨부 파일을 할당하지만이 기능을 사용하면 게임의 스프라이트 자산을 사용하여 런타임에 직접 만들 수 있습니다. 이 함수를 사용하려면 첨부 파일에 이름 (문자열)을 지정한 다음 사용할 sprite_index 및 image_index 와 x 및 y 원점 (스프라이트 속성에서 스프라이트에 의해 정의 된 것과 다를 수 있음)을 설정해야합니다. 첨부 할 때 이미지에 적용 할 변환뿐만 아니라 이미지와 혼합 할 색상 및 알파 (투명도) 값을 설정할 수도 있습니다. 첨부 파일이 성공적으로 생성되면 함수는 1 을 반환하고 그렇지 않은 경우 (유효하지 않은 스프라이트 인덱스를 제공했거나 기본 스프라이트가 스파인 스프라이트가 아닌 경우) -1 을 반환합니다.

인스턴스 스프라이트는 혼합 색상 및 알파 설정 (가질 수 있다는 것을 주목할 가치가있다 image_blend 및 image_alpha를 장착 슬롯 (함수시피) skeleton_slot_colour_set을 () ) 및에 할당 부착 스프라이트 그 최종 색상 및 알파 그래서 slot은이 모든 값을 합성 한 것입니다.

 

통사론:

skeleton_attachment_create_colour(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot, colour, alpha);

논의 기술
이름 작성할 첨부 파일의 이름 (문자열)입니다.
스프라이트 첨부 이미지를 가져올 sprite_index
산업 첨부 파일 이미지를 가져올 image_index입니다.
Xorigin 사용중인 이미지의 x 원점.
요리 긴 사용중인 이미지의 y 원점입니다.
xscale 승수로 이미지의 수평 스케일링 : 1 = 일반 스케일링, 0.5는 절반 등 ...
yscale 승수로 이미지의 수직 스케일링 : 1 = 일반 스케일링, 0.5는 절반 등 ...
썩음 이미지의 회전 : 0 = 보통, 90 = 시계 반대 방향으로 90도 회전
색깔 사용할 색상 값 (상수, 정수 또는 16 진 값)입니다.
알파 사용할 알파 값입니다 (0에서 1까지).

 

보고:

1 if successful, -1 if not

 

예:

skeleton_attachment_create_colour("sword", spr_Weapons, 0, 0, 80, 1, 1, 90, c_red, 1);
skeleton_attachment_create_colour("knife", spr_Weapons, 1, 0, 45, 1, 1, 90, c_yellow, 1);
skeleton_attachment_create_colour("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0, c_white, 0.5);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

위의 코드는 슬롯 "slot_leftHand"에 대해 현재 할당 된 첨부 파일 이름을 확인하고 빈 문자열이 반환되면 새 스프라이트가 첨부됩니다.
