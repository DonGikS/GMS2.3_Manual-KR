타일 ​​맵 레이어

GameMaker Studio 2 Room Editor를 사용하면 레이어를 사용하여 주어진 방에 타일을 추가 할 수 있습니다. 이름에서 알 수 있듯이 타일 맵 레이어는 단순히 방 안에서 같은 깊이에있는 것으로 간주되는 타일 세트 타일의 모음입니다. 레이어를 쌓아서 다른 것들 위에 또는 그 아래에 무언가를 그릴 수 있습니다. 코드에서 레이어의 특정 측면을 제어하거나, 항목을 추가 또는 제거하거나, 레이어의 특정 속성이나 런타임에 포함 된 속성을 변경하는 것도 가능합니다. 룸 편집기에서는 레이어 당 하나의 타일 맵 ( 타일 ​​레이어 )으로 제한되지만 코드에서 타일 맵으로 작업 할 때 단일 맵에 여러 개의 타일 맵을 할당 할 수 있으며 각 맵에는 고유 한 ID와 속성 :

  • 타일 ​​세트 는 타일 레이어 및 타일 맵을 만드는 데 사용하는 리소스입니다.
  • 타일 ​​레이어 는 룸 편집기에서 타일 세트를 추가하기 위해 생성 한 것입니다.
  • 타일 ​​맵 은 룸 편집기 또는 코드를 통해 해당 레이어의 단일 요소로 레이어에 추가 된 타일 모음입니다.

타일 ​​맵을 설정하기 위해 위에 나열된 기능 외에도 개별 타일의 데이터를 조작하는 데 사용할 수있는 여러 기능이 있습니다. 타일 ​​데이터의 "블롭 (blob)"은 타일 색인과 몇 개의 추가 비트로 구성되어 타일의 회전, 반전 또는 미러링 여부를 보여줍니다. 타일 ​​데이터에 고유 한 비트 마스크를 추가 할 수도 있습니다. 타일 인덱스 용으로 예약 된 비트를 마스킹 할 때만 고급 사용자를위한 것이지만 기본적으로 타일 인덱스 용으로 예약 된 19 비트가 있습니다. 그 중 8 개만 사용하면 11 개의 비트가 남게되어 사용자 지정 마스크를 만드는 데 사용할 수 있습니다. 그런 다음이 마스크를 코드에서 사용하고 확인하여 사용자 지정 타일 충돌 등을 만들 수 있습니다. 그러나 일반적으로 자신의 타일 데이터 얼룩을 만들 필요가 없으며 대신 tilemap_get () 함수를 사용하여 타일 데이터를 가져온 다음 아래 함수를 사용하여 조작 한 다음 tilemap_set ( )
