asset_remove_tags

이 기능을 사용하면 자산 브라우저에서 자산에 대한 하나 이상의 태그 문자열을 제거 할 수 있습니다. 자산 이름 (문자열) 또는 자산 색인뿐만 아니라 단일 태그 문자열 또는 각 항목이 단일 태그 문자열 인 배열을 제공합니다. 자산 색인 값을 제공하는 경우 다른 유형의 자산은 동일한 이름을 가질 수 없더라도 동일한 색인을 가질 수 있으므로 선택적 자산 유형 인수 (상수)를 제공해야합니다. 사용 가능한 자산 유형이 아래 표에 나열되어 있습니다.

 

일정한 기술
asset_object 이름은 객체를 나타냅니다.
asset_sprite 이름은 스프라이트를 나타냅니다.
asset_sound 이름은 소리를 나타냅니다.
asset_room 이름은 방을 나타냅니다.
asset_tiles 이름은 타일 ​​세트를 나타냅니다.
asset_path 이름은 경로를 나타냅니다.
asset_script 이름은 스크립트를 나타냅니다.
asset_font 이름은 글꼴을 나타냅니다.
asset_timeline 이름은 타임 라인을 나타냅니다.
asset_shader 이름은 셰이더를 나타냅니다.
asset_animationcurve 이름은 애니메이션 커브를 나타냅니다.
asset_sequence 이름은 Sequence를 나타냅니다.

 

함수가 태그를 성공적으로 제거하면 true 를 반환 하고 그렇지 않으면 false 를 반환합니다.

 

 

통사론:

asset_remove_tags(name_or_index, tags, [asset_type]);

논의 기술
name_or_index 자산의 이름 (문자열) 또는 색인 값 (정수).
태그 단일 자산 태그 문자열 또는 다양한 자산 태그가있는 배열입니다.
[자산 _ 유형] 선택 과목! 태그를 제거 할 자산 유형으로, 첫 번째 인수에 색인을 제공 할 때만 사용됩니다.

 

보고:

Boolean

 

예:

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object);

위의 코드는 태그 배열을 만든 다음 지정된 객체에서 태그를 제거합니다.
