증기

이 섹션은 게임을 해당 플랫폼에 게시하기 위해 Steam API에 액세스 할 수 있고 프로젝트에서 사용자 컨텐츠를 추가 및 생성하기 위해 Steam 창작 마당 및 커뮤니티에서 제공 할 수있는 가능성을 사용하려는 사용자를위한 것입니다. 가장 간단한 형태의 사용자 생성 콘텐츠는 사용자가 스크린 샷을 찍고 공유 할 수있는 기능으로, 다음 두 가지 기능을 사용하여 촉진됩니다.

 

  • steam_is_screenshot_requested
  • steam_send_screenshot

 

이 페이지의 나머지 기능은 Steam Worksop 사용과 관련이 있으며 일부는 프로젝트의 Steam 앱 ID 와 게임을하는 사람의 사용자 ID 또는 사용자 이름을 필요로합니다. 다음 기능들 :

 

  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_persona_name

 

중 하나를 사용하기 전에 UGC (U 버리는 G enerated C ontent) API는 스팀 대시 보드에서 제대로 게임을 설정하고 당신이 필요한 문서를 읽어해야 여기에서 찾을 필요로하는 증기를위한 기능 내장 :

  • 사용자 생성 컨텐츠 공유

참고 : Steam 온라인 상점에서 게임을 수락하고 Steam API 설명서의 개발자 영역에 액세스 할 수 있어야합니다.

가입 한 모든 UGC 항목은 Steam 클라이언트에 의해 자동으로 다운로드되며,이를 스팀으로 잡아서 다른 UGC 기능에 사용하기 위해 다운로드 된 UGC의 ID를 저장하려면 Steam 비동기 이벤트 에 코드가 있어야합니다.

중대한! Steam UGC ID는 600,000,000 이상이 될 수 있습니다. 이것은 때로는 버퍼로 작업하거나 텍스트 파일에 값을 쓰려고 할 때 실제 값으로 저장하지 않고 문자열로 저장해야 할 수도 있습니다 (단순한 표준 형식으로 변환하기 때문에) "6.6624e + 003"과 같이 문제를 다시 읽습니다).

UGC를 게임에 도입하기위한 일반적인 작업 과정은 다음과 같습니다.

  1. 사용자는 항목을 구독합니다 ( steam_ugc_subscribe_item ()을 사용하는 게임 또는 클라이언트 / 브라우저).
  2. 게임은 Steam Async Event에서 적절한 코드로 새로운 구독을 "듣습니다".
  3. 구독 비동기 이벤트가 발생하면 steam_ugc_get_item_install_info ()를 사용 하여 항목이 설치되어 있는지 (예 : 다운로드) 확인합니다.
  4. 항목이 설치되어 있지 않은 경우 다운로드 진행 상황을 추적하려면 steam_ugc_get_item_update_info () 를 사용하여 확인해야합니다.

다음 섹션은 GameMaker Studio 2에서 UGC 기능을 수행하는 데 필요한 모든 기능을 설명합니다.

 

다음 함수는 본질적으로 서버에 컨텐츠를 작성하고 업로드하기 위해 Steam API에서 제공되는 기능에 대한 "래퍼"함수입니다. 따라서 링크 된 Steam 설명서를 사용하기 전에 내용을 읽고 이해하는 것이 좋습니다. 내용 작성 및 업로드 .

 

  • steam_ugc_create_item
  • steam_ugc_start_item_update
  • steam_ugc_set_item_title
  • steam_ugc_set_item_description
  • steam_ugc_set_item_visibility
  • steam_ugc_set_item_tags
  • steam_ugc_set_item_content
  • steam_ugc_set_item_preview
  • steam_ugc_submit_item_update
  • steam_ugc_get_item_update_progress

 

 

사용자 컨텐츠가 작성되고 워크샵에서 다운로드 할 수있게되면 사람들은 Steam 앱 또는 웹 포털을 통해 해당 컨텐츠를 구독 할 수 있습니다. 그러나 GameMaker Studio 2에는 또한 구독을 생성 및 취소하고 사용자가 현재 구독 한 대상에 대한 정보를 얻기 위해 Steam API를 사용하는 다음 기능이 포함되어 있습니다.

 

  • steam_ugc_subscribe_item
  • steam_ugc_unsubscribe_item
  • steam_ugc_num_subscribed_items
  • steam_ugc_get_subscribed_items
  • steam_ugc_get_item_install_info
  • steam_ugc_get_item_update_info
  • steam_ugc_request_item_details
