인스턴스 변수 설정

이 조치를 사용하면 내장 변수 중 하나를 설정할 수 있습니다. 모든 객체의 모든 인스턴스에는 현재 x / y 위치, 속도, 방향, 가시성 및 기타 많은 것들과 같은 필수 정보를 보유하기 위해 이미 작성된 여러 변수가 있습니다. 사용할 수있는 인스턴스 변수는 다음과 같습니다.

  • X 좌표 -방의 x 축을 따른 위치입니다.
  • Y 좌표 -방에서 y 축을 따른 위치입니다.
  • 방향 -인스턴스의 이동 방향 (기본값은 0, 오른쪽은 0 °, 위쪽은 90 °, 왼쪽은 180 °, 아래쪽은 270 °)입니다.
  • 속도 -인스턴스가 이동하는 속도 (게임 프레임 당 픽셀 단위)입니다.
  • 수평 속도 -인스턴스가 움직이는 수평 속도 (게임 프레임 당 픽셀 단위)입니다. 음의 속도는 왼쪽으로 이동하고 양의 속도는 오른쪽이며 기본값은 0입니다.
  • 수직 속도 -인스턴스가 움직이는 수직 속도 (게임 프레임 당 픽셀 단위)입니다. 음의 속도는 올라가고 양의 속도는 내려 가고 기본값은 0입니다.
  • Visible- 가시성 토글 변수입니다 ( Object Editor 에서 다르게 설정하지 않으면 기본값은 true 이며 보이지 않게하려면 false 로 설정 됨). 보이지 않는 인스턴스는 Draw 이벤트를 제외한 모든 이벤트를 계속 실행합니다.
  • 영구 - 변수 토글 지속성 (기본 그렇지 않으면에서 설정하지 않는 거짓 오브젝트 편집기 , 지속하기 위해 true로 설정). 이렇게하면 인스턴스가 방마다 "지속적으로"유지됩니다. 즉, 인스턴스는 방 끝에서 정리되지 않고 대신 다음 방으로 이월됩니다. 인스턴스가 지속 될 때이 점에 유의 하지 그것의 작성이나 이벤트를 파괴 부르지 만, 정리, 룸 끝, 및 룸 시작 이벤트를 호출합니다.
  • 레이어 -인스턴스가 할당 된 레이어입니다.
  • 깊이 -레이어에 지정된 깊이 값입니다.
  • Sprite- 인스턴스에 할당 된 스프라이트 리소스입니다.
  • 이미지 인덱스 -지정된 스프라이트의 이미지 인덱스입니다. 이 값은 인스턴스에 할당 된 스프라이트의 프레임을 설정하며, 값이 애니메이션의 프레임 수보다 큰 경우 프레임 7로 설정된 경우 5 프레임의 스프라이트가 실제로 프레임 2를 표시하도록 "루프"됩니다.
  • 이미지 각도 -인스턴스가 설정되어야하는 각도 (기본값은 0, 오른쪽은 0 °, 위쪽은 90 °, 왼쪽은 180 °, 아래쪽은 270 °)입니다.
  • 이미지 알파 -인스턴스의 알파 값 (기본값은 1)부터 0까지입니다. 여기서 0은 완전 투명하고 1은 완전히 불투명합니다.
  • 이미지 블렌드 -인스턴스 스프라이트와 블렌딩 할 색상입니다 (기본값은 흰색).
  • 이미지 속도 -인스턴스의 애니메이션 속도입니다.
  • 수평 스케일 -인스턴스 스프라이트를 그려야하는 수평 스케일입니다 (이는 인스턴스 경계 상자 및 충돌에도 영향을 미칩니다). 1의 스케일은 스케일링 없음 (1 : 1)을 나타내며, 더 작은 값은 스케일 다운됩니다 (예 : 0.5는 스프라이트 너비의 절반), 더 큰 값은 스케일 업되고 음수 값은 스프라이트를 미러링 하고 스케일링하지 않는 한 스케일링합니다 사용 된 값은 정확히 -1입니다 (이 경우 스프라이트는 스케일링없이 y 축을 따라 미러링됩니다).
  • 수직 스케일 -인스턴스 스프라이트가 그려지는 수직 스케일입니다 (이는 인스턴스 경계 상자 및 충돌에도 영향을 미칩니다). 작은 값 (0.5, 예를 들면, 스프라이트 절반 높이는 것) 축소되며, 큰 값은 확장되고 음수가 스프라이트 플립 것이다 : (1)의 크기는 스케일링 (1) 나타냄 과 않으면 그것을 스케일 사용 된 값은 정확히 -1입니다 (이 경우 스프라이트는 스케일링없이 y 축을 따라 뒤집 힙니다).
  • 마스크 -인스턴스의 마스크 색인입니다. 기본값은 -1 (인스턴스가 현재 할당 된 스프라이트 인덱스에 충돌 마스크를 사용함을 의미 함)이지만 다른 스프라이트 리소스로 설정하면 해당 인스턴스에 대한 충돌 마스크가 스프라이트에 할당 된 스프라이트 대신 사용됩니다 인덱스.
  • Friction- 게임의 각 게임 프레임을 사용하여 인스턴스를 느리게하는 마찰입니다. 이 값은 각 게임 프레임의 현재 속도 값에서 공제됩니다 (또는 속도 값이 음수 인 경우 추가됨).
  • 중력 -중력은 인스턴스의 중력 방향 (아래 참조)에 일정한 힘을 가하여 속도에 영향을줍니다 (기본값은 0). 중력은 누적 힘이며 최종 속도를 제한하지 않으면 객체를 가속시킵니다.
  • 중력 방향 -인스턴스 중력이 0보다 큰 경우 (기본값은 270) 이동 방향을 설정하는 데 사용할 수 있습니다. GameMaker Studio 2의 방향은 0 °는 오른쪽, 90 °는 위쪽, 180 °는 왼쪽, 270 °는 아래쪽으로 계산됩니다.
  • 점수 - 점수 변수를 설정합니다.
  • 생활 - 생활 변수를 설정합니다.
  • 건강 - 건강 변수를 설정합니다.

사용할 내장 변수를 선택한 후 해당 변수의 값을 설정하십시오. 사용하는 값은 선택한 변수에 따라 다르며 (기본값 및 범위는 위 목록 참조) 변수의 현재 값을 기준으로 값을 표시 할 수 있습니다. 더하기 아이콘을 클릭하여 동일한 동작으로 설정할 변수를 추가 할 수도 있습니다 작업 옆에 다른 인스턴스 변수를 선택하고 값을 지정하십시오.
