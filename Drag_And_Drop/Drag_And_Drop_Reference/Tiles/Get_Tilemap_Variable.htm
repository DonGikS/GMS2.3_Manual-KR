타일 ​​맵 변수 가져 오기

이 조치를 사용하면 레이어의 타일 ​​맵 요소에 대해 지정된 수의 변수 중 하나를 검색 할 수 있습니다. 회의실 편집기에서 타일 도면층을 작성하면이 도면층에 타일 ​​맵 요소 가 포함되어 타일 세트의 타일로 채워집니다. 이 타일 맵 요소에는 오프셋 위치, 사용중인 타일셋, 너비, 높이 등과 같은 특정 값이 있습니다. 얻을 수있는 전체 값 목록은 다음과 같습니다.

  • X 좌표 : 타일 ​​맵 요소가 배치 된 레이어에서 x 축을 따른 위치입니다.
  • Y 좌표 : 타일 ​​맵 요소가 배치 된 레이어에서 y 축을 따른 위치입니다.
  • Columns : 타일 ​​맵 요소의 너비 (타일 셀의 열 수로 정의)
  • 행 : 타일 ​​맵 요소의 높이 (타일 셀의 행 수로 정의)
  • Tile Width : 단일 타일 셀의 너비입니다.
  • 타일 ​​높이 : 단일 타일 셀의 높이입니다.
  • Tile Set Resource : 타일 ​​맵 요소에 할당 된 타일셋 리소스입니다.
  • 현재 프레임 : 애니메이션 타일을 위해 그려지는 현재 프레임.
  • 마스크 : 타일 ​​맵 요소의 마스크 데이터입니다.
  • Global Mask : 모든 타일 맵 요소에 대한 글로벌 마스크 데이터.

When using this action you supply the layer name (a string, as defined in the Room Editor) to get the tile map element data from, then the type of data that you want to retrieve (as shown in the list above). The returned value will then be stored in the target variable which can have been created previously or can be a new temporary one (if you check the "Temp" check-box). Note that you can retrieve additional values by clicking the plus icon 액션 옆에 다른 변수를 선택하고 반환 된 값을 저장할 다른 변수를 제공합니다.
