드로잉 액션 라이브러리

그리기 작업 라이브러리는 스프라이트, 텍스트 또는 모양을 그리고 특정 그리기 속성을 설정하는 데 필요한 작업을 찾을 수있는 곳입니다. 이러한 액션의 대부분은 객체 의 다양한 Draw 이벤트 에서만 사용되며 Draw 이벤트 외부에서 사용하는 경우 작동하지 않을 수 있습니다. 이에 대한 예외는 이벤트에 추가 할 수있는 설정 작업이며 이후 모든 인스턴스의 모든 도면에 영향을 미칩니다.

오브젝트의 메인 Draw 이벤트 에 액션을 추가하면, Draw 와 같은 액션을 사용하여 GameMaker Studio 2에 명시 적으로 지시하지 않는 한 인스턴스에 지정된 스프라이트를 그리지 않습니다. 자기 . 기본적으로 GameMaker Studio 2는 Draw 이벤트에 다른 항목이 없는 경우에만 인스턴스에 지정된 스프라이트를 기본적으로 그립니다.
