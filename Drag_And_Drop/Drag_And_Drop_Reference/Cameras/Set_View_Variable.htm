뷰 변수 설정

이 조치를 통해 뷰 포트 가 방의 카메라 뷰를 표시하는 방법과 관련된 특정 내장 변수를 설정할 수 있습니다. 편집 할 뷰 포트 변수를 선택한 다음 뷰 포트를 대상 (0 ~ 7)에 제공하고 마지막으로 변수의 새 값을 지정하십시오. 뷰 포트는 룸 편집기에서 설정 한대로 주어진 카메라 뷰를 그리는 데 사용되는 화면 영역이며,이 방법으로 설정할 수있는 다양한 변수는 아래 표에 요약되어 있습니다.

변하기 쉬운 기술
카메라 다른 카메라 뷰를 표시하도록 뷰포트를 설정하기위한 것입니다. 로 변경하려면 카메라의 고유 ID 값을 제공해야합니다 (자세한 내용은 여기 참조).
시계 이 뷰 포트에 대한 가시성 전환하고 true로 설정하면 뷰 포트가 보이게됩니다 true 또는 false 값 예상 (예 :가 화면에 그리는 것), 거짓의 값은 아무것도 그리지 않습니다 화면.
포트 X 좌표보기 게임 창 내에서 뷰 포트의 X 위치입니다. 분할 화면 효과 만들기와 같이 둘 이상의 뷰 포트가 활성화 된 경우에만 필요하며 포트가 하나만 활성화 된 경우 사용하면 예기치 않은 결과가 발생합니다.
포트 Y 좌표보기 게임 창 내에서 뷰 포트의 Y 위치입니다. 분할 화면 효과 만들기와 같이 둘 이상의 뷰 포트가 활성화 된 경우에만 필요하며 포트가 하나만 활성화 된 경우 사용하면 예기치 않은 결과가 발생합니다.
포트 너비보기 뷰 포트의 너비 (픽셀)입니다. 뷰 포트의 너비 (또는 둘 이상의 활성화 된 경우 결합 된 뷰 포트)는 게임 시작시 게임 창 또는 배경 캔버스의 너비를 정의하므로 게임이 시작된 후이 값을 변경해 도 아무런 영향을 미치지 않습니다. GML 함수 window_set_size () 와 함께 호출되지 않는 한 게임 창 크기 또한 카메라에 할당 된 것보다 크거나 작은 포트 크기가 있으면 카메라보기의 크기가 축소 또는 확대됩니다.
포트 높이보기 뷰 포트의 높이 (픽셀)입니다. 뷰 포트의 높이 (또는 둘 이상의 활성화 된 경우 결합 된 뷰 포트)는 게임 시작시 게임 창 또는 배경 캔버스의 높이를 정의하므로 게임이 시작된 후이 값을 변경해 도 아무런 영향을 미치지 않습니다. GML 함수 window_set_size () 와 함께 호출되지 않는 한 게임 창 크기 또한 카메라보기에 할당 된 것보다 크거나 작은 포트 크기가 있으면 카메라보기의 크기가 축소 또는 확대됩니다.
포트 표면 ID보기 이 변수를 사용하면 지정된 뷰 포트의 내용을 표면에 그리도록 설정할 수 있으며, 제공 한 값은 뷰 포트를 그리는 데 사용할 표면의 값입니다. 자세한 내용은 여기 를 참조 하십시오 .
