마우스와 키보드 액션 라이브러리

마우스 및 키보드 키보드 또는 마우스 장치에서 설문 조사와 프로세스 입력에 필요한 조치를 찾을 수있는 조치 라이브러리입니다. 사용 가능한 조치가 모두 아래에 나열되어 있습니다.

 

마우스를 누르면
마우스 아래로
마우스가 풀린 경우
키를 누르면
키를 누르면
키를 놓은 경우

 

 

 

 

 

 

 

 

 

여기서 다른 세 가지 동작은 가상 키보드를 나타 냅니다. 게임에서 터치 스크린 장치의 텍스트 또는 숫자 입력이 필요한 경우 제공된 가상 키보드 동작을 사용하여 장치 OS 키보드를 사용할 수 있습니다. 이 액션은 OS 키보드를 불러 오거나 숨기고 물리적 키보드와 동일한 입력을 받아들이므로 일반적인 GameMaker Studio 2 키보드 액션은 모두 가상 키보드에서 작동해야합니다 (예 : "키가 풀린 경우"액션). 가상 키보드의 입력 내용은 keyboard_string 또는 keyboard_last_char 등과 같은 다른 키보드 변수에 저장됩니다.

가상 키보드를 열거 나 닫으면 System Asynchronous Events 가 생성되어 async_load DS 맵 에 다음 키 / 값 쌍이 채워집니다.

  • " event_type "-수신 된 시스템 이벤트 유형으로, 가상 키보드의 경우 " 가상 키보드 상태 "가됩니다.
  • " screen_height "-가상 키보드의 높이 (픽셀). 키보드가 보이지 않으면 0이됩니다.
  • " keyboard_status "- 키보드 의 현재 상태로, 다음 문자열 중 하나로 반환됩니다. "숨김"
  • "숨겨진"
  • "전시"
  • "명백한"

사용 가능한 가상 키보드 동작은 다음과 같습니다.

 

가상 키보드 표시
가상 키보드 숨기기
가상 키보드 높이 확보
가상 키보드가 표시되는 경우
