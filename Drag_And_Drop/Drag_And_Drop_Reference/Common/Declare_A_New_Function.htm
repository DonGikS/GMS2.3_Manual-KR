새로운 기능 선언

이 액션은 새로운 사용자 정의 함수를 만드는 데 사용될 수 있으며, 나중에 코드에서 언제든지 호출 할 수 있습니다. 사용자 정의 함수는 사용 가능한 DnD ™ 액션 또는 GML 러너 함수에서 다루지 않는 작업을 수행하도록 정의한 함수이며 전역, 인스턴스 또는 로컬 범위 일 수 있습니다 (범위에 대한 자세한 내용은 여기 참조). 또한 구조체 를 생성하는 생성자 로도 사용할 수 있습니다. 이 조치를 사용하여 글로벌 함수를 작성하는 방법에 대한 자세한 정보는 조치 함수 작성 섹션을 참조하십시오. 함수를 static 으로 플래그를 지정할 수도 있습니다. 이는 처음 사용할 때 한 번만 정의됨을 의미합니다.

이 조치를 사용하려면이를 인스턴스 이벤트에 추가 한 후 이름을 선언하려는 함수를 제공하십시오. 그런 다음 함수가 사용할 각 매개 변수에 필요한 인수 이름을 제공하십시오. 함수가 생성자로 사용되는 경우 (즉, 구조체를 생성 할 예정인 경우) 생성자 플래그를 확인한 다음 New 액션을 사용해야하는 함수를 호출해야합니다. 함수가 같은 이벤트에서 로컬에서만 사용될 예정이라면 Temp 플래그를 선택할 수도 있습니다. 이름과 매개 변수를 정의한 후에는 아래 이미지와 같이 코드를 호출 할 때 함수 옆에 놓아 함수를 실행할 동작을 추가 할 수 있습니다.

아래에서 일반 함수와 생성자 함수로 사용하는 예를 볼 수 있습니다.
