자산 브라우저 에서 가장 중요한 자산 중 하나는 스크립트 자산입니다. 스크립트는 본질적으로 사용자 정의 함수를위한 "컨테이너"이며 실제로 게임에서 직접 호출하지는 않는다는 점에서 특별한 자산입니다. 모든 스크립트는 범위 내에서 전역 적 이며 게임이 시작될 때 GameMaker에서 자동으로 실행되므로 게임 전체에서 사용할 전역 변수 및 함수를 정의하는 데 사용할 수 있습니다.

여기서 정의한 함수는 범위 내에서 전역 적이며 내장 된 DnD ™ 액션과 매우 유사하게 작동합니다. 즉, 인수 (또는 매개 변수)라고하는 다른 값을 가져 오거나 전혀 사용할 수 없으며 작업을 수행하여 반환합니다. 필요에 따라 값을 반환하거나 아무것도 반환하지 않습니다. 그런 다음 함수 호출을 사용하여 게임에서 함수를 호출합니다. 동작은 다음과 같습니다.

DnD ™ 스크립트를 만들려면 마우스 오른쪽 버튼을 사용하면됩니다. 자산 브라우저의 아무 곳에서나 생성을 선택한 다음 스크립트를 선택 하십시오 . 그러면 새로운 DnD ™ 스크립트가 생성되고 편집기 창이 열립니다.

위에서 언급했듯이 여러 인수를 함수에 전달할 수 있으며,이를 사용하여 작성한 작업을 수행합니다. 이를 더 잘 이해하려면 예를 들어 내장 조치를 취하고 작동 방식에 대해 생각하십시오. 이벤트에 배치하고 많은 값을 제공하면 인스턴스가 무언가를 수행하게됩니다. 사용자 정의 함수는 정확히 동일하지만 사용자가 작성하는 차이점은 다릅니다.
