끌어서 놓기 개요

드래그 앤 드롭 (DnD ™)은 실제로 코드를 입력하지 않고도 게임을 만드는 데 사용할 수있는 시각적 스크립팅 도구입니다. DnD ™를 사용할 때 코드를 프로그래밍하지 않는다고 말하는 것은 아닙니다. 코드는 텍스트가 아닌 동작의 "블록"을 사용하여 시각적으로 생성되고 표시됩니다. 동작은 객체의 인스턴스가 방에 배치 될 때 무언가를 수행하도록 함께 "연결"될 수있는 간단한 표현 또는 진술 입니다. 예를 들어, Create Event 에서 작업을 임의의 방향으로 이동시키고 Collision Event 에서 다른 작업을 수행하여 충돌 하는 항목을 제거 할 수 있습니다 (사용 가능한 다른 이벤트 및 사용 방법에 대한 자세한 내용 참조). 함께 작업하면 객체 이벤트 섹션을 참조하십시오.

위 이미지는 편집 할 작업 영역 에서 열린 일반적인 개체를 보여줍니다. 여기에는 "이벤트"창과 "코드"창이 연결되어 있으며 코드 창에서 DnD ™를 구성한 위치를 확인할 수 있습니다. 이 창을 조금 더 자세히 살펴보고 어떤 옵션을 사용할 수 있는지 살펴 보겠습니다.

아래에 표시된 각 섹션에 대한 자세한 정보가 있습니다.

이벤트 탭 이벤트 탭

객체에 이벤트 를 추가하면 코드 창에 새 탭이 열리고 해당 이벤트의 내용이 표시됩니다. 여러 이벤트를 추가하고 동시에 여러 탭을 열어서 원하는대로 전환하고 편집 할 수 있습니다. 언제든지 탭을 닫아 혼란을 줄일 수 있습니다 (오브젝트에서 이벤트를 다시 선택하면 다시 열림). 창 밖으로 탭을 자체 코드 창으로 드래그하십시오.이 경우 객체를 닫으면 코드 창이 새 창을 닫지 않습니다. 탭을 왼쪽이나 오른쪽으로 드래그하여 재정렬 할 수도 있습니다.

 

동작 개요 동작 개요

이벤트에 작업 블록을 추가하면 진행중인 작업에 대한 일반 영어 개요로 "속기"형태로 나타납니다. 개요 순서는 해당 이벤트에 대해 각 조치 블록이 실행되는 순서이며, 목록에서 임의의 항목을 클릭하여 조치 블록보기를 해당 위치로 확대하여 편집 할 수 있습니다.

 

작업 블록 작업 영역 작업 블록 작업 영역

이 작업 영역은 도구 상자 에서 작업을 끌어 DnD ™ 코드를 만드는 곳입니다. 딸깍 하는 소리 조치에서이 영역으로 끌어서 조치 목록에 추가하고 편집하십시오. 대부분의 액션에는 설정할 수있는 다양한 인수 (매개 변수)가 있으며 대부분 키워드 , 인스턴스 변수 또는 GML 함수 를 사용합니다. 예를 들어 방향 설정 액션을 호출하고 GML 함수 random (val) 을 사용하여 임의의 방향.

코드에 동작 블록을 추가하면 서로 다른 동작이 발생하는 순서를 나타 내기 위해 시각적으로 "연쇄 적으로"연결되지만,이 체인을 만드는 방법은 사용 된 초기 블록에 따라 다릅니다. 예를 들어, If Variable 블록에는 두 개의 체인이 있습니다. 하나는 검사중인 조건이 true 인 경우 호출 될 블록에 대한 체인이며 그 다음에 다음 블록입니다. 액션 블록을 코드로 구성하는 방법에 대한 자세한 내용은 아래 링크 된 섹션을 참조하십시오.

작업 블록 작업 공간에서 마우스 가운데 버튼을 길게 누를 수 있습니다 이동하고 마우스 휠 조치 목록을 위 또는 아래로 스크롤하십시오. 당신이 개최하는 경우 마우스 휠을 움직이면 뷰가 왼쪽 / 오른쪽으로 스크롤됩니다. / 마우스 휠을 움직이면보기가 확대 및 축소되어 작업과 함께 더 많은 작업 공간을 볼 수 있습니다. 클릭 할 수 있습니다 왼쪽의 작업 목록에있는 작업에서 해당 작업에 초점을 맞추고 확대 / 축소 및 위치를 재설정합니다. 또한 마우스를 블록 가장자리쪽으로 이동 한 다음 커서가 변경 될 때 필요한 방향으로 드래그하여 동작 블록의 크기를 조정할 수 있습니다.

마우스 오른쪽 버튼을 클릭해도됩니다 DnD ™ 작업 공간의 어느 곳에서나 RMB 메뉴를 불러옵니다. 여기 에 자세히 설명되어 있습니다 .

 

도구 상자 도구 상자

툴박스 는 모든 DnD ™ 액션이 저장되는 곳이며, 서로 다른 섹션에는 유사하거나 관련된 액션 모음 (라이브러리)이 있습니다. 각 작업 모음은 특정 테마 또는 공통 목적과 관련이 있으며 클릭하면됩니다. 필요한 조치를 도구 상자에서 조치 블록 작업 공간으로 끌어 현재 이벤트에 추가하십시오. 작업이 많을수록 작업을 단순화하기 위해 상단의 "검색"막대를 사용하여 특정 작업을 검색하거나 표시되는 작업을 필터링 할 수 있습니다 (예 : "그리기"를 입력하여 모든 그리기 작업을 가져옵니다) 을 클릭해도됩니다 라이브러리에서 즐겨 찾기 폴더로 아이콘을 끌어서 자주 사용하는 작업을 함께 유지할 수 있습니다. 이것은 당신이 몇 번이고 또 할 일이있을 때 특히 유용합니다. 작은 "빈"아이콘을 클릭하면 즐겨 찾기 폴더를 지울 수 있습니다.

사용 가능한 조치와 수행 할 인수 및 연결 방법에 대한 전체 목록은 드래그 앤 드롭 참조 섹션을 참조 하십시오.

마우스 오른쪽 버튼을 사용할 수 있습니다 도구 상자의 모든 작업에서 다음 메뉴를 엽니 다.

다음과 같은 옵션이 있습니다.

  • < action > 사용 : 마우스 오른쪽 단추를 클릭 할 때 마우스 아래에 있었던 조치를 현재 이벤트 작업 공간에 추가합니다.
  • 즐겨 찾기 만들기 / 제거 : 도구 상자 상단의 즐겨 찾기 라이브러리에 작업을 추가하거나 제거합니다. 이 라이브러리는 정기적으로 사용하는 사용자 정의 조치 그룹을 작성하여 실제 라이브러리에서 조치를 찾는 데 필요한 시간을 줄입니다.
  • 즐겨 찾기 지우기 : 이것을 선택하면 즐겨 찾기 라이브러리에서 모든 작업이 제거됩니다.
  • 최근에 사용한 항목 지우기 :이 항목을 선택하면 최근에 사용한 라이브러리에서 모든 작업이 제거됩니다.
  • 모두 축소 :이 옵션은 모든 DnD ™ 라이브러리를 축소하여 작업 노드가 아닌 라이브러리 제목 만 표시되도록합니다. 각 라이브러리를 클릭하면 다시 확장 할 수 있습니다.
