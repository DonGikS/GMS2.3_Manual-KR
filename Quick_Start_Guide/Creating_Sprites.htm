스프라이트 만들기

프로그램이 무엇인지 알았 으므로 이제 GameMaker Studio 2에서 게임을 만들기 위해 사용할 수있는 여러 가지 자산 을 살펴볼 차례입니다. 우선 스프라이트를 살펴 보겠습니다. 스프라이트 는 일반적으로 프로젝트를 만들 때 가장 먼저 필요한 것 중 하나입니다. 에셋 브라우저 섹션에서 설명한 것처럼 스프라이트는 애니메이션을 적용한 후 (필요하지 않아도) 화면에 그릴 수있는 이미지입니다. 일반적으로 스프라이트는 객체와 연결되지만 코드 또는 룸 편집기 자산 레이어 에서 스프라이트를 자체적으로 그릴 수 있습니다 (자세한 내용은 나중에 룸 섹션에서 설명 합니다 ). 타일 ​​세트 로 사용할 스프라이트를 만들 수도 있지만 이후 섹션에서도 해당 옵션을 살펴 보겠습니다.

When you create a new project from the Start Page, the asset browser will only contain empty folders for the resources you may need, so you need to right-click 스프라이트 자산 폴더에서 생성 옵션을 선택합니다 (스프라이트 자산 폴더가없는 경우 마우스 오른쪽 버튼 메뉴에서 폴더를 만든 다음 스프라이트를 만들거나 자산 브라우저의 아무 곳이나 마우스 오른쪽 버튼으로 클릭하면됩니다) 그것을 만듭니다). 이것은 새로운 스프라이트 자산을 만들고 열립니다 스프라이트 편집기를 (당신을 위해 스프라이트 편집기를 열 수 없습니다, 단순히 더블 클릭을 수행하는 경우 새 자산) :

보시다시피, 창의 왼쪽 상단에는 스프라이트 이름 필드가 있습니다. 모든 스프라이트 (및 기타 모든 자산)에는 사용자 (및 GameMaker Studio 2)가 쉽게 식별 할 수 있도록 이름이 있어야합니다.이 이름은 실제로 "지시하는"ID 값을 보유하는 변수 일뿐입니다. 리소스에이 경우 스프라이트입니다. 각 스프라이트에 설명적인 이름을 지정하는 것이 가장 좋습니다. 특정 자산이 스프라이트인지 또는 오브젝트인지를 한눈에 파악할 수 있으며 많은 사람들이 " spr "문자로 접두어를 붙이거나 접미사를 붙여이를 수행합니다. 예 : " spr_Ball " 리소스 이름은 문자, 숫자 및 밑줄 기호 "_"만 사용하도록 제한되며 모든 자산의 이름은 숫자가 아닌 문자로 시작해야합니다.

이 섹션의 끝에서 설명 할 스프라이트 편집기의 다른 기능이지만 먼저 스프라이트를 그리는 방법을 설명해야합니다. 이것은 이미지 편집기 에서 수행됩니다. 이미지 편집기는 게임에서 그래픽을 작성하기위한 매우 강력한 도구이며 이미지 편집 버튼을 클릭하면 열립니다.

우리는 여기에 사용 가능한 도구에 대해 너무 깊이 들어 가지 않을 것입니다. 매뉴얼의 이미지 편집기 섹션이 있기 때문입니다.

  • 왼쪽 상단의 프레임보기 -스프라이트의 프레임이 표시되며 돋보기 아이콘을 사용하여 크기를 조정할 수 있습니다.
  • 중앙에는 그리기 캔버스가 있습니다 -현재 선택된 프레임을 그릴 준비가되었으며 마우스 가운데 버튼을 사용할 수 있습니다 이동하고 마우스 휠 확대 및 축소합니다.
  • 오른쪽 상단에는 기본 브러시 가 있습니다. 여기에서 그리기 브러시를 선택할 수 있으며 여기에는 모든 사용자 정의 브러시가 추가됩니다.
  • 오른쪽 가운데에는 색상 선택기 가 있습니다. 여기에서 왼쪽 또는 오른쪽 마우스 버튼에 사용할 색상을 선택할 수 있습니다. 동시에 하나 또는 다른 버튼으로 캔버스에 그릴 수 있기 때문입니다.
  • 그냥 우리가 우리의 도구를 가지고 색상 선택기 아래 - 이러한 브러쉬, 홍수 채우기, 마스킹 및 모양 도구를 포함하여 다양한 그리기 도구가 있습니다. 일부 도구는 대각선으로 나뉘며 클릭하여 선의 어느 쪽을 선택 하느냐에 따라 윤곽선 또는 채워진 모양을 그립니다.
  • 오른쪽 하단에는 레이어보기 가 있습니다. 다른 많은 그리기 도구와 마찬가지로 다른 레이어를 만들어 그릴 수 있으므로 아래 내용을 파괴 할 염려없이 위치와 색상 등을 실험 할 수 있습니다.

이미지 편집기에 무언가를 그려서 옵션으로 재생하는 데 시간이 걸리면 준비가되면 이미지 편집기 작업 영역을 닫고 (이미지를 저장함) 스프라이트 편집기가있는 기본 작업 영역으로 돌아갑니다. 방금 시작할 때 알아야 할 스프라이트 편집기의 몇 가지 다른 기능에 대해 살펴 보겠습니다.

가장 먼저 알아야 할 것은 스프라이트의 원점 을 설정하는 방법입니다. 원점은 단순히 방 안에서 스프라이트를 "고정"하는 데 사용될 지점이며,이 경우 중앙에 설정하려고합니다.

So we click 출발지 설정을 위해 아래에서 메뉴에서 "중간 중심"을 선택하십시오.

You'll see the blue origin marker in the preview image move to the center of the sprite. Note that you can place the origin anywhere you wish by simply clicking 미리보기 이미지에서 상단의 x / y 필드에 값을 추가하여 원점 위치를 수동으로 입력 할 수도 있습니다.

알아야 할 또 다른 중요한 사항은 스프라이트에 대한 충돌 마스크 를 설정하는 방법입니다. 충돌 마스크는 GameMaker Studio 2가 게임에서 개체의 다른 인스턴스와 충돌이 있는지 확인하기 위해 확인하는 영역이며, 기본적으로 사각형으로 설정되어 있습니다 (이는 가장 빠른 해결 방법 임). 경계 상자 의 감지 모드를 자동 (기본값) 또는 수동으로 설정할 수 있습니다. 수동을 선택하면 경계 상자 값을 왼쪽, 오른쪽, 위쪽 및 아래쪽으로 설정할 수 있습니다.

사각형 이외의 모든 유형의 충돌 마스크는 해결 속도가 느리지 만 작고 간단한 게임을 만들 때는 일반적으로 문제가되지 않습니다. 스프라이트 편집기에서 가져 오기 버튼을 클릭하여 다른 소프트웨어로 만든 스프라이트를 가져올 수도 있고, 이미지 파일을 GameMaker Studio 2 IDE로 드래그하여 스프라이트로 자산 브라우저에 추가 할 수도 있습니다. 자산 브라우저에서 스프라이트를 드래그하여 사물에 직접 추가 할 수도 있습니다.이 경우 IDE는 스프라이트를 추가 할 수있는 위치를 강조 표시합니다.

Sprite Editor 및 Image Editor에 대한 자세한 내용을 보려면 다음 매뉴얼 섹션을 참조하십시오.
