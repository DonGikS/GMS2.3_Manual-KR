객실

GameMaker Studio 2에서 만드는 모든 게임에는 최소한 하나의 공간이 필요하지만 (더 많은 것이 더 많을 수 있음), 공간은 게임을 구성하는 개체와 타일 및 다른 그래픽 리소스 자산 브라우저 에서 룸을 처음 만들면 다음과 같이 속성을 편집 할 수있는 새로운 룸 편집기 작업 공간이 표시됩니다.

기본적으로 룸 도구는 3 개의 주요 섹션과 함께 왼쪽에 배치됩니다.

  • 레이어 편집 : 방에있는 모든 레이어에 배치되어, 당신은 (나중에 더 많은)에서 선택할 수있는 다른 레이어 유형이있다. 레이어는 깊이 별로 정렬되며이 깊이는 게임 실행시 레이어 내용이 화면에 렌더링되는 순서를 정의합니다. 깊이 순서는 가장 높은 순서에서 가장 낮은 순서로, 깊이가 낮을수록 "카메라"에 가까울수록 깊이가 멀수록 멀어집니다. 예를 들어, 깊이가 -300 인 레이어는 깊이가 -100 인 레이어 위에 렌더링되고 깊이가 1000 인 레이어는 이보다 깊이가 낮은 모든 항목 아래에 렌더링됩니다. 기본적으로 GameMaker Studio 2는 레이어 편집기에서 위치에 따라 레이어 심도를 주문하며 마우스를 사용하여 레이어를 드래그하여 순서를 다시 지정할 수 있으며 GameMaker Studio 2는 심도를 재정렬하여 일치하도록 순서를 조정합니다.
  • 레이어 속성 : 방에 추가하는 각 레이어에는 고유 한 속성이 있으며 해당 속성은 레이어 유형에 따라 변경됩니다. 이 창에서 해당 속성을 변경하고 레이어 렌더링 방법을 편집 할 수 있습니다.
  • 객실 속성 : 객실에는 여러 속성이 있으며 여기에서 설정할 수 있습니다. 룸 크기, 활성화 된 카메라 뷰 포트 및 기타 몇 가지 사항을 여기에서 설정할 수 있습니다.

우리는 서로 다른 레이어 유형이 있다고 언급 했으므로, 레이어가 무엇인지, 게임을 빌드 할 때 레이어가 어떻게 사용되는지 살펴 보겠습니다.

배경 레이어 배경 레이어는 단일 색상 또는 단일 이미지로 채울 수있는 레이어입니다. 룸 내에서 이동 및 배치 할 수 있으며 여러 배경 레이어를 가질 수 있습니다. 일반적으로 이름에서 알 수 있듯이 방의 다른 모든 레이어에 대해 일정한 배경을 생성하는 데 사용됩니다. 기본적으로 새 회의실에는 항상 배경 레이어가 포함되어 있지만 필요하지 않은 경우 제거 할 수 있습니다.
인스턴스 레이어 인스턴스 레이어는 게임에 필요한 오브젝트의 모든 인스턴스를 배치하는 곳입니다. 레이어에 인스턴스를 추가하려면 클릭 자산 브라우저의 객체에서 룸 편집기 작업 공간으로 드래그합니다. 마우스 버튼을 놓으면 해당 개체의 인스턴스가 회의실에 추가됩니다. 기본적으로 새 회의실에는 항상 인스턴스 계층이 포함되어 있지만 필요하지 않은 경우 제거 할 수 있습니다.
타일 ​​맵 레이어 타일 ​​세트 를 만든 후에는 타일 ​​맵 레이어를 만들어 타일을 방에 추가해야합니다. 타일 ​​맵 레이어는 생성 한 타일 세트 자산에서 타일을 추가 할 수있는 레이어로, 타일 세트 셀 크기의 격자를 사용하도록 자동으로 설정됩니다. 각 격자 셀에 단일 타일 만 추가 할 수 있으므로 동일한 공간을 차지하기 위해 여러 타일이 필요한 경우 여러 타일 맵 레이어를 사용해야합니다.
경로 레이어 경로 레이어는 게임을 실행할 때 실제로 렌더링되지 않는 유일한 레이어입니다. 이 레이어는 실제 룸 공간 내에서 경로를 추가하거나 편집 할 수 있으므로 경로 자산을 작성하거나 편집하는 데 도움이되는 "편의"레이어에 가깝습니다.
자산 계층 때로는 멋진 그래픽 효과를 원하지만 인스턴스 사용으로 인한 오버 헤드를 원하지 않거나 그 자체를 그리는 것 이외의 다른 작업을 수행 할 필요가 없습니다. 타일 ​​맵을 사용할 수는 있지만 타일 맵이 그리드를 사용하도록 제한되고 그리드 셀당 하나의 이미지 만 사용되는 경우 약간 제한적일 수 있습니다. 이때 자산 레이어를 사용하게됩니다. 자산 레이어는 단순히 스프라이트 리소스를 가져 와서 룸에 추가 할 때 설정 한 매개 변수를 사용하여 그립니다. 객체의 인스턴스가 인스턴스 레이어에 추가되는 것과 같은 방식으로 스프라이트와 시퀀스를이 레이어에 추가 할 수 있습니다. 자산 브라우저의 스프라이트에서 룸 편집기 작업 공간으로 드래그합니다.

여기서는 룸 편집기에 대한 자세한 내용은 다루지 않겠지 만 (본 매뉴얼 의이 섹션 이 있기 때문에) 인스턴스 레이어에 인스턴스를 추가하는 방법에 대해 간략히 설명합니다. 빠른 시작 안내서의 다음 섹션.

회의실 편집기에 실례를 추가하려면 먼저 왼쪽의 도면층 편집기에서 실례 도면층을 선택하거나 존재하지 않는 경우이를 작성해야합니다. 그런 다음 자산 브라우저로 이동하여 간단히 원하는 오브젝트를 거기에서 드래그하여 원하는 위치에 놓으면 게임에 해당 오브젝트의 단일 인스턴스가 추가됩니다.

회의실에 여러 인스턴스를 배치하려면 리소스 트리에서 객체를 선택한 다음 길게 누르십시오. 리소스를 미리보고 마우스 왼쪽 버튼을 추가로 클릭하면 그런 다음 룸 레이어에 "페인트"할 수 있습니다.

룸 원점을 기준으로 X 및 Y 축 위치를 기반으로 룸 편집기에 자산을 추가 할 때 자산에 위치가 제공됩니다. 룸 원점은 (0, 0) 위치이며, GameMaker Studio 2에서 상단 렛트 코너이며 오른쪽은 + X이고 아래는 + Y입니다.

위의 예에서 인스턴스는 방의 중간 위치 (X : 512, Y : 384)에 배치됩니다. 룸 경계를 벗어나 룸 축 길이보다 길거나 룸 원점보다 작은 위치를 가질 수도 있습니다.이 경우 X / Y 좌표는 음수 값이됩니다.
