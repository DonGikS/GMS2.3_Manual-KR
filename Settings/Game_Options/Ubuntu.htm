우분투 게임 옵션

이 섹션에서는 Ubuntu (Linux) 게임 프로젝트를 컴파일하는 방법을 제어 할 수있는 다양한 옵션에 대해 간략하게 설명합니다. 다른 섹션은 다음과 같습니다.

 

일반

이 탭에서는 게임을 실행하는 동안 표시 될 게임 이름을 설정하고 연락처 이메일 및 설명 (짧고 긴)과 게임 홈페이지 또는 개발자를위한 URL을 제공 할 수 있습니다. .

게임에서 Steam 을 활성화하는 옵션도 있습니다. 이 옵션을 활성화하면 Steamworks SDK가 설치되어 있고 ( 여기 참조) 게임의 일반 옵션 에 Steam 앱 ID 를 추가했는지 확인하십시오.

마지막으로 파일 시스템 샌드 박스 비활성화 옵션이 있습니다. 이를 체크하면 파일을 저장하고로드하기위한 GameMaker Studio 2 샌드 박싱이 비활성화되어 게임이 게임을 실행하는 시스템의 어느 곳에서나 파일에 액세스 할 수 있습니다. 모든 시스템의 파일에 대해 GameMaker Studio 2 샌드 박스를 유지하는 것이 좋습니다.이 옵션을 선택하면 OS 수준 권한에 따라 일부 저장 /로드 영역을 사용하지 못할 수도 있습니다. 샌드 박스에 대한 자세한 내용은 GameMaker Studio 2 파일 시스템 페이지를 참조하십시오.

 

제도법

그래픽 옵션은 게임에서 대상 Ubuntu 시스템의 그래픽 카드를 사용하는 방법을 결정하기 위해 구성해야하는 옵션입니다. 수정할 수있는 옵션은 다음과 같습니다.

  • 전체 화면 시작 :이 옵션을 선택하면 게임이 전체 화면 모드에서 시작되고 그렇지 않으면 창이 시작됩니다. 기본적으로 해제되어 있습니다.
  • 전체 화면 전환 허용 :이 확인란을 선택하면 표준 Ubuntu OS 바로 가기를 사용하여 전체 화면에서 창으로 전환 한 후 다시 전환 할 수 있습니다. 기본적으로 해제되어 있습니다.
  • 픽셀 간 색상 보간 : 기본적으로 "매끄러운"픽셀 인 선형 보간을 켭니다. 선명한 픽셀 그래픽의 경우 꺼져 있어야하지만 알파 블렌드와 가장자리가 매끄럽고 멋진 그래픽이 있으면 기본적으로 꺼져 있습니다.
  • 커서 표시 :이 옵션이 활성화되면 기본 OS 커서가 게임에 표시됩니다. 이 기능을 비활성화하면 마우스가 게임 창 위에있는 동안 게임 프로젝트의 일부로 마우스 커서를 만들지 않으면 커서가 보이지 않습니다 (기본적으로 켜져 있음).
  • 동기화를 사용하여 테어 링 방지 : v-sync를 켜거나 끕니다. 룸 속도가 120 인 게임의 플레이어에 새로 고침 빈도가 60 인 모니터가있는 경우이 옵션을 켜면 게임 속도도 60으로 고정됩니다. 기본적으로 해제되어 있습니다.
  • 창 크기 조정 허용 :이 옵션을 선택하면 사용자가 게임 창의 크기를 변경할 수 있습니다 (이 작업을하려면 Borderless Window 옵션이 꺼져 있어야합니다). 이 옵션은 기본적으로 해제되어 있습니다.
  • 크기 조정 : 가로 세로 비율을 유지하거나 (16 : 4에서 4 : 3 회의실이 "편지함"으로) 또는 전체 크기로 전체 화면에 맞게 이미지를 늘릴 수 있습니다. 경고! 응용 프로그램 화면을 끄면 다시 켜질 때까지 Windows 게임 옵션에 설정된 모든 크기 조정 옵션이 비활성화됩니다. 자세한 내용은 응용 프로그램 표면을 참조하십시오.

 

마지막으로 텍스처 페이지 의 크기를 설정하는 옵션이 있습니다. 기본 (가장 호환되는) 크기는 2048x2048이지만 256x256에서 최대 8192x8192 사이에서 선택할 수 있습니다. 또한이 플랫폼에 대한 텍스처 페이지를 생성 한 다음 창을 열어서 모양을 볼 수있는 미리보기 버튼이 있습니다. 텍스처 페이지가 어떻게 구성되어 있는지 확인하고 텍스처 페이지가 필요 이상으로 커지는 것을 방지하려는 경우 매우 유용 할 수 있습니다. 텍스처 페이지에 대한 자세한 내용은 여기 를 참조 하십시오 .

참고 : 텍스처 페이지의 크기가 클수록 게임 사양이 낮은 PC에서 게임 호환성이 떨어집니다.

 

 

이미지

여기서 게임에 스플래시 화면 을 추가하고 사용할 게임 아이콘 을 설정할 수 있습니다. 아이콘은 .png 형식의 64x64 픽셀 이미지 여야하며 스플래시 화면은 게임의 첫 번째 방 (또는보기 포트)과 크기가 같아야하며 .png 형식 이어야 합니다. 게임이로드되는 동안 스플래시 화면이 표시되며 스플래시 화면 사용 옵션 (기본적으로 해제되어 있음)을 선택하여 활성화 또는 비활성화 할 수 있습니다.

GameMaker Studio 2에는 게임이 컴파일되는 모든 다른 대상 플랫폼에 필요한 모든 이미지를 자동으로 만드는 데 사용할 수있는 프로젝트 이미지 생성기 도구가 있습니다. 이 도구를 사용하는 경우 필요한 이미지가되도록 만들어진 이미지를 수정해야합니다.
