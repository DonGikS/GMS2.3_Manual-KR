오디오 그룹

오디오 그룹 관리자 는 IDE의 도구 메뉴 에서 사용할 수 있습니다. 여기에서 오디오 그룹을 추가 , 삭제 및 이름 변경 하고 플랫폼 내보내기 옵션을 설정할 수 있습니다. GameMaker Studio 2를 사용하면 각 오디오 리소스 (사운드 효과 및 음악)를 서로 다른 오디오 그룹에 할당하여 한 번에 재생되는 사운드 수를 최적화하고, 어떤 플랫폼을보다 효과적으로 제어 할 수 있습니다 에 내보내집니다.

To define an audio group you need to click 새로 추가 버튼을 클릭하면 이름을 지정할 수있는 새 그룹이 생성됩니다. 그룹을 변경하려면 클릭 드롭 다운 메뉴에서 편집 할 메뉴를 선택하십시오.

참고 : 사운드 편집기 에서 사운드에 대해 스트리밍 오디오를 선택하면 스트리밍 사운드가 다른 사운드와 같은 방식으로 패키지되지 않고 사운드가 완전히 제어 될 수 있으므로 오디오 그룹에 사운드를 할당 할 수 없습니다. 코드를 사용하여로드, 재생 및 언로드됩니다.

To add a sound to an audio group, you need to open the Sound Editor for that sound and then select the group from the option you will find there, or alternatively you can add assets to an audio group via the right mouse button 자산 브라우저의 메뉴. 하나 이상의 사운드 자산 (또는 전체 자산 그룹 폴더)을 선택한 다음 RMB 메뉴를 사용하여 그룹에 사운드를 추가 할 수 있습니다. 오디오 그룹 편집기 하단에있는 리소스 추가 버튼을 클릭하여 자산을 그런 식으로 추가 할 수도 있습니다.

사용자 정의 오디오 그룹에 사운드가 추가되면이 창을 사용하여 해당 사운드를 내보낼 플랫폼을 선택적으로 선택할 수 있습니다. 기본적으로 모든 사운드의 품질은 최고이지만 HTML5의 경우 (예를 들어) 더 적은 메모리를 사용하도록 낮은 품질의 사운드 파일을 사용하려고합니다. 이 경우 품질이 낮은 파일을 GameMaker Studio 2에 추가 한 다음 새 오디오 그룹에 할당하십시오. 그런 다음 드롭 다운 메뉴에서 해당 오디오 그룹을 선택하고 HTML5 로만 내보내도록 설정하고 고품질 오디오 그룹 내보내기 옵션에서 HTML5 내보내기를 제거합니다. "기본"오디오 그룹의 내보내기 옵션을 변경할 수 없으며 최종 게임 패키지를 빌드 할 때 항상 사용 가능한 모든 플랫폼으로 내보내집니다.

오디오 그룹을 정의하고 해당 그룹에 사운드를 할당하면 그룹을 선택할 때 왼쪽 목록에서 해당 그룹을 볼 수 있습니다. 항상 "기본"오디오 그룹을 사용할 수 있으며이 그룹 내에있는 모든 사운드는 항상 모든 플랫폼의 게임 패키지에 포함되며 시작시 메모리에 모두로드됩니다 ( "스트리밍 됨"으로 표시되지 않은 경우) (사운드 편집기 속성)에서 사용자 정의 오디오 그룹을 만들 때 추가 된 파일은 audio_group_load () 함수를 호출 할 때까지 메모리에로드되지 않습니다 . 이 방법으로 언제든지 메모리에있는 오디오를 제어 할 수 있습니다. 모든 오디오 그룹 기능 에 대해 오디오 그룹 ID 값을 제공해야합니다. 이것은 단순히 오디오 그룹에 지정한 이름입니다.

오디오 그룹도 구성 설정에 연결됩니다. 구성을 선택할 때 오디오 그룹 창을 열고 해당 구성의 오른쪽에서 내보내기 대상을 선택한 다음 구성을 변경하면 이러한 출력 대상이 변경됩니다. 구성별로 사운드 리소스를 다른 그룹으로 설정할 수 없으며 지정된 그룹의 내보내기 대상 만 설정할 수 있습니다.
