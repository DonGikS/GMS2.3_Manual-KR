게임 옵션

게임 옵션은 메인 옵션과 플랫폼 설정의 두 섹션으로 나뉩니다. 기본 옵션은 일반 옵션으로 시작합니다. 일반 옵션은 실행하도록 설계된 플랫폼에 관계없이 게임에 영향을 줄 특정 사항을 설정할 수 있습니다.

이 창에는 다음과 같은 섹션이 있습니다.

  • DnD ™를 기본 스크립트 유형으로 사용 :이 옵션을 선택하면 모든 새 스크립트 및 개체 이벤트가 코드 편집기가 아닌 DnD ™ 편집기를 사용하여 열립니다.
  • 초당 게임 프레임 :이 값은 GameMaker Studio 2가 실행 중 게임을 유지하려고하는 초당 프레임 수의 제한을 설정합니다. 이 값은 게임 루프가 1 초 안에 실행되는 횟수 (단일 게임 루프는 단일 "프레임"으로 간주 됨)를 나타내며 기본값은 60으로 설정되지만 30도 자주 사용됩니다. 어떤 룸 속도가 이상적인 지에 대한 많은 논증이 있지만 실제 답변은 제작중인 게임 유형, 룸 크기, 인스턴스 수 및 코드 강도에 따라 다릅니다. 예를 들어, 넓은 방과 30의 방 속도가있는 경우,보고있는 영역이 클수록 게임이 "고르지 않은"것처럼 보일 수 있지만, 눈이 낮은 방 속도를 더 쉽게 인식 할 수 있지만 게임이 처리량이 많은 경우 실내 속도를 120으로 설정하면 fps가 떨어질 때 게임이 "지연"될 수 있습니다. 따라서이 값에주의를 기울이고 항상 게임에 필요한 값으로 조정하십시오.
  • 기본 그리기 색상 : 코드를 사용하여 색상을 설정하지 않은 경우 텍스트 또는 포럼 (및 기타 항목)을 그리기위한 기본 색상입니다.
  • 룸 영역 외부의 기본 그리기 색상 : 게임 룸 외부 영역의 배경을 지우는 데 사용되는 기본 색상입니다.
  • 프로젝트 이미지 생성 : 클릭하면 프로젝트 이미지 생성기 도구가 열립니다.
  • 프로젝트 시작 날짜 : 프로젝트가 시작된 날짜입니다.
  • 프로젝트 사용 시간 : 프로젝트를 시작한 후 경과 한 시간입니다.
  • 프로젝트 GUID : GameMaker Studio 2 내의 각 프로젝트에 대해 생성 된 고유 식별자입니다.이를 변경하려면 Create New GUID 버튼을 클릭하여 새 GUID를 생성 할 수 있습니다.
  • 별도의 텍스처 페이지를위한 밉맵 생성 :이 옵션은 GameMaker Studio 2가 스프라이트 에디터의 텍스처 설정에서 텍스처 페이지 분리 옵션을 표시 한 스프라이트에 대한 밉맵을 생성하게합니다. 밉 매핑을 활성화하면 스프라이트에 대한 원본 텍스처 페이지의 크기가 조정되고 텍스처 파일 내에서 여러 해상도로 필터링됩니다. 이것은 일반적으로 꺼져 있지만 뷰를 사용하여 게임 룸의 넓은 영역을 확대 또는 축소하거나 3D 게임으로 작업하는 경우이를 활성화 한 다음 Mipmapping 기능을 사용하여 게임은 다양한 밉맵 레벨을 처리합니다. 스프라이트에 개별 텍스처 페이지를 사용하는 것으로 플래그를 지정하지 않은 경우에도 텍스처 그룹 관리자에서 텍스처 그룹별로 밉 매핑을 활성화 할 수 있습니다. 이 옵션은 기본적으로 해제되어 있습니다.
  • Steam ID : Steam을 통해 배포 할 게임을 만들려면 여기에 Steam 앱 ID를 제공해야합니다. 이 옵션을 사용할 때 플랫폼 환경 설정에서 Steamworks SDK에 대한 경로를 추가했는지 확인하고 Steamworks 대시 보드에 모든 것이 올바르게 설정되어 있는지 확인하십시오.
  • 소스 제어 사용 : 프로젝트를 SCM (소스 제어 관리)하에 두려면이를 확인해야합니다. 기본적으로 꺼져 있습니다 (소스 제어에 대한 자세한 내용은 여기 참조).
  • Author : 게임 저자 (또는 퍼블리셔 또는 스튜디오)의 이름입니다.
  • Project Last Changed :이 섹션에는 프로젝트가 마지막으로 변경된 시간 / 날짜 스탬프가 표시됩니다 (편집 할 수 없음).
  • Spine 라이센스 수락 : 여기에서 Spine 런타임 라이센스가 표시되도록 확인란을 선택하고 동의 할 수 있습니다. 이를 통해 게임에서 스파인 스프라이트를 사용할 수 있습니다 (이 라이센스는 프로젝트에 스파인 스프라이트를 처음 추가하려고 할 때 표시되며 사전에 라이센스를 확인하지 않은 상태입니다).

또한 특정 플랫폼에서 리더 보드 또는 인앱 구매와 같은 확장 된 기능을 사용하려면 GameMaker Studio 2에서 특정 애드온을 프로젝트의 확장으로 설치해야 할 수도 있습니다. 이것은 애드온 섹션에서 처리됩니다.

" 다운로드 "버튼을 클릭하면 확장 프로그램이 다운로드되어 현재 프로젝트에 설치되어 바로 사용할 수 있으며 삭제 버튼을 클릭 할 수 있습니다 필요한 경우 프로젝트에서 추가 기능 확장을 제거합니다.

또한 자산 브라우저 에는 보유하고있는 라이센스에 사용 가능한 대상 플랫폼에 따라 다른 게임 옵션을 사용할 수 있으며 모든 게임 옵션을 사용할 수있는 것은 아닙니다.

다음과 같은 플랫폼 별 게임 옵션이 있습니다.

  • 윈도우
  • 맥 OS
  • 우분투
  • Windows UWP
  • HTML5
  • 아마존 파이어
  • 기계적 인조 인간
  • iOS
  • tvOS
