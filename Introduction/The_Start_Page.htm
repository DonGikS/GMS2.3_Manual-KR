시작 페이지

GameMaker Studio 2를 처음 열면 YoYo 계정 에 로그인하라는 메시지가 표시됩니다 (또는 아직 등록하지 않은 경우 등록 하십시오 ). 일단 등록하고 로그인하면 메인 GameMaker Studio 2 IDE 가 시작 페이지에서 열립니다 :

맨 위에는 다음 매뉴얼 섹션에 설명 된 일반 IDE 메뉴가 있습니다.

  • 파일 메뉴
  • 편집 메뉴
  • 빌드 메뉴
  • 윈도우 메뉴
  • 도구 메뉴
  • 마켓 플레이스 메뉴
  • 레이아웃 메뉴
  • 도움말 메뉴

현재 초점을 둔 창에 따라 상단 메뉴 표시 줄에 상황에 맞는 메뉴 옵션이 나타납니다. 예를 들어, 이미지 편집기 창에 초점이있는 경우 " 이미지 ", " 보기 "및 " 효과 "에 대한 추가 메뉴 항목이 있습니다. 이러한 추가 메뉴 항목은이를 생성 한 작업 공간 또는 창에 대한 매뉴얼의 관련 섹션에 설명되어 있습니다.

나머지 시작 페이지는 다음과 같습니다.

  • 홈 -시작 페이지의 다른 섹션을 탐색 할 때 마우스 왼쪽 단추를 클릭하십시오. 이것으로 간단히 메인 페이지로 돌아갑니다.
  • 버전 세부 정보 -이 섹션에서는 현재 런타임 버전뿐만 아니라 사용중인 현재 IDE 버전에 대한 세부 정보를 제공합니다. 또한이 섹션에서 IDE 또는 런타임에 사용 가능한 변경 사항에 대한 알림을받습니다.
  • 최근 프로젝트 -시작 페이지 왼쪽에 열 수있는 이전 프로젝트 목록이 표시됩니다. 그 위에 마우스를 올리면 전체 프로젝트 이름과 경로를 볼 수 있고 마우스 왼쪽 버튼을 클릭 할 수 있습니다 프로젝트가 열립니다.
  • 소셜 -시작 페이지의 상단 표시 줄에는 쉽게 탐색 할 수있는 여러 개의 링크가 있습니다. 그것들은 아래와 같습니다 : YoYo Games- 주요 YoYo Games 웹 사이트로 이동합니다
  • 소셜 미디어 아이콘 -GameMaker Studio 2에 대한 뉴스를 최신 상태로 유지할 수있는 다양한 YoYo Games 소셜 미디어 사이트로 이동합니다.
  • 프로젝트 옵션 -프로젝트 섹션에서 프로젝트를 생성, 열거 나 가져올 수 있습니다. 이에 대해서는이 페이지에서 자세히 설명합니다.
  • 정보 -시작 페이지의 정보 섹션에서는 YoYo Games Marketplace 를 탐색하고 YoYo Games 팀이 만든 자습서 및 데모를 탐색하고 열어 GameMaker Studio 2의 특정 측면을 보여줄 수 있습니다 (자세한 내용은 아래 참조).

    새로운 프로젝트 시작

    새로 만들기 버튼을 클릭하여 새 프로젝트를 생성하거나 열기 버튼을 클릭하여 기존 프로젝트를 열거 나 가져 오기 버튼을 클릭하여 GameMaker : Studio 1.4 GMX 프로젝트 파일을 열거 나 압축 된 YYZ GameMaker Studio 2 프로젝트 파일을 열 수 있습니다. 열기와 가져 오기 모두 파일 탐색기를 열어 필요한 프로젝트 파일을 찾습니다.

    참고 : GameMaker Studio 2는 GameMaker : Studio 1.4 프로젝트와 완전히 호환되지 않지만 호환성 스크립트를 사용하여 자동으로 사용되지 않는 기능이 다시 생성되었으므로 가져온 1.4 프로젝트는 계속 실행되어야합니다. 가능한 문제와 GML에 대한 변경 사항에 대한 자세한 내용은 도움말 센터 문서 GMS 1.4 프로젝트를 GameMaker Studio 2로 이식 및 사용되지 않는 기능 설명서 섹션을 참조하십시오.

    새 프로젝트를 만들려면 New (새로 만들기)라고 표시된 버튼을 클릭하여 다음 화면으로 이동하십시오.

    GameMaker Studio 2를 사용하면 어떤 유형의 프로젝트를 만들 것인지 지정할 수 있으며 코딩 지식이나 습관에 따라 선택하고 선택할 수 있습니다. 사용 가능한 옵션은 다음과 같습니다.

    • 새로운 드래그 앤 Dropâ - "¢ 프로젝트는 이렇게하는 새 프로젝트 생성됩니다 개체 편집기 ¢ 인터페이스 (및 허가 코드가 다른 편집자)가 GameMaker 스튜디오 2 드래그 앤 드롭 â을 사용하도록 설정됩니다." 끌어서 놓기 (DnD ™)에 대한 자세한 내용은 여기 를 참조 하십시오 . 이 인터페이스는 독점적 인 DnD ™ 인터페이스가 아니며 마우스 오른쪽 버튼을 사용하여 DnD ™와 GML 코드 간을 전환 할 수 있습니다. .
    • New GameMaker Language Project : GML (GameMaker Language)을 사용하여 기본 프로그래밍이 수행되는 프로젝트를 만듭니다. GML에 대한 자세한 내용은 여기를 참조하십시오 . 이 인터페이스는 독점적 인 GML 인터페이스가 아니며 마우스 오른쪽 버튼을 사용하여 GML 코드와 DnD ™간에 전환 할 수 있습니다. .

    튜토리얼 및 데모

    튜토리얼 또는 데모 버튼을 클릭하면 소유 한 튜토리얼 / 데모 및 다운로드 할 수있는 공식 튜토리얼 / 데모를 표시하는 화면으로 이동합니다.

    공식 튜토리얼 또는 데모를 다운로드하려면 원하는 아이콘을 클릭하기 만하면 상단의 " 소유 "섹션에 추가됩니다 (이 전에 마켓 플레이스에 로그인해야 할 수도 있습니다) 할 수 있습니다). 튜토리얼 또는 데모가 "소유"섹션에 있으면, 마우스를 가져 가서 설치하거나 마켓 플레이스에서 정보를 볼 수 있습니다.

    튜토리얼 또는 데모가 설치되면, 간단히 클릭하여 튜토리얼 또는 데모를 열면 IDE를 시작하기 전에 튜토리얼 프로젝트를 저장할 위치를 제공하라는 메시지가 표시됩니다. 설치된 튜토리얼 및 데모에는 추가 아이콘이 있습니다. 컴퓨터에서 프로젝트를 제거하는 데 사용할 수 있습니다 ( "소유"목록에서 자습서 또는 데모는 제거되지 않음).

    시작 페이지의 튜토리얼 / 데모 섹션에는 자산을보다 잘 사용하고 탐색하는 데 도움이되는 몇 가지 버튼이 있습니다.

    이것을 클릭하면 마켓 플레이스에서 프로젝트 목록을 새로 고침하여 업데이트 된 것이 있는지 확인하고 최신 버전을 얻을 수 있습니다.
    이 버튼을 클릭하면 소유 한 튜토리얼 또는 데모와 설치 한 데모간에보기가 전환됩니다.

    마켓 플레이스에서도 타사 튜토리얼 및 데모를 찾을 수 있으며, 구매 및 계정에 추가 한 후에는 해당 페이지의 "소유"섹션에도 나타납니다.
