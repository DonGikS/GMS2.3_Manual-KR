서비스 리스팅

마켓 플레이스에 자산이있는 것 외에도 서비스 를 만들 수도 있습니다. 계속하기 전에 서비스 제안을 제출하기 전에 아래 설명 된 단계를 완료해야하므로 아래에 나와있는 동반자 페이지를 읽으십시오.

  • 게시자 계정 만들기

서비스는 개인 또는 회사가 타사 서비스를 제공 할 수있는 YoYo Games Marketplace의 하위 섹션입니다. 예를 들어, 광고 API 또는 GameMaker Studio 2 사용자에게 제공하려는 유사한 것을 보유한 회사 인 경우 여기에서 서비스를 제공 할 수 있습니다. 마찬가지로, 당신이 사람들을 보여줄 수있는 포트폴리오를 가진 음악 가나 예술가 또는 프로그래머라면 일을 위해 자신을 제안하고자한다면 여기에서도 서비스를 제안 할 것입니다.

중대한! YoYo Games는 계약자와 계약자 간의 문제에 대해 책임을지지 않으며 분쟁이있을 경우 중재하거나 개입하지 않습니다.

마켓 플레이스 메인 페이지 , 상단 "카테고리"메뉴 또는 오른쪽 메뉴에서 서비스 패널을 선택하여 서비스 패널에 액세스 할 수 있습니다.

녹색 화살표를 클릭하면 이용 가능한 서비스 카테고리 ( 수익 창출 또는 광고 등 )를 표시하는 추가 하위 메뉴가 열리거나 서비스 자체를 선택하여 모든 서비스를 볼 수 있습니다.

제공된 서비스 목록에서 서비스 중 하나를 클릭하면 서비스 개요로 이동하여 제공되는 서비스에 대한 모든 정보와 공급자에게 연락하거나 서비스에 가입하는 링크 등이 포함됩니다. .. 또한 서비스에 필요한 추가 리소스에 대한 추가 링크 (예 : 리더 보드 및 업적 서비스를 제공하는 사이트의 SDK 자산에 대한 링크)가있을 수 있습니다.

마켓 플레이스에 공지 된 서비스 제공 업체에 연락하기 전에 다른 사용자가 제공 한 리뷰 (있는 경우)를 확인하고 제공된 EULA를 읽고 지정된 서비스 약관을 이해해야합니다. YoYo Games는 최종 사용자와 특정 서비스 간의 거래에 대해 책임을지지 않으며 사람들이 광고 할 수있는 편리한 장소 만 제공합니다 .

자신의 서비스 목록을 설정하려면 먼저 다음을 수행해야합니다.

  • Marketplace 게시자 계정 페이지를 설정하십시오 (무료 30 일 평가판 라이센스 만있는 경우 게시자 계정을 만들 수 없음).
  • 게시자 이름 및 이메일과 같은 게시자 ID에 대한 기본 정보를 입력하고 제시된 자산 유형 목록에서 "기타"를 선택해야합니다.

완료되면 새 계정을 활성화 할 수있는 링크가 포함 된 확인 이메일이 전송됩니다. 확인 이메일을 받으면 서비스 목록을 설정할 준비가되었습니다. 게시자 창의 왼쪽에서 서비스 메뉴 항목을 클릭하면이 작업을 수행 할 수 있으며, 그 후에 계속하기 전에 동의해야하는 서비스 약관 이 표시됩니다. 수락하면 현재 제공되는 모든 서비스가 나열된 페이지로 이동합니다. 새로 만들려면 " 새 서비스 추가 "라고 표시된 버튼을 클릭하십시오.

이 페이지에서는 추가하는 서비스 유형을 정의하고 제공하는 내용 및 기타 관련 정보를 제공합니다. 필요한 정보는 다음과 같습니다.

  • 기본 카테고리 : 귀하가 제공하는 서비스 유형입니다. 기본 카테고리를 설정 한 후에는 마켓 플레이스를 검색 할 때 사람들이 적절한 서비스를 찾는 데 도움이되는 추가 하위 카테고리 (사용 가능한 경우)를 계속 선택할 수 있습니다.
  • 표시 이름 : 서비스 페이지에 표시 될 서비스 이름 (최대 255 바이트)입니다.
  • 설명 : 서비스, 기능 및 가능한 용도, 추가 문서 링크 등을 설명하는 간단한 설명 (2000 단어 미만)입니다.
  • 키워드 : 마켓 플레이스에서 서비스를 찾기 위해 검색 범위를 좁히는 데 사용할 수있는 하나 이상의 단어입니다. 이 키워드는 제공되는 서비스와 관련이 있어야합니다!
  • 관련 자산 : 서비스에 추가 자산 (GMZ, SDK 등)이 필요한 경우 일반 마켓 플레이스 자산 목록 을 생성하고 여기에 연결할 수 있습니다. 자산 이름을 제공 한 다음 나타나는 목록에서 자산 이름을 선택해야합니다. 자산이 이전에 게시되어 있어야합니다.
  • 지원 이메일 : 제공되는 서비스에 대해 사람들이 귀하에게 연락하기를 원하는 경우 귀하에게 연락 할 수있는 이메일 계정입니다.
  • 지원 웹 사이트 : 사용자가 서비스에 대한 지원을받을 수있는 URL (예 : FAQ, 포럼, 연락처 등)
  • 사용자 정의 EULA : 사용자가 제공되는 서비스에 대해 작성된 특정 EULA를 읽을 수있는 URL (이를 지정하지 않으면 귀하의 서비스는 여기 에있는 일반적인 YoYo Games EULA에 속함).
  • 개인 정보 보호 정책 URL : 서비스에 장치 또는 사용자 데이터가 필요한 경우 또는 어린 이용으로 설계된 경우 개인 정보 보호 정책에 대한 링크를 제공해야합니다.
  • 마케팅 URL : 사용자가 서비스, 게시자에 대한 추가 정보를 얻을 수있을뿐만 아니라 비디오, 스크린 샷 및 기타 관련 마케팅 자료를 볼 수있는 URL입니다.

마지막으로 마켓 플레이스에서 서비스를 올바르게 마케팅하려면 여러 이미지를 제공해야합니다. 아이콘 (200x200 PNG ) 및 기능 그래픽 (1140x360 PNG ), 적어도 하나 이상의 스크린 샷 ( PNG 여야하지만 다음 크기로 제작할 수 있음) ), 마지막으로 YouTube 또는 Vimeo 의 비디오를 추가 할 수 있습니다. 리스팅 자산을 설정하는 방법에 대한 자세한 정보 및 권장 사항은 마켓 플레이스 스타일 안내서를 참조하십시오.

이러한 세부 사항을 모두 입력했으면 이제 "저장"을 클릭하면 게시자 대시 보드의 일반 서비스 목록으로 돌아갑니다. 이제 새 서비스가 " 검토 중"상태로 표시됩니다.

서비스에 SDK가 있거나이를위한 확장을 작성하려는 경우 마켓 플레이스 자산 작성 기사에서이를 수행하는 방법을 찾을 수 있습니다. 이 작업을 수행 한 후에는 자산 목록의 " 관련 자산 "섹션에 자산 패키지의 URL을 추가하십시오.

마켓 플레이스에 서비스 제안을 처음 제출할 때 YoYo Games는 해당 서비스 제안을 검토하여 최소 진입 레벨 기준을 충족하는지 확인해야합니다. 이것은 다음을 의미합니다.

  • 스크린 샷, 아이콘 및 기능 그래픽은 명확해야하며 제공되는 서비스와 관련이 있어야합니다.
  • 서비스에 대한 설명은 연락을받은 후에 제공 한 서비스와 일치해야합니다.
  • 서비스에 대한 귀하의 설명은 일관되고 상세하며 잘 작성되어야합니다.
  • 설명 또는 목록 페이지의 모든 링크가 정확하고 관련되어 있습니다.
  • 귀하가 제공하는 서비스는 허용되는 자료에 대한 규칙 및 지침에 해당하는 것입니다 (스크린 샷 및 아이콘 포함)

검토 프로세스는 하루나 이틀이 걸릴 수 있지만 완료되면 서비스가 마켓 플레이스 서비스 페이지에 공개됩니다. 검토에 실패한 경우 이메일로 알려 드리며, 실패한 이유 또는 검토 프로세스를 수행하기 위해 무엇을 할 수 있는지 잘 모르겠 으면 헬프 데스크에 문의하십시오. 그러면 누군가가 귀하에게 올바른 도움을 줄 것입니다.
