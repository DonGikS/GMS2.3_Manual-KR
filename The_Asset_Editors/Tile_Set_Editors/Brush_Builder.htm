타일 ​​세트 브러시 빌더

룸 편집기에서 타일을 타일 맵 레이어에 "페인트"할 때 기본적으로 단일 타일을 선택하고 해당 타일로 페인트합니다. 그러나 타일 세트는 거의 항상 전체 영역 또는 항목을 형성하기 위해 서로 다른 방식으로 함께 섹션을 갖도록 설계되었습니다. 예를 들어, RPG 타일 세트에는 사용 된 타일 수에 따라 나무 및 건물과 같이 더 크거나 작은 피처를 만들기 위해 연결할 수있는 가로 피처 타일이있을 수 있습니다. 이제 룸 레이어에 이와 같은 여러 기능을 배치하려면 타일을 변경하기 위해 여러 번 앞뒤로 이동해야하므로 워크 플로에 적합하지 않습니다. 이 문제를 해결하기 위해 GameMaker Studio 2를 사용하면 타일 세트 편집기에서 타일 브러시 를 만들 수 있습니다. 브러시 빌더 버튼에서 :

브러시 빌더 에서 왼쪽에 원래 타일 세트가 있고 오른쪽에 빈 "캔버스"가 있습니다. 이제 왼쪽에서 타일을 선택하고 오른쪽에서 페인트하여 룸 편집기에서 사용할 수있는 사용자 정의 "브러시"를 만들 수 있습니다. 마우스 왼쪽 버튼을 클릭 한 상태로 유지할 수 있습니다. 그런 다음 타일 세트를 드래그하여 브러시 캔버스에 페인트 할 여러 타일을 선택합니다. / + 특정 개별 타일을 추가하거나 길게 누르기 + 특정 타일을 제거합니다.

다음 이미지는 단일 타일 세트로 만든 세 가지 사용자 지정 브러시의 예인 앞에서 언급 한 가로 방향 기능을 보여줍니다.

오른쪽에는 이미지에서 주황색 상자로 강조 표시된 3 가지 기능이 있습니다. 각 지형지 물 사이에 하나의 타일 간격을 두는 방법에 주목하십시오. 이는 터치하는 타일 그룹이 룸 편집기에서 단일 브러시로 취급 되므로 각 타일이 우리가 만들고 싶은 독특한 브러쉬. 브러쉬를 만드는 동안 마우스 왼쪽 버튼으로 페인트 마우스 오른쪽 버튼으로 삭제 . 타일 ​​시트 또는 브러시 캔버스를 사용하여 확대 / 축소 할 수도 있습니다. / 그리고 마우스 휠 마우스가 창 중 하나 위에 있거나 + 또는 가운데 마우스 버튼 그리고 끌기. 편집기 창의 도구 상자 에서 확대 / 축소 도구를 사용하여 격자 가시성 및 색상을 토글 할뿐만 아니라 창의 확대 / 축소 레벨을 변경할 수도 있습니다.

오른쪽 상단에서 현재 선택한 도구를 볼 수 있으며 페인트하려는 브러시의 크기를 설정할 수도 있습니다. 기본 크기는 1이며 단일 타일이지만 더 높은 값으로 설정하면 아래 이미지와 같이 선택한 타일로 구성된 더 큰 브러시를 반복하여 페인트하고 지우는 것이 가능합니다.

툴박스는 타일 세트 편집기에서 다양한 작업에 사용할 툴을 선택할 수있는 곳으로, 자동 타일 라이브러리에 정의 된 항목이 있는지 여부에 따라 다릅니다. 각 도구에 대한 간략한 개요가 아래에 제공됩니다 (룸 편집기에서 타일 레이어를 선택한 경우이 도구 상자가 추가 작업 도구와 함께 룸 작업 공간 상단에 표시됨).

연필 이것은 연필 도구입니다. 선택한 타일을 사용하여 마우스 왼쪽 버튼으로 브러쉬 창에서 페인트 마우스 오른쪽 버튼으로 지울 수 있습니다 .
지우개 지우개 도구를 사용하면 마우스 왼쪽 버튼을 사용할 수 있습니다 브러쉬 창에서 지정된 타일을 지우려면 기본적으로이 모든 작업은 타일 인덱스를 0으로 설정하는데, 이는 예약 된 "빈"타일입니다.
선택 이것은 선택 도구로서, 작업 할 브러쉬 창의 영역을 정의하는 데 사용할 수 있습니다. 마우스 왼쪽 버튼을 클릭 할 수 있습니다 마우스를 끌어 직사각형 영역을 만들거나 / + 여러 선택을 추가하고 + 선택의 일부를 지우십시오. 전체 선택을 지우려면 . 브러쉬 창의 영역을 선택한 경우 나머지 도구 (연필, 뒤집기, 회전 등)는 선택한 영역 내에서만 작동합니다. 창에서 타일을 선택한 다음 복사 ( / + " C ") 컷 ( / + " X ") 및 붙여 넣기 ( / + " V ")를 사용하면 도구가 연필로 바뀌고 "페인트"할 수 있습니다.
자동 타일 이 도구를 클릭하면 자동 타일링 페인트 스타일이 활성화됩니다. 이 기능이 활성화되면 Autotile Libraries 탭에서 타일을 선택한 다음 Brushes 창에 페인트하면 GameMaker Studio 2는 자동 타일 탭을 올바르게 설정 한 경우 주변 타일과 일치하도록 자동으로 타일을 변경합니다. 자동 라이브러리의 일부가 아닌 타일 세트에서 타일을 선택하면 그리기 도구가 표준 연필 도구로 재설정됩니다.
튀기다 마우스 왼쪽 버튼으로 뒤집기 도구 클릭 (또는 키보드 단축키 " X ")를 사용하면 그리기 도구를 변경하지 않고 수평 축을 따라 현재 그리기 위해 선택된 타일을 뒤집습니다 (그리기 위해 사용자 정의 브러시가 선택된 경우 전체 브러시가 뒤집 힙니다). 그릴 타일을 선택하지 않았지만 브러시 창에서 타일 그룹을 선택한 경우 뒤집기 도구는 선택한 타일을 뒤집습니다.
거울 마우스 왼쪽 버튼으로 미러 도구 클릭 (또는 키보드 단축키 " Y ")를 사용하면 그리기 도구를 변경하지 않고 현재 세로 축을 따라 그리기 위해 선택된 타일을 미러링합니다 (그리기 위해 사용자 정의 브러시가 선택된 경우 전체 브러시가 미러링됩니다). 그릴 타일을 선택하지 않았지만 브러시 창에서 타일 그룹을 선택한 경우 미러 도구는 선택한 타일을 미러링합니다.
회전 마우스 왼쪽 버튼으로 회전 도구 클릭 키보드 단축키 " Z "를 사용하면 그리기 도구를 변경하지 않고 현재 그리기 위해 선택한 타일을 시계 방향으로 90도 회전합니다 (그리기 위해 사용자 정의 브러시가 선택된 경우 전체 브러시가 회전합니다). 그릴 타일을 선택하지 않았지만 브러시 창에서 타일 그룹을 선택한 경우 회전 도구는 선택한 타일을 회전시킵니다.

도구 아래에는 현재 타일 세트 이미지를 사용하여 만든 자동 타일 또는 애니메이션 타일 을 선택하기위한 두 가지 섹션이 있습니다. 타일 ​​세트에 사용되는 단일 스프라이트에는 여러 개의 단일 셀 이미지가있을 수 있으며, 애니메이션 또는 자동 타일 편집기에서 결합하여 이러한 섹션에 표시되고 함께 사용할 수있는 사용자 정의 브러시를 만들 수 있습니다. 일반 정적 타일을 사용하여 브러시를 만듭니다 (라이브러리 또는 기본 타일 세트에서 선택했는지 여부에 관계없이 애니메이션 타일이 애니메이션됩니다).

필요한 모든 브러쉬를 설정 한 다음이를 사용하여 룸 편집기 내의 타일 맵 레이어에 타일을 페인트 할 수 있습니다.
