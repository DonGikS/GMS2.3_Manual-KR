이미지 편집기

이미지 편집기 에서 직접 스프라이트를 만들거나 이전에 가져 오거나 만든 스프라이트를 편집 할 수 있습니다. 다른 이미지 도구를 사용했다면 친숙 할 것입니다. 그러나 GameMaker Studio 2 이미지 편집기를 사용할 때주의해야 할 몇 가지 사항이 있습니다.

  • 여러 프레임을 만들 때 애니메이션이 적용되는 프레임을 그릴 수 있습니다! 재생 버튼을 누르면 프레임 컨트롤 막대에서 브러시를 선택하고 편집기 창에서 그리기를 시작하면 편집기에서 프레임에 애니메이션이 적용되고 그리는 내용이 해당 순간에 해당 프레임에 추가됩니다. 원하는 것을 얻을 때까지 다른 프레임 속도로 실험 할 수 있지만, 이것이 작동하는 방식에 익숙해지면 동적 스프라이트와 효과를 즉시 만들 수 있습니다.
  • 기본 작업 공간 창에서 이미지 편집기 탭을 자체 창으로 끌어 올 수 있습니다. 예를 들어 여러 모니터를 사용할 때 편리합니다.
  • 예를 들어, 복사 기능을 사용하여 선택한 이미지를 복사하면 복사 된 선택이 브러시 목록에 추가됩니다. 즉, 이미지의 일부를 선택한 다음 나머지 페인트 도구에서 이미지를 사용하여 선, 다각형을 그리거나 자유롭게 페인트 할 수 있습니다. 브러시를 스케일링하고 색상과 혼합 할 수도 있습니다. 붙여 넣기 기능은 복사 한 항목이 브러시 목록에 추가되므로 편집기에서 내부적으로 사용하기위한 것이 아니므로 "붙여 넣기"를 위해 선택할 수 있습니다. 대신 붙여 넣기는 단순히 외부 프로그램에서 무언가를 캡처하여 이미지 편집기에 붙여 넣기위한 것입니다.
  • 편리한 레이어 편집기 덕분에 각 스프라이트에 대해 여러 레이어를 가질 수 있습니다. 이 레이어는 더 복잡한 이미지를보다 쉽게 ​​생성 할 수 있도록 폴더에 정렬하고 배치 할 수도 있습니다.
  • 키보드의 " 1 "및 " 2 "키를 사용하여 편집기에서 하위 이미지 (프레임)를 빠르게 변경할 수 있습니다.

이것들은 이미지 편집기가 제공하는 고급 기능 중 일부입니다. 전체 기능 범위를 찾으려면 편집기의 모든 부분의 기능을 자세히 설명하는 아래 항목을 읽으십시오.

캔버스 컨트롤 캔버스 컨트롤

캔버스 컨트롤은 메인 그리기 캔버스와 그 내용이 표시되는 방법의 다양한 측면을 처리합니다. 옵션은 다음과 같습니다.

  • 캔버스 그리드 전환 : 캔버스 그리드를 켜거나 끕니다. 이것은 GameMaker Studio 2가 메인 캔버스 위에 그려서 여러 섹션으로 나누는 그리드이며 기본적으로 1px 크기로 설정되어 있습니다. 그러나 그리드 메뉴 아이콘을 클릭하면 그리드 옵션을 엽니 다 : 이 옵션을 사용하면 X 및 Y 축을 따라 그리드의 셀 값뿐만 아니라 그리드 색상 및 알파를 설정할 수 있습니다. 그리드 스냅을 활성화 또는 비활성화하는 옵션도 있습니다 (기본적으로 비활성화되어 있음).
  • 캔버스 줌 컨트롤 :이 버튼은 현재 캔버스 줌 레벨을 제어합니다. 확대 또는 축소하고 버튼을 누르면 이미지가 편집되는 캔버스가 1 : 1로 재설정됩니다. 창 맞춤 버튼을 클릭해도됩니다 전체 이미지 캔버스를 현재 편집기 작업 공간에 맞 춥니 다 (적절하게 확대 / 축소합니다). 을 사용하여 확대 및 축소 할 수도 있습니다. / 그리고 마우스 휠 을 누르고 / + 이미지와 함께 룸 캔버스를 1 : 1로 설정합니다.
  • 분할 캔버스 :이 버튼을 클릭하면 캔버스가 가로로 분할되고 세로로 단일 캔버스가 유지됩니다. 분할 캔버스보기에서 각 캔버스에는 고유 한 캔버스 컨트롤 세트가 있으므로 예를 들어 하나를 확대하거나 축소 할 수 있으며 변경 사항을 다른 캔버스에 반영하여 하나의 캔버스를 편집 할 수 있습니다. 캔버스보기 토글을 변경하려면 분할 옵션을 클릭하면됩니다. 단추를 클릭하고 사용하려는 단추를 선택하거나 원하는 단추로 순환 할 때까지 캔버스 분할 단추를 클릭하십시오.

브러쉬 브러쉬

Here you can select the brush to paint with. By default you have a selection of different sized square and round brushes to use, but you can easily create your own brush by painting on the image layer, and then selecting the part you wish to use for the new brush and using / + " C "로 복사하십시오. 복사 된 섹션이 브러시에 추가되고 다른 브러시와 마찬가지로 브러시로 페인트 할 수 있습니다. 여기에는 선 그리기와 같은 다양한 페인트 도구 사용이 포함됩니다.

Note that the default blending colour for the new brush will be white, but if you change the colour with the colour picker and paint using the brush, it will be blended with this new colour. If you want to remove a brush from the list, simply click on it and press the 키와 제거됩니다.

 

색상 피커 색상 피커

따라서 스프라이트 이미지에서 사용자 정의 색상 팔레트를 생성 한 다음이를 사용하여 자산 브라우저에서 다른 스프라이트에 대한 팔레트를 설정하거나 스프라이트 이미지를 열고 자산 브라우저의 다른 스프라이트에서 색상 팔레트를 가져올 수 있습니다 . 이 스프라이트 팔레트 기능은 다음 YoYo Games 기술 블로그에 자세히 설명되어 있습니다.

  • 스트립 이미지 가져 오기 -이 옵션은 파일 탐색기를 열고 프레임을 만드는 데 사용할 단일 스트립 이미지 파일을 찾도록 프롬프트합니다. 이미지를 선택하면 위에 표시된 프레임으로 변환 옵션과 동일하게 스트립 가져 오기 창이 열립니다.
  • PNG로 내보내기 -이 옵션을 선택하면 스프라이트를 단일 PNG 이미지로 내 보냅니다. 스프라이트에 둘 이상의 이미지가있는 경우 모든 이미지와 함께 "스트립"스프라이트가 하나씩 생성되고 저장 파일 이름에 " _stripX "가 자동으로 추가됩니다. 여기서 " X "는 내 보낸 프레임. 예를 들어, 5 개의 프레임 스프라이트는 " sprite_name_strip5.png "로 내보내집니다.
  • 모든 프레임 크기 조정 -이 옵션을 선택하면 스프라이트를 구성하는 모든 애니메이션 프레임의 크기를 조정하는 방법을 설정할 수있는 다음 창이 열립니다. 당신도 확장 할 수있는 이미지를 선택하거나 캔버스의 크기를 조정 할 수 있습니다. 이미지의 크기를 조정하기로 선택한 경우, 동일한 종횡비를 사용하여 크기를 조정할지 여부와 픽셀 또는 백분율로 크기를 조정할지 여부를 각 프레임의 새 너비 및 / 또는 높이를 선택하면됩니다. 참고 활성화 할 수 확장 때 직선 보간 또는 오프 그것을 가진 곳이지만 픽셀 아트 당신을위한 이미지의 크기를 조절할로 픽셀 당신이에 원하는 하이 해상도 예술을 위해, 일반적으로 (흐리게 또는 "얼룩"하게됩니다 )이 해제를 할 수 있습니다. 캔버스 크기를 조정하는 옵션은 이미지 내의 픽셀을 변경하지 않고 대신 이미지가 그려지는 캔버스의 크기를 늘리거나 줄입니다. 크기 조정에 대한 "앵커"가 화살표 아이콘을 사용하는 위치를 설정 한 다음 새 크기 (픽셀 또는 백분율)를 설정하고 원본 프레임의 가로 세로 비율을 유지할지 여부를 설정할 수 있습니다. 프레임을 원본 스프라이트 이미지보다 작게 만들면 픽셀이 잘립니다.
  • 모든 프레임을 선택하여 자르기 -이 옵션은 기본 그리기 캔버스에서 이미지 영역을 선택한 경우에만 사용할 수 있습니다. 그런 다음이 옵션을 선택하면 편집중인 이미지 (및 애니메이션의 다른 모든 프레임)가 선택 영역으로 잘립니다.
  • 모든 프레임 자동 다듬기 -이 옵션을 사용하면 스프라이트의 가장자리 주위에서 모든 영 알파 픽셀을 다듬을 수 있습니다. 이것은 스프라이트 내의 모든 이미지 프레임을 고려하므로, 각면은 가장 영이 아닌 알파 픽셀을 가진 프레임의 가장자리까지 잘립니다.
  • Reverse Frames- 이 옵션을 선택하면 스프라이트의 프레임 순서가 반대로 바뀝니다.
  • Mirror- 이 옵션을 사용하면 현재 프레임을 스프라이트를 구성하는 모든 프레임을 세로 축을 따라 미러링 할 수 있습니다.
  • 뒤집기 -이 옵션을 사용하면 스프라이트를 구성하는 모든 프레임을 가로 축을 따라 현재 프레임 중 하나를 뒤집을 수 있습니다.
  • 모든 프레임 회전 (시계 방향으로 90도) -이 옵션은 애니메이션의 모든 프레임을 시계 방향으로 90도 회전합니다.
  • 모든 프레임 회전 (시계 반대 방향 90 °) -이 옵션은 애니메이션의 모든 프레임을 시계 반대 방향으로 90 ° 회전시킵니다.

 

전망 전망

보기 메뉴에는 다음과 같은 옵션이 있습니다.

  • 1 : 1- 해상도 (예 : 실제 크기)로 이미지를 1 : 1로 조정하고 이미지 편집 창 중앙에 놓습니다.
  • 화면 맞추기 -이미지 편집 창을 채우도록 이미지 크기를 조정합니다.
  • 격자 표시 -픽셀 격자를 토글합니다. 이것은 캔버스 컨트롤에있는 그리드 버튼과 동일합니다 (이 페이지에서 더 자세히 설명합니다).
  • 어니언 스킨 설정 - 어니언 스킨 설정을 변경합니다. 그러면 스킨 스킨의 불투명도와 색상뿐만 아니라 양파 스킨 버튼으로 표시되는 프레임을 설정할 수있는 다음 창이 열립니다.

 

효과 효과

효과 메뉴는 사용자 정의 효과 플러그인을 추가 할 수있는 곳이며 기본적으로 다음과 같은 효과가 있습니다.

  • 흐림 -이미지에 흐림 효과를 적용합니다. 사용할 흐림 효과 유형 (상자 또는 가우시안)뿐만 아니라 강도를 설정하고 가로 또는 세로 축을 따라 흐릿하게 만 또는 둘 다를 따라 흐릿하게할지 여부를 설정할 수 있습니다. 그런 다음 현재 레이어, 보이는 레이어 또는 모든 레이어에만 적용하도록 선택할 수 있습니다.
  • 그레이 스케일 -정해진 양만큼 이미지의 채도를 떨어 뜨립니다. 현재 레이어, 보이는 레이어 또는 모든 레이어에만 적용하도록 선택할 수 있습니다.
