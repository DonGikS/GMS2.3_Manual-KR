도프 시트 사용

도프 시트 는 시퀀스에 추가 된 서로 다른 트랙의 데이터가 시간이 지남에 따라 표시되는 위치로, 각 트랙의 개요뿐만 아니라 트랙의 위치 및 매개 변수 변경 타이밍을 편집 할 수있는 수단을 제공합니다. 기타...

시퀀스에 새 자산을 추가하면 트랙 패널에 자산 트랙 이 만들어지고 자산 키 가 현재 재생 헤드 위치의 도프 시트에 추가됩니다. 이 키는 자산의 재생 길이를 프레임으로 반영합니다 :

마우스를 키 프레임의 시작 또는 끝으로 이동하고 커서가 클릭을 변경하면 자산 키 자체를 편집 할 수 있습니다 그리고 끌기. 선택한 항목에 따라 키 시작 또는 키 끝이 이동하지만 둘 다 키 길이에 영향을줍니다. 간단히 클릭하여 자산 키 위치를 변경할 수도 있습니다. 어디서나 왼쪽 또는 오른쪽으로 드래그하십시오.

자산 키 이외의 도프 시트에도 매개 변수 키가 표시 됩니다 . 매개 변수 키는 처음에 관련 자산의 속성에 대한 변경을 알리는 "포인트"이며 시퀀스는 해당 지점의 값을 사용하여 자산을 변경합니다. 이 작업을 수행하는 방법은 트랙 패널 에서 키를 선형 보간으로 설정했는지 여부에 따라 다릅니다. 클릭하면 트랙 키와 같은 파라미터 키를 이동할 수 있습니다 타임 라인을 따라 새 위치로 드래그하고 키를 사용하여 키를 하나 이상의 프레임으로 확장 할 수도 있습니다. 드래그하는 동시에 키를 누릅니다 (커서가이를 표시하도록 변경됨). 이는 다음 키에 대한 보간을 계속하기 전에 시간이 지남에 따라 키 값을 유지할 수 있음을 의미합니다.

도프 시트로 작업 할 때 일반적인 키보드 단축키를 사용할 수도 있습니다. / 마우스 왼쪽 버튼 순서에 상관없이 하나 이상의 트랙을 선택하려면 마우스 왼쪽 버튼 연속 트랙을 선택하고을 사용하여 잘라 내기, 복사 및 붙여 넣기를 할 수도 있습니다. / + " X ", / + " C "및 / + " V "키 조합.

마우스 오른쪽 버튼에서 사용할 수있는 키에 대한 다양한 옵션도 있습니다 메뉴는 아래에 자세히 설명되어 있습니다. 이러한 명령을 사용하여 도프 시트에 키를 붙여 넣을 때 키가 재생 헤드의 현재 위치에 붙여집니다.

이것이 도프 시트를 사용하는 기본 사항이지만 아래에서는 각기 다른 섹션의 용도와 사용법에 대해 좀 더 자세히 설명합니다.

 

지역 시작 / 지역 끝 지역 시작 / 지역 끝

이 마커는 클립 영역 시작 및 끝 지점입니다. 게임 실행시 룸에 표시 될 시퀀스의 특정 섹션을 설정하는 데 사용할 수 있으며 편집기에서 시퀀스를 미리 볼 때 루프 포인트로도 사용됩니다. 이 포인트는 시퀀스 편집기 시퀀스 컨트롤 에서 설정 한 총 시퀀스 길이보다 크거나 작을 수 없습니다. 클립 영역 마커의 위치를 ​​변경하려면 클릭 원하는 프레임으로 드래그하십시오.

줌 컨트롤 줌 컨트롤

이 막대는 도프 시트의 현재 줌을 표시하는 데 사용되며이 줌 레벨을 변경하는 데 사용될 수도 있습니다. 클릭 할 수 있습니다 끝 위젯에서 드래그하여 확대 / 축소 수준을 변경합니다.

도프 시트를 사용하여 도프 시트를 확대 또는 축소 할 수 있습니다. / 키와 마우스 휠 .

자산 트랙 자산 트랙

자산 트랙은 도프 시트에서 "키"로 표시되며, 자산의 위치와 시퀀스에서 활성화 된 시간을 나타내는 색상 막대로 표시됩니다. 상술 한 바와 같이, 이들은 필요에 따라 이동 및 연장 및 단축 될 수있다. 예를 들어 사운드가 5 프레임 동안 지속될 수 있고 키를 15 프레임으로 늘릴 수있는 것과 같이 지속 시간 동안 자산 자체보다 긴 자산 트랙 키는 전체 키 지속 시간 동안 반복됩니다. 표준 키보드 단축키를 사용하여 자산 트랙 및 키를 복사하여 붙여 넣거나 마우스 오른쪽 버튼을 사용하여 추가로 수정할 수 있습니다 메뉴 옵션 (이 페이지 아래에 설명)

자산 트랙 키를 삭제하면 트랙 자체는 삭제되지 않으며 추가 된 트랙 매개 변수도 삭제되지 않고 도프 시트에서 자산 키를 삭제하기 만합니다. 이 트랙에 대한 추가 키가없는 경우 자산 브라우저에서 자산을 도프 시트로 드래그하여 자산을 다시 추가 할 수 있으며 새 키로 추가되지만 고유 한 자산을 추가 할 수는 없습니다. 원래 자산 (예 : 스프라이트의 키를 삭제하면 해당 키 위치에 사운드 자산을 추가 할 수 없음)

파라미터 트랙 파라미터 트랙

매개 변수 트랙은 도프 시트에서 그들이 속한 주요 자산 트랙 아래에 컬러 막대로 표시됩니다. 색상 막대 자체는 편집 할 수 없으며 매개 변수 키가 적용되는 트랙을 시각적으로 표현한 것이며 편집되는 매개 변수 트랙의 개별 지점입니다 (아래의 매개 변수 키 참조).

플레이 헤드 플레이 헤드

재생 헤드는 타임 라인이 현재 미리보기되고있는 시점입니다. 이 위치는 자산을 시퀀스 캔버스 에 놓을 때 추가 된 위치뿐만 아니라 매개 변수 추가 또는 키 분할과 같은 다른 여러 가지를 설정하는 데 사용됩니다. 시퀀스 편집기 캔버스에서 "재생"을 누르면 재생 헤드가 시퀀스를 따라 이동하며 클릭 할 수도 있습니다. 그 위에 드래그하여 시퀀스 내 특정 프레임으로 이동하거나 여러 프레임을 "스크럽"하고 전체 내용을 재생하지 않고 빠르게 미리 볼 수 있습니다.

파라미터 키 파라미터 키

파라미터 키는 해당 파라미터의 변경이 발생한 곳마다 파라미터 트랙에 표시되는 "포인트"입니다. 이 포인트는 필요에 따라 이동하고 확장하여 둘 이상의 프레임을 지속 할 수 있으며 (이 페이지에서 추가로 설명) 트랙 패널 에서 또는 캔버스 에서 자산을 변경하여 해당 값을 편집 할 수 있습니다 (있는 경우) 스케일, 위치 또는 회전 매개 변수). 모든 매개 변수 트랙 키는 매개 변수가 속한 자산 키 트랙의 한 지점으로 반영되지만 이러한 지점은 편집 할 수 없으며 트랙 패널에서 다른 매개 변수 트랙을 축소 한 경우 정보를 제공합니다.

 

 

 

인민폐 메뉴

마우스 오른쪽 버튼 클릭 자산 트랙 의 도프 시트 내에서 다음 메뉴 옵션을 표시합니다 (클릭 한 트랙의 일부에 키가 있는지 여부에 따라 일부 옵션이 표시되지 않을 수 있음).

  • 잘라 내기, 복사 및 붙여 넣기 키 :이 옵션을 사용하면 자산을 잘라내거나 복사 / 붙여 넣기 할 수 있습니다. 이러한 옵션을 사용하여 도프 시트에 붙여 넣은 키는 재생 헤드 위치에 붙여집니다.
  • 키 붙여 넣기 :이 옵션을 사용하면 복사 또는 잘라낸 키가 재생 헤드 위치가 아닌 마우스 오른쪽 버튼을 클릭 한 위치에 붙여집니다.
  • Delete Key (s) : 이것을 선택하면 현재 선택된 키가 삭제됩니다.
  • Copy To Start :이 옵션은 선택한 키를 클립 영역의 시작이 아닌 시퀀스의 맨 처음에 복사합니다.
  • Copy To End :이 옵션은 선택한 키를 시퀀스의 맨 끝 (클립 영역의 끝이 아닌 )으로 복사합니다.
  • 자산 키 늘이기 :이 옵션을 선택하면 선택한 자산 키가 클립 영역의 끝이 아닌 시퀀스의 끝까지 늘어납니다.
  • 자산 키 삭제 : 이것을 사용하면 선택된 자산 키가 삭제되고 자산 브라우저에서 삭제 된 도프 시트 트랙으로 드래그하여 새 자산을 트랙에 추가 할 수 있습니다. 그러면 자산과 관련된 모든 파라미터 트랙을 유지하면서 편집 할 수있는 새로운 자산 키가 생성됩니다.

마우스 오른쪽 버튼 클릭 매개 변수 트랙 의 도프 시트 내에서 다음 메뉴 옵션이 나타납니다 (클릭 한 트랙의 일부에 키 프레임이 있는지 여부에 따라 일부 옵션이 표시되지 않을 수 있음).

  • 잘라 내기, 복사 및 붙여 넣기 키 :이 옵션을 사용하면 매개 변수 키 잘라 내기 / 복사 / 붙여 넣기를 수행 할 수 있습니다. 이러한 옵션을 사용하여 도프 시트에 붙여 넣은 키는 재생 헤드 위치에 붙여집니다.
  • 키 붙여 넣기 :이 옵션을 사용하면 복사 또는 잘라낸 키 프레임이 마우스 오른쪽 버튼을 클릭 한 위치에 표시되며 플레이 헤드 위치가 아닙니다.
  • 키 삭제 : 이것을 선택하면 현재 선택된 키 프레임이 삭제됩니다.
  • 포함 된 애님 커브 로 변환 :이 옵션은 선택한 매개 변수 트랙의 모든 키 프레임을 포함 된 애니메이션 커브로 변환합니다 (자세한 내용은 애니메이션 커브 사용 섹션 참조).
  • 외부 애님 커브로 설정 :이 옵션을 사용하면 애니메이션 커브 자산을 매개 변수 트랙에 할당 할 수 있습니다 (자세한 내용은 애니메이션 커브 사용 섹션 참조).
  • 키로 변환 :이 옵션을 사용하면 애니메이션 곡선을 개별 키 프레임으로 변환 할 수 있습니다. 곡선이 catmull-rom 보간법을 사용한 경우,이 값은 손실되고 키 프레임은 대신 곡선의 점 사이에 선형 보간법 을 사용하며 애니메이션 곡선에서 생성 된 키 는 프레임에 있어야합니다 . 즉, 키 사이로 변환 할 때 프레임 사이에 점이있는 커브는 가장 가까운 정수 프레임 값에 배치되어 커브의 동작이 변경 될 수 있습니다.
  • 임베드 된 애님 커브 내보내기 :이 옵션을 사용하면 임베드 된 애니메이션 커브를 가져 와서 애셋 브라우저에 표시되어 다른 곳에서 사용할 수있는 애니메이션 커브 애셋 으로 변환 할 수 있습니다.
  • Copy To Start :이 옵션은 선택한 키 프레임을 클립 영역의 시작이 아닌 시퀀스의 맨 처음에 복사합니다.
  • Copy To End :이 옵션은 선택한 키 프레임을 클립 영역의 끝이 아니라 시퀀스의 맨 끝으로 복사합니다.
  • Stretch Parameter Key :이 옵션을 선택하면 선택한 자산 키가 클립 영역의 끝이 아닌 시퀀스의 끝까지 확장됩니다. 늘려 진 매개 변수 키는 늘어난 프레임 수만큼 값을 유지한다는 의미입니다.
  • 매개 변수 키 삭제 :이 매개 변수 를 사용하면 선택한 매개 변수 키 프레임이 삭제됩니다.
