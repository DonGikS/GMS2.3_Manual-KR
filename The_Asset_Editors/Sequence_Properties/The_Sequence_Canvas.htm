시퀀스 캔버스

프로젝트의 시퀀스를 만들 때 작업의 많은 부분이 시퀀스 편집기 의 캔버스 에서 수행됩니다. 캔버스는 단순히 시퀀스 요소를 포함하는 데 사용되는 영역이며 시퀀스를 표시 할 룸에 추가 할 때 표시되는 영역을 정의합니다. 시퀀스를 처음 만들면 캔버스가 기본 크기 인 1366x768 픽셀로 설정되지만이 값을 클릭하면 도구 상자 에서 변경할 수 있습니다. 캔버스 프레임 토글 옆에 단추:

캔버스 크기는 시퀀스를 구성하는 자산을 배치 할 수있는 시각적 영역을 정의하지만 프레임은 단순히 크기를 안내하는 것으로서 렌더링 방식에 영향을 미치지 않습니다 (예 : 스프라이트 자산을 프레임 외부에 배치하고 추가하는 경우) 룸 순서는 자산이 여전히 렌더링됩니다.

프레임은 시퀀스 내에서 자산의 위치를 ​​안내하는 데 사용하려는 참조 이미지 를 수용하는 데에도 사용됩니다. 참조 이미지는 캔버스 프레임의 크기를 설정하는 동일한 캔버스 옵션 메뉴에서 추가 할 수 있으며 시퀀스 캔버스의 배경 이미지로 사용됩니다. 이를 통해 프로젝트 아티스트의 UI 모형에 따라 항목을 배치 할 수 있습니다.

참조 이미지는 프로젝트와 함께 저장되지 않으며 시퀀스가 ​​룸에 추가 될 때 렌더링되지 않습니다.

캔버스가 원하는 방식으로 설정되면 애셋을 추가 할 수 있습니다. 자산 브라우저 에서 캔버스로 드래그하면 트랙 패널 에 자산 트랙 이 생성됩니다.

시퀀스에 추가 할 수있는 자산은 다음과 같습니다.

  • 스프라이트
  • 사물
  • 소리
  • 시퀀스

필요한 자산을 추가 한 후에는 애니메이션의 초기 위치에 배치 할 수 있으며 다양한 변형 제어 기즈모를 사용하여 변형 할 수도 있습니다. 이들은 두 가지 방법으로 액세스됩니다 :

  • 캔버스에서 자산의 가장자리로 마우스를 이동하고 클릭 드래그 (커서가 변경 될 다른 변환을 표시하도록 변경됨) :
  • 자산의 원점에서 변형 기즈모를 사용하면 캔버스 도구 상자에서 변경할 수 있습니다.

자산의 초기 변환을 설정 한 경우 (필요한 경우) 캔버스를 사용하여 트랙 패널 에서 자산 트랙에 매개 변수 키 를 추가 할 수 있습니다. 이것이 어떻게 작동하는지에 대한 간단한 예는 자산을 시작 위치에 놓은 다음 Dope Sheet 의 재생 헤드 를 이동을 끝내려는 프레임으로 옮긴 다음 자산을 다시 이동시키는 기본 위치 변환입니다.

위 이미지에서 볼 수 있듯이 " 변경 사항 자동 기록 "을 활성화하면 트랙에 새 위치가 자동으로 기록됩니다. 트랙 패널에서 또한 자산의 시간을 통한 이동을 나타 내기 위해 캔버스보기에 점선이 추가 된 것을 볼 수 있습니다. 자동 녹음을 사용하도록 설정하지 않은 경우에도 자산을 이동 한 다음 재생 헤드를 설정 한 다음 마지막으로 " 새 키 녹음 "을 클릭하여 포인트를 추가 할 수 있습니다 단추.

마지막으로 언급해야 할 점은 캔버스를 통해 스마트 가이드 를 사용하고 추가 할 수 있다는 것입니다.이 가이드는 에셋을 정렬 할 수 있도록 캔버스에 추가 할 수있는 가이드입니다. 기본적으로 캔버스는 중심점 주위의 가로 및 세로 안내 선과 캔버스의 크기에 따라 시퀀스 캔버스의 "가장자리"주위의 직사각형 안내선을 표시합니다. 이 안내선은 방에 추가 될 때 최종 순서대로 렌더링되지 않으며 공간에 대한 아이디어를 얻고 물건을 배치하는 데 도움이됩니다. 클릭하여 자신 만의 가이드를 추가 할 수도 있습니다 눈금자에서 캔버스로 드래그 :

안내선을 제거하려면 간단히 안내선을 클릭 한 다음 눈금자를 향해 다시 드래그하십시오. 시퀀스에 안내선을 추가 한 경우 경계 상자 나 원점이 충분히 가까워지면 자산을 이동하면 안내선에 "스냅"되고 자산 사이의 거리를 표시하고 자산을 표시하는 추가 스마트 안내선이 표시됩니다 모두 같은 안내선이나 같은 위치에 있습니다 (색상이 다르게 표시됨).

 

 

인민폐 메뉴 및 탐색

캔버스보기를 탐색하려면 일반 작업 공간과 동일한 기본 제어를 사용할 수 있습니다. 예 : 가운데 마우스 버튼 사용 드래그하여 캔버스를 이동 (또는 + 마우스 휠로 가로로 스크롤 할 수 있습니다. 또는를 사용하여 확대 및 축소 / 키와 마우스 휠 .

자산을 선택하려면 마우스 왼쪽 버튼을 클릭하십시오. 그것에, 또는 사용하여 여러 자산을 선택할 수 있습니다 / 과 각 자산을 클릭하십시오. 다음을 사용하여 여러 자산을 선택할 수도 있습니다 클릭 그리고 끌기. 그러면 캔버스에 "상자"가 생성되고 상자 영역에 닿는 자산이 선택됩니다.

자산을 선택하면 다음을 사용하여 자산을 잘라 내고 복사하여 붙여 넣을 수 있습니다 / + " X ", / + " C "및 / + " V "키 조합.

캔버스에서 여러 자산을 선택하면 각 자산에 대해 경계 상자가 표시되고 모든 자산에 대해 더 큰 경계 상자가 표시됩니다. 왼쪽 상단의 아이콘에서이 경계 상자의 동작을 변경할 수 있습니다.

원 아이콘을 클릭하면 변환 제어 기즈모가 선택된 모든 자산에 대한 전체 경계 상자의 원점으로 변경되며, 상자 아이콘을 클릭하면 선택 항목의 각 자산에 대한 개별 제어 위젯이 표시됩니다. 단일 자산과 마찬가지로 그룹화 된 자산을 이동하고 복사 / 붙여 넣기 등을 수행 할 수 있습니다.

겹치는 자산이 여러 개 있고 나머지 자산을 방해하지 않고 특정 자산을 선택하려는 경우 마우스를 자산 위로 가져 가서 항목 도구 상자 를 표시하면 마우스 아래에 각 항목의 아이콘 목록이 표시됩니다. 그런 다음이 항목 중 하나를 클릭하여 선택하고 편집 할 수 있습니다.

또한 마우스 오른쪽 버튼으로 여러 가지 추가 옵션을 사용할 수 있습니다 순서대로 자산을 클릭하면

여기서 사용할 수있는 옵션은 다음과 같습니다.

  • 기록 :이 옵션을 사용하면 자산에 대한 매개 변수 키를 기록 할 수 있으며, 그 위에 마우스를 놓으면 스케일 , 회전 또는 위치 를 기록하는 옵션이 표시됩니다.
  • Group : 이것을 선택하면 트랙 패널에 새 그룹 폴더가 생성되고 선택된 자산이 그 폴더로 이동합니다.
  • 이름 바꾸기 :이 버튼을 클릭하면 트랙이 트랙 패널에 강조 표시되고 이름을 편집 할 수있게되어 변경할 수 있습니다.
  • 시트에서 찾기 :이 버튼을 클릭하면 선택된 자산의 키가 강조 표시됩니다.
  • 기타 찾기 :이 버튼을 클릭하면 동일한 유형의 다른 모든 자산 목록이 표시됩니다.
  • 선택 : 클릭하면 클릭 한 위치에서 겹치는 모든 자산 목록이 표시되며 편집 할 자산을 선택할 수 있습니다.
  • 키 제거 :이 항목을 선택하면 자산 트랙 키가 현재 재생 헤드 위치의 도프 시트에서 제거됩니다. 트랙에는 포함 된 모든 파라미터 트랙과 마찬가지로 트랙 자체가 여전히 존재하며 자산 브라우저에서 도프 시트로 드래그하여 다른 자산을 트랙에 다시 추가 할 수 있습니다.
  • 트랙 제거 : 트랙 패널에서 자산 트랙 (및 할당 된 경우 매개 변수 트랙)과 도프 시트의 관련 키를 모두 제거합니다.
  • 편집기 열기 : 자산 유형에 대한 기본 편집기에서 편집 할 자산을 엽니 다.
