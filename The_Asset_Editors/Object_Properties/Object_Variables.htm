객체 변수

변수 버튼을 클릭하면 객체 변수 창이 열리 며, 객체의 이벤트 생성 전에 변수를 생성하는 데 사용할 수 있습니다.

이 창에서 추가 버튼을 클릭하여 목록에 새 변수를 추가 할 수 있습니다. 그런 다음 변수 이름 을 지정하고 유형 을 설정하고 속성을 조정할 수 있습니다 .

변수 이름은 문자로 시작해야하며 최대 64 자 길이의 문자, 숫자 및 밑줄 기호 '_'만 포함 할 수 있습니다. 따라서 유효한 변수는 fish , foo_bar , num1 과 같 으며 유효하지 않은 변수는 6fish , foo bar 또는 * num 입니다. 변수 이름을 지정하면 변수 유형을 설정해야합니다. 다음 중 하나 일 수 있습니다. 이 기능은 부모 / 자식 인스턴스로 작업 할 때 특히 유용합니다. 정의 된 변수 세트를 가진 부모 개체를 가질 수 있고 자식 개체에서 간단히 수정할 수 있기 때문입니다 (아래의 개체 변수 및 자식 개체 섹션 참조). 더 자세한 정보)- 룸 편집기 에 인스턴스를 추가 할 때 ( 레이어 > 인스턴스 레이어 섹션 참조), 룸에 배치 된 개별 인스턴스에 대해 이러한 변수를 수정할 수도 있습니다.

작성할 수있는 다양한 유형의 변수는 다음과 같습니다.

  • 실수 : 실수는 정수가 아닌 양수이거나 음수가 될 수있는 숫자입니다. 따라서 124.5 , 45639.566546456 , 0.9 , -45.5 등은 모두 실수의 예입니다. 모든 실수는 64 비트 부동 소수점 값 (더블)으로 저장됩니다. 실수 (및 정수 ) 데이터 유형에는 옵션을 클릭하여 액세스 할 수있는 값 범위를 설정하는 옵션이 있습니다. 단추: 이를 확인하면 시작 값과 끝 값을 입력 할 수 있으며 Object Variables 창에서 변수에 대해 고정 값을 표시하는 대신 다음 값으로 고정되는 슬라이더가 있습니다.
  • 정수 : 정수는 정수이며 양수 또는 음수 일 수 있습니다 (예 : 30004 , 19 , 0 , -300) . 실수에 대해 위에서 설명한대로 출력 정수의 값 범위를 설정할 수 있습니다.
  • String : 문자열은 "fish", "Hello World"또는 "12345"와 같이 큰 따옴표 안에있는 것입니다.
  • 부울 : 부울 값은 true 또는 false입니다. 오브젝트 편집기 변수 창에서 이는 단순히 확인하고 false를 선택 취소하는 상자로 표시됩니다.
  • 표현식 : 표현식은 일반 숫자, 변수, 문자열 또는 GML 함수와 하나 이상의 연산자를 포함 할 수있는 수학 구문입니다. 예를 들어 sqrt (85 * 6) + 5.5 는 표현식입니다.
  • 자원 : 자원은 단순히 자산 브라우저에서 정의한 자원 중 하나입니다. 이 리소스 유형을 선택하면 Open Asset Explorer 를 클릭하십시오. 버튼을 클릭하여 자산 탐색기를 열고 필요한 리소스를 선택하십시오.
  • 목록 : 입력 목록 유형을 선택하면 값 선택 (문자열, 실수, 표현식 등)을 생성 할 수 있으며 변수에서 반환 할 변수를 하나 이상 선택할 수 있습니다. 목록에있는 항목을 정의하려면 먼저 옵션에 항목을 추가해야합니다 ( 단추): 드롭 다운 메뉴에서 기본 창의 변수에 대한 기본값으로 항목을 선택할 수 있습니다. 리스트 옵션에서 다중 선택을 선택한 경우 변수는 선택한 모든 옵션의 배열이됩니다.
  • 색상 : 색상 유형은 변수에 저장할 색상 값을 정의하기위한 것입니다. 실수 (0-16777216, 알파 값은 255 임) 또는 16 진 값 ($ RRGGBBAA 형식)을 입력하거나 두 번 클릭 할 수 있습니다. 색상 견본에서 색상 피커 를 열고 색상 을 정의합니다.

 

 

객체 변수와 자식 객체

객체 변수의 중요한 기능은 자산 브라우저에있는 모든 하위 객체에 의해 상속 된다는 것입니다 (자세한 내용은 상위 객체 섹션 참조). 따라서 모든 또는 일부를 재정의하거나 변경하도록 선택할 수 있습니다. 당신이 원한다면 그들.

변수가 정의 된 다른 오브젝트 자원의 하위 오브젝트를 작성하면 다음과 같이 오브젝트 변수 창에 표시됩니다.

상단 이미지에서 Parent 객체에 4 개의 객체 변수가 있고 하단 이미지에도 "회색으로 표시"되고 Inherited From Parent가 있음을 확인할 수 있습니다. 아이콘이 상속되었음을 나타냅니다. 변수 무시 를 클릭하면 상속 된 변수를 편집 할 수 있습니다 버튼을 클릭하면 범위 슬라이더 또는 값을 조정하거나 목록 등에서 다른 항목을 선택할 수 있습니다 . 상위 변수 를 편집 할 때 정의 된 값만 변경할 수 있지만 이름은 변경할 수 없으며 변수를 변경할 수도 없습니다. 옵션. 또한 새 개체 변수를 자식 개체에도 추가 할 수 있습니다 (위의 예제 이미지에서 "문자 유형"은 자식 개체에 대해서만 새로운 개체 변수라는 것을 알 수 있습니다).

상속 된 객체 변수를 편집 한 경우 삭제를 클릭 할 수 있습니다 변수를 제외하고 편집을 제거하려면 버튼을 클릭하십시오. 변수는 여전히 상위에서 상속되지만 상위 기본값을 다시 사용합니다. 변수를 완전히 제거해야하는 경우 상위 오브젝트에서이를 수행해야합니다.
