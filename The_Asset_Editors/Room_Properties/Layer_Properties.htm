레이어 유형 및 속성

룸 편집기 에서 작성한 룸에 들어가는 모든 것은 레이어에 배치됩니다. 레이어 편집기에서 레이어를 추가, 제거 및 정렬 할 수 있으며 각각 고유 한 옵션 세트와 작업 방식이있는 다양한 유형이 있습니다. 각 유형의 레이어 수에 제한을 두지 않고 여러 타일 레이어, 경로 레이어 또는 인스턴스 레이어 등을 가질 수 있습니다. 각 레이어 유형에는 고유 한 속성 창이 있습니다. 레이어 타입 런타임에 보이지 않는 레이어를 처리 하지 않고 레이어 가시성을 전환 할 수도 있습니다 (그러나 적절한 레이어 기능을 사용하여 언제든지 다시 표시 할 수 있음).

위의 이미지는 레이어 편집기를 보여줍니다. 현재 레이어가 맨 위에 나열되고 다른 버튼이 맨 아래에 나열됩니다. 이름을 천천히 두 번 클릭하여이 편집기에서 레이어 이름을 바꿀 수 있습니다 (문자 숫자와 이름에 밑줄 "_"문자 이외의 다른 것을 사용할 수 없음). 눈 아이콘 . 방에 부모 방에서 상속 된 레이어가있는 경우 맨 아래에있는 단추에서 상속을 토글 할 수 있지만 모든 레이어에 영향을 미칩니다 (상속에 대한 자세한 내용은 여기 참조). 당신은 또한 사용할 수 있습니다 또는 / 왼쪽 클릭 여러 레이어에서 복제 또는 삭제를 위해 선택할 수 있습니다 (이 옵션은 마우스 오른쪽 버튼으로도 제공됨) 메뉴).

레이어 순서를 정돈하고 깔끔하게 유지하려면 폴더 아이콘을 클릭하여 레이어 폴더에 레이어 그룹을 만들 수 있습니다. . 삭제 버튼을 클릭하여 하나 이상의 선택된 레이어를 삭제할 수도 있습니다 . 모든 레이어는 깊이 값을 가지며, 이는 그 내용이 룸에서 렌더링 될 때 레이어가 그릴 순서에서 배치 될 위치를 정의합니다. 레이어는 가장 높은 깊이에서 가장 낮은 깊이까지 그려 지므로 깊이가 -100 인 레이어는 깊이가 200 인 레이어 위에 그려집니다.

중대한! 최소 및 최대 레이어 심도는 -16000 ~ 16000입니다. 해당 범위 밖의 레이어에 배치 된 항목은 모든 이벤트가 레이어의 모든 항목에 대해 정상적으로 실행 되더라도 그려지지 않습니다.

마우스 오른쪽 버튼을 클릭하면 모든 레이어에서 레이어 메뉴를 얻을 수 있습니다 :

여기서 선택한 레이어의 레이어 속성 창을 열거 나 레이어를 삭제하거나 레이어 이름을 바꾸거나 하위 레이어를 추가 할 수 있습니다. 하위 레이어를 추가하도록 선택하면 현재 레이어에 "폴더"아이콘이 표시되어 하위 레이어가 첨부되어 있고 그 아래에 새 레이어가 생성되어 오른쪽에 탭으로 표시됩니다. 그런 다음 하위 레이어가 부모 레이어의 속성을 상속 받도록 선택하고 부모 레이어의 가시성을 상속할지 여부를 설정할 수도 있습니다.

창에서 레이어를 위나 아래로 드래그하여 순서를 바꿀 수 있으며, 다음 중 하나를 사용하여 여러 레이어를 선택하고 이동할 수도 있습니다. + 한 레이어에서 다른 레이어로 (또는 그 사이의 모든 레이어를 포함하여) 선택하거나 / + 한 번에 하나씩 레이어를 선택합니다. 레이어를 레이어 폴더 위에 놓으면 레이어가 이동되어 놓은 폴더의 하위 레이어로 설정됩니다. 잠금 버튼을 사용하여 실수로 편집 할 수 없도록 레이어를 잠글 수도 있습니다 .

참고 : 레이어에 다른 자산을 배치하는 실제 편집기 창에서 " P "+를 길게 누르면 자산을 즉시 선택하고 배치 된 레이어로 건너 뛸 수 있습니다.

배경 배경

를 클릭하여 새 배경 레이어를 추가 할 수 있습니다 단추. 새 레이어가 추가되고 배경 속성 창이 열립니다.

새 배경 레이어를 만들면 기본적으로 비어 있으며 정의 된 배경색을 그립니다. 배경 레이어 이미지에 스프라이트를 할당하지 않으면 대신 그려집니다. 색상 상자를 클릭하여 색상 선택기를 열고 사용 된 색상을 변경할 수 있습니다. 스프라이트를 선택하면 타일링 옵션을 선택하여 가로 또는 세로로 바둑판 식으로 배열하거나 전체 방을 채우기 위해 이미지를 늘릴 수 있습니다.

배경을 한 방향으로 스크롤하려면 수평 및 / 또는 수직 속도도 설정하십시오 (각 게임 단계는 여기에서 설정 한 픽셀 수만큼 주어진 액세스를 따라 이동합니다). x 및 y 오프셋 값을 설정하여 배경 이미지를 배치합니다. 선택한 스프라이트 리소스에 여러 개의 하위 이미지가있는 경우 배경의 애니메이션 속도를 설정할 수도 있습니다 (기본적으로 사용되는 이미지에 대해 스프라이트 편집기에서 설정 한 값으로 잠겨 있음). 마지막으로 레이어의 깊이를 명시 적으로 설정할 수 있지만 기본적으로이 깊이는 잠겨 있지만 레이어 편집기에서 레이어 순서를 끌어서 순서를 바꾸면이 값을 변경할 수 있습니다.

맨 위에있는 버튼을 클릭하여이 창에서도 배경 상속을 전환 할 수 있습니다.

 

인스턴스 인스턴스

을 클릭하여 새 인스턴스 레이어를 추가 할 수 있습니다 단추. 새 레이어가 추가되고 인스턴스 속성 창이 열립니다.

이 창에는 현재 레이어에있는 모든 인스턴스 목록이 표시됩니다. 자산 브라우저 를 열고 객체를 실에서 드래그하여 레이어에 인스턴스를 더 추가하거나 자산 브라우저에서 인스턴스를 선택한 다음 길게 누를 수 있습니다 "현장"리소스를 미리 보거나 마우스 왼쪽 버튼을 추가로 클릭하면 버튼을 누른 상태에서 마우스를 움직여 인스턴스를 추가하고 룸 레이어에 "페인트"할 수도 있습니다.

클릭하여 레이어 목록을 주문할 수 있습니다 목록의 인스턴스를 필요한 위치로 드래그하지만, 이것은 룸 편집기 내에서 조직적인 목적으로 만 사용되며 특정 생성 순서를 설정하려면 생성 순서 창을 사용하십시오. 룸 속성 창 또는 룸 상황에 맞는 메뉴 에서

더블 클릭 룸 에디터 자체 또는 속성 목록에있는 모든 인스턴스에서 새 단일 창을 열어 해당 단일 고유 인스턴스의 측면을 수정할 수 있습니다.

기본적으로 인스턴스 이름은 " inst_38B0BFF "와 같지만 이름에 영숫자 문자와 밑줄 "_"(공백 없음) 만 사용하는 경우 원하는 이름으로 변경할 수 있습니다. 그러나 여기에 주어진 이름 (사용자가 지정한 이름이든 IDE가 제공하는 기본 이름이든)은 상수 로 간주되므로 인스턴스에 고유 해야합니다 . 또한 게임에서 코드를 사용하여 인스턴스를 식별 할 수 있지만 인스턴스와 같은 방에있는 경우에만 사용할 수 있습니다.

인스턴스 이름 아래에는 인스턴스가, 부모 방에서 상속 당신이의 인스턴스 또는 심지어 개방하려는 객체를 변경하는 것입니다 여부를 세트로 버튼이 편집기 개체 거기 변경 아무것도하고자한다. Object Variable 창을 통해 변수를 확인하고 편집 할 수있는 버튼이 있으며이 인스턴스의 초기 특성도 변경하여 색상을 혼합 (기본적으로 흰색)하거나 회전 및 뒤집을 수 있습니다. 또한 축 중 하나 또는 둘 다를 따라 인스턴스의 크기를 조정하고 룸 내 위치와 스프라이트에서 사용할 이미지 속도 및 초기 프레임을 설정할 수 있습니다.

마지막으로 객체 생성 이벤트 코드 바로 다음에 실행되며이 인스턴스에 고유 한 생성 코드를 인스턴스에 제공 할 수 있습니다. 따라서 모든 인스턴스가 생성시 수정해야하는 객체에 변수를 설정할 수 있습니다. 하나의 인스턴스에 고유 한 값이되도록 코드). 이것을 클릭하면 스크립트 창 을 인스턴스에 연결하여 함수를 추가하거나 변수를 수정할 수 있으며 인스턴스가 생성되고 인스턴스 생성 이벤트 가 실행 된 후이 코드가 직접 실행됩니다. 인스턴스 상속과 독립적 으로 생성 코드 상속을 설정하거나 해제 할 수 있습니다.

게임이 만들어 질 때 레이어의 각 인스턴스는 자동으로 내보내기 플래그가 지정됩니다. 그러나 특히 상속 작업을 할 때 최종 게임에서 특정 인스턴스를 회의실에 추가하지 않으려는 경우가있을 수 있습니다 (또는 모든 회의실에 디버그 인스턴스가 있고 최종 게임에서 제외하려는 경우). ). 이 경우 목록에서 인스턴스 왼쪽의 " Export "확인란을 선택 취소하면 실행 파일의 일부로 내 보내지 못하게됩니다. 그러나 인스턴스를 참조하는 코드가있는 경우 게임이 제대로 실행되지 않으므로이 기능을주의해서 사용하십시오.

 

타일 ​​맵 타일 ​​맵

를 클릭하여 새 타일 ​​맵 레이어를 추가 할 수 있습니다 단추. 새 레이어를 추가 한 다음 타일 ​​맵 레이어 속성 창과 타일 ​​편집 창을 엽니 다.

타일 ​​맵 레이어 속성에서 레이어 의 상속을 토글하고, 사용중인 타일 세트를 변경하고 레이어 그리드가 (0, 0)에 정확히 맞지 않게하려면 레이어의 x 또는 y 오프셋을 설정할 수 있습니다 방의 위치. 레이어를 렌더링 할 깊이 (독특한 상속 버튼이 있음)를 설정할 수도 있으며이 깊이도 "잠금"으로 만들면 레이어를 만든 후에는 더 이상 편집 할 수 없습니다.

레이어 속성 창이 열리면 사용중인 타일 세트를 표시하는 타일 편집 창도 열립니다 (실수로이 창을 닫은 경우 IDE 상단의 룸 > 타일 ​​편집 메뉴에서 다시 열 수 있음) . 클릭 할 수 있습니다 타일에서 현재 브러시 로 선택한 다음 클릭하여 룸 편집기의 레이어에 그립니다. 룸 캔버스에서 또는 여러 타일을 함께 그리려면 마우스 왼쪽 버튼을 클릭하여 그룹으로 선택할 수 있습니다 드래그하기 전에 타일셋에서 드래그하여 선택합니다. 함께 그릴 타일 범위가 블록에 없으면 누르고 있으면됩니다. / 마우스 왼쪽 버튼을 사용하여 개별 타일을 선택하십시오 그런 방식으로 사용자 정의 브러시를 만듭니다 (사용자 정의 브러시에서 타일을 제거 할 수도 있음) + ). 이 방법으로 사용자 정의 브러쉬를 만드는 것은 일시적이며 일회성 용도로 설계되었지만 브러쉬 탭을 사용하여 타일 맵 레이어와 함께 저장 될 영구 브러쉬를 만들 수 있습니다.

Tile Editing (타일 편집) 창에서는 마우스 휠 또는 상단의 버튼을 사용하여 타일을 확대 및 축소 할 수 있습니다. 가운데 마우스 버튼 또는 + . 창에서 타일을 구분하기 위해 켜거나 끌 수있는 그리드 옵션도 있으며 타일의 표시 여부에 관계없이 항상 그리드에 스냅되지만 그리드의 색상도 설정할 수 있습니다. .

타일 ​​편집 창을 사용하면 라이브러리 탭에서 브러시 , 애니메이션 타일 및 자동 타일 을 선택할 수 있습니다. 타일 탭은 모두 타일 ​​세트 편집기 의 섹션에서 다룹니다.

룸 편집기의 상단에서 타일 레이어를 선택하면 타일과 관련된 도구 세트가 표시됩니다.

연필 이것은 연필 도구입니다. 선택한 타일을 사용하여 마우스 왼쪽 버튼으로 룸 편집기에서 페인트합니다. 마우스 오른쪽 버튼으로 지울 수 있습니다 . 타일의 색인 값은 룸 편집기의 룸 상태 표시 줄 맨 아래에 표시되며 페인팅을 위해 여러 타일을 선택한 경우 (예 : 브러시 ) 사용 된 모든 타일 인덱스 목록이 표시됩니다. .
지우개 지우개 도구를 사용하면 마우스 왼쪽 버튼을 사용할 수 있습니다 룸 편집기에서 지정된 타일을 지우려면 기본적으로이 모든 작업은 타일 인덱스를 0으로 설정하는데, 이는 예약 된 "빈"타일입니다.
가득 따르다 채우기 도구를 사용하면 현재 타일 브러시로 전체 타일 맵 레이어 또는 선택한 타일을 채울 수 있습니다. 사용하는 브러시는 단일 타일 또는 여러 타일로 구성된 복합 브러시이거나 애니메이션 라이브러리의 타일 및 클릭 레이어에서 선택한 브러시를 사용하여 타일을 채 웁니다. 선택 도구를 사용하지 않은 경우 채우기는 연속 된 모든 빈 타일 ​​또는 동일한 인덱스의 모든 연속 된 타일 에서 작동합니다 . 레이어 내에서 선택 항목이 있으면 해당 선택 내에서 채우기가 제한됩니다.
선 선 도구를 선택할 때 그런 다음 끌어서 놓으면 클릭 한 위치와 놓은 위치 사이에 타일 선이 만들어집니다. 선택 도구를 사용하여 타일 맵 레이어의 영역을 선택한 경우 선은 선택 범위 내에서만 추가됩니다. 단일 타일 브러시, 다양한 타일의 합성 브러시 또는 라이브러리의 애니메이션 타일을 사용하여 선을 그릴 수 있습니다.
직사각형 사각형 도구를 선택할 때 그런 다음 끌어서 놓으면 채워진 사각형의 타일이 만들어집니다. 왼쪽 위 모서리는 클릭하는 위치이고 오른쪽 아래 모서리는 놓는 위치입니다. 단면 도구를 사용하여 타일 맵 레이어 영역을 선택한 경우 사각형은 선택 범위 내에서만 추가됩니다. 사각형은 단일 타일 브러시, 다양한 타일의 합성 브러시 또는 라이브러리의 애니메이션 타일을 사용하여 그릴 수 있습니다.
선택 이 도구는 작업을 위해 타일 레이어의 영역을 정의하는 데 사용할 수있는 선택 도구입니다. 마우스 왼쪽 버튼을 클릭 할 수 있습니다 마우스를 끌어 직사각형 영역을 만들거나 / + 여러 선택을 추가하고 + 선택의 일부를 지우십시오. 전체 선택을 지우려면 . 타일 ​​레이어 영역을 선택한 경우 나머지 도구 (연필, 뒤집기, 회전 등)는 선택한 영역 내에서만 작동합니다. 복사 할 수도 있습니다 ( / + " C ") 컷 ( / + " X ") 및 붙여 넣기 ( / + " V ")를 선택하면 룸 편집기에서 타일을 선택한 다음 도구를 연필로 전환하고 선택한 타일로 룸 레이어를 페인트 할 수 있습니다.
자동 타일 이 도구를 클릭하면 자동 타일링 페인트 스타일이 활성화됩니다. 이 기능이 활성화되면 Autotile Libraries 탭에서 타일을 선택한 다음 룸 레이어에 페인트 할 수 있으며 GameMaker Studio 2는 자동 타일 탭을 올바르게 설정 한 경우 주변 타일과 일치하도록 자동으로 타일을 변경합니다. 자동 라이브러리의 일부가 아닌 타일 세트에서 타일을 선택하면 그리기 도구가 표준 연필 도구로 재설정됩니다.
계승 이 도구를 사용하면 주어진 레이어의 특정 타일에 대한 상속 을 활성화 / 비활성화 할 수 있습니다. 이 작업을 수행하려면, 방은 첫째 아이 방이어야하며 계층은 그것의 상속 계층 속성에서 켜져 있어야합니다. 이 상태에서 룸 편집기는 기본적으로 모든 상속 된 타일을 빨간색으로 표시하고 마우스 오른쪽 버튼을 클릭합니다. 해당 타일의 상속을 끄고 마우스 왼쪽 버튼을 클릭합니다 다시 켤 것입니다. 이런 식으로 "베이스"룸의 요소를 재사용 할 수 있으며 나머지 타일은 모두 상속하면서 필요한 타일 만 선택적으로 변경할 수 있습니다.
튀기다 마우스 왼쪽 버튼으로 뒤집기 도구 클릭 (또는 키보드 단축키 " X ")를 사용하면 그리기 도구를 변경하지 않고 현재 가로 축을 따라 그리기 위해 선택된 타일을 뒤집습니다 (그리기 위해 사용자 정의 브러시가 선택된 경우 전체 브러시가 뒤집 힙니다). 그리기 위해 선택한 타일이 없지만 레이어에서 타일 그룹을 선택한 경우 뒤집기 도구는 선택한 타일을 뒤집습니다.
거울 마우스 왼쪽 버튼으로 미러 도구 클릭 (또는 키보드 단축키 " Y "를 사용하면) 그리기 도구를 변경하지 않고 세로 축을 따라 그리기 위해 현재 선택된 타일을 미러링합니다 (그리기 위해 사용자 정의 브러시가 선택된 경우 전체 브러시가 미러링됩니다). 그리기 위해 선택한 타일이 없지만 레이어에서 선택한 타일 그룹이있는 경우 미러 도구는 선택한 타일을 미러링합니다.
회전 마우스 왼쪽 버튼으로 회전 도구 클릭 키보드 단축키 " Z "를 사용하면 그리기 도구를 변경하지 않고 현재 그리기 위해 선택한 타일을 시계 방향으로 90도 회전합니다 (그리기 위해 사용자 정의 브러시가 선택된 경우 전체 브러시가 회전합니다). 그릴 타일을 선택하지 않았지만 레이어에서 타일 그룹을 선택한 경우 회전 도구는 선택한 타일을 회전시킵니다.

선택 도구를 사용하는 경우 룸 편집기 캔버스에서 타일 그룹을 선택한 다음 잘라내거나 복사하면 일시적으로 페인팅을위한 현재 브러시가됩니다.

 

경로 경로

를 클릭하여 새 경로 레이어를 추가 할 수 있습니다 단추. 그러면 새 레이어가 추가되고 경로 속성 창이 열립니다.

여기에서 자산 브라우저에서 레이어에 사용할 경로를 선택한 다음 정의 된 연결 유형 ( 부드럽게 보간 또는 직선 사용)과 경로의 닫힘 여부 및 경로 생성 정밀도를 변경할 수 있습니다 . 여기에서 상속을 토글하고 경로를 편집기에서 더 잘 보이도록 경로를 특정 색상 (기본값은 빨간색)으로 설정할 수 있습니다. 경로 리소스가 정의되어 있지 않으면 새로 만들기 버튼을 클릭하여 만들 수 있습니다. 패스 레이어는 사용시 실제로 게임에서 렌더링 되는 것이 없더라도 깊이 값이 기본적으로 잠겨 있습니다.

마우스 왼쪽 버튼을 클릭하여 경로에 점을 추가 할 수도 있습니다. 마우스 오른쪽 버튼을 사용하여 점을 제거 할뿐 아니라 을 클릭하거나 실제 룸 편집기 내에서 점을 클릭하고 드래그하여 점을 선택하면 경로를 매우 쉽게 작성하고 편집 할 수 있습니다 (여전히 경로 편집기 에서 경로를 편집 할 수는 있음). 누르고 있으면 / + 다른 경로 지점에서 모두 선택되고 메뉴 옵션 (아래에 설명)에서 함께 이동하거나 삭제할 수 있습니다.

마우스 오른쪽 버튼을 클릭하면 경로를 조작하는 동안 룸 편집기 창에서 다음 옵션을 사용할 수 있습니다.

여기에서 경로 방향을 반전시킬 수 있습니다 (이것은 눈에 띄는 효과는 없지만 시작점이 끝 점이되고 반대의 경로 점이 계산되는 순서를 반대로합니다). 이 메뉴에서 선택한 지점을 삭제할 수도 있습니다. " T ", " S "또는 " R "을 누른 상태에서 드래그하여 전체 경로를 변환, 축척 또는 회전시킵니다.

 

자산 자산

을 클릭하여 새 자산 레이어를 추가 할 수 있습니다 단추. 새 레이어가 추가되고 자산 레이어 속성 창이 열립니다.

자산 레이어는 단순히 인스턴스와 독립적으로 방에 배치 된 그래픽 자산 (스프라이트 또는 시퀀스)이며 자산 브라우저 에서 드래그하거나 자산 브라우저 에서 원하는 것을 선택하여 배치 할 수 있습니다. 그런 다음 + "페인트"합니다. 스프라이트 애셋은 타일과 유사하며, 사용 된 스프라이트에 타일에 적용되는 제한없이 (즉, 사용되는 스프라이트에 표시 할 하위 이미지가있는 경우) 애니메이션을 적용 할 수 있습니다. 어떤 길이와 속도가 될 수 있습니다. 시퀀스는 방에 들어올 때 재생할 스프라이트, 사운드 및 인스턴스의 "선택"입니다.

자산 레이어에 자산을 배치하는 동안 그리드의 룸 편집기 상단에 설정된 그리드 스냅 값에 스냅됩니다. 그러나 자유롭게 배치하려면 그리드를 끄거나 키를 사용하십시오. / 마우스 왼쪽 버튼 . 해당 키를 누른 상태에서 그리드 해상도에 맞추지 않고 어디서나 자산을 자유롭게 배치 할 수 있습니다.

룸에 자산을 배치하면 두 번 클릭하여 자산의 모양을 제어 할 수 있습니다 자산 속성 창을 열려면 다음을 수행하십시오.

여기에서 자산에 고유 한 이름을 지정하거나, 상위 회의실에서 상속할지, 표시하려는 스프라이트 또는 시퀀스를 변경할지 여부를 설정할 수 있습니다. 시퀀스의 경우 애니메이션 속도 및 초기 재생 헤드 프레임과 혼합 할 색상 (기본적으로 흰색) 및 방의 위치를 ​​설정할 수 있습니다. 스프라이트 에셋의 특성을 변경하거나 블렌드 할 색상을 설정 (기본적으로 흰색)하거나 회전 및 뒤집을 수도 있습니다. 축 중 하나 또는 둘 모두를 따라 스프라이트를 스케일링하고 룸 내에서 위치를 설정할 수 있습니다. 스프라이트에 하위 이미지가있는 경우 애니메이션 속도를 선택하고 표시 할 초기 프레임을 설정할 수 있습니다.

게임을 만들 때 레이어의 각 자산에 내보내기 플래그가 자동으로 지정됩니다. 그러나 특히 상속 작업을 할 때 최종 게임에서 방에 특정 스프라이트를 추가하지 않으려는 것일 수 있습니다. 이 경우 레이어 속성 목록에서 스프라이트 왼쪽에있는 " 내보내기 "확인란을 선택 해제하면 실행 파일의 일부로 내 보내지 못하게됩니다. 그러나 해당 자산을 참조하는 코드가 있으면 게임이 제대로 실행되지 않으므로이 기능을주의해서 사용하십시오.
