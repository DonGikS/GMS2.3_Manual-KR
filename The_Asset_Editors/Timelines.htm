타임 라인 편집기

대부분의 게임에서 특정 순간에 정확한 일이 발생하기를 원할 것입니다. 이제 알람 이벤트를 사용하여이 작업을 수행 할 수 있지만 상황이 너무 복잡하면 더 이상 작동하지 않습니다. 특히 12 개의 알람으로 제한되어 있기 때문입니다. 이것이 타임 라인 자산을 보유한 이유입니다. 타임 라인에서 게임 시간의 특정 순간에 어떤 동작을 수행해야하는지 지정하고, 다른 이벤트 나 코드에서 오브젝트에 사용 가능한 모든 동작을 사용하여이를 정의 할 수 있습니다. 타임 라인을 생성 한 후에는이를 객체에 할당 할 수 있으며 해당 객체의 인스턴스는 룸 내에 배치하거나 생성 할 때 표시된 순간에 작업을 실행합니다.

예를 들어, 경비원을 원하고이 경비원이 20 개의 시간 단계를 왼쪽으로 이동 한 다음 10을 위로, 20을 오른쪽으로, 10을 아래로 이동 한 다음 마지막으로 정지시켜야한다고 가정합니다. 이를 위해 모션을 왼쪽으로 설정하는 것으로 시작하는 타임 라인을 만들 수 있습니다. 그런 다음 모멘트 20 ( "모멘트"는 "게임 단계"또는 "게임 프레임"을 말하는 또 다른 방법 임)을 만들고 모션을 위쪽으로 설정 한 다음, 30을 순간에 오른쪽으로, 50을 순간에 아래로, 마지막으로 순간 60에 당신은 운동을 멈출 것입니다. 이 작업이 완료되면이 타임 라인을 가드 객체에 할당 할 수 있으며 가드 인스턴스는 방에 배치 할 때 계획 한대로 정확하게 수행합니다. 일정 시간 후에 알람이 반복되도록 시간 표시 막대에 알람을 설정하여 경비원이 지역을 순찰 할 수도 있습니다. 또한 타임 라인을 사용하여 보이지 않는 컨트롤러 오브젝트를 생성 한 다음 특정 순간에 적을 생성 한 타임 라인을 컨트롤러 오브젝트에 할당하고 인스턴스를 룸에 배치함으로써 게임을보다 전체적으로 제어 할 수 있습니다. 이것은 우주 정거장 게임이나 그와 비슷한 게임에서 적의 시간을 산란하는 쉬운 방법입니다.

타임 라인을 만들려면 마우스 오른쪽 버튼을 클릭하십시오. Asset Browser 에서 "Create Timeline"을 선택하십시오. 그러면이 페이지 상단에 그림으로 표시된 창이 다음 섹션과 함께 열립니다.

 

이름 이름

여기서 타임 라인 리소스에 이름을 지정 합니다. 여기에는 영숫자 문자 또는 "_"밑줄 만 포함되어야하며 코드를 통해 타임 라인을 참조하는 데 사용되므로 숫자로 시작할 수 없습니다. 이 이름은 또한 타임 라인을 식별하기 위해 DnD ™ 액션 또는 코드 기능에 사용될 수있는 고유 한 타임 라인 ID입니다.

순간 순간

여기서 모멘트 값을 설정할 수 있습니다. 모멘트 값은 타임 라인이 시작된 후 게임 단계 수로 계산됩니다. 따라서 위치 0과 60에 모멘트가있는 타임 라인이있는 경우 타임 라인이 시작될 때 모멘트 0이 트리거되고 모멘트 60은 60 단계 후에 발생합니다. 룸 속도도 60으로 설정되어 있으면 타임 라인이 시작된 후 1 초 후에 60 분이 발생합니다. 더블 클릭 할 수 있습니다 모멘트 값 또는 오른쪽 버튼을 사용할 수 있습니다 메뉴에서 " 변경 "을 선택하십시오 (아래 버튼 메뉴의 섹션 참조). 새로운 순간을 추가하려면 편집기 하단의 추가 버튼을 클릭하십시오. 새로운 모멘트를 추가하면 해당 시점에 호출하려는 기능 또는 DnD ™를 추가 할 수있는 코드 편집기 (또는 액션 편집기)가 열립니다.

순간 이름 순간 이름

타임 라인이하는 일을 이해하는 데 도움이되도록 순간을 식별 할 수있는 이름을 부여 할 수 있습니다. 이것은 단순히 문자열이며 게임 코드 또는 다른 곳에서는 사용되지 않습니다. 이는 타임 라인의 기능을 순수하게 구성하고 이해하기위한 것입니다. 순간에 이름 / 설명을 추가하려면 다음과 같이 코드 블록의 첫 번째 줄에 /// @ 설명 형식을 사용해야합니다.

/// @description Hello World

다음과 같이 보일 것입니다.

 

타임 라인 편집기를 사용하면 + 첫 번째 클릭 사이의 모든 순간을 선택하십시오 두 번째 클릭 또는 사용할 수 있습니다 / + 단일 순간을 클릭하여 독립적으로 선택합니다. 마우스 오른쪽 버튼을 사용하여 타임 라인 순간을 편집 할 수 있습니다 선택한 요소 중 하나의 메뉴 :

  • 추가 :이 옵션을 사용하면 타임 라인에 새로운 순간을 추가 할 수 있습니다. 마지막 순간 직후에 추가되므로 타임 라인의 마지막 순간이 120이면 추가 한 다음 순간은 121이됩니다. 두 번 클릭하여 순간 위치를 변경할 수 있습니다. 에디터에서 모멘트 값 또는 마우스 오른쪽 버튼을 클릭하여 " 변경 "을 선택 하십시오 .
  • 편집 : 편집 할 현재 타임 라인 순간을 엽니 다.
  • 순간 변경 : 선택한 타임 라인 순간의 값을 변경합니다. 느린 더블 클릭과 동일 편집기 창의 타임 라인 값
  • 설명 변경 : 선택한 타임 라인 순간의 설명 이름을 변경합니다. 느린 더블 클릭과 동일 편집기 창의 타임 라인 설명에
  • 복제 : 선택한 모든 타임 라인 순간을 복제하여 현재 타임 라인의 끝에 연속적으로 추가합니다.
  • 병합 : 타임 라인에서 둘 이상의 순간을 선택하면 순간을 병합하도록 선택할 수 있습니다. 그러면 모든 코드와 작업이 동시에 선택된 첫 번째 순간의 위치에 단일 타임 라인 순간이 만들어집니다.
  • 제거 : 타임 라인에서 선택한 순간을 제거합니다.
  • 잘라 내기 / 복사 / 붙여 넣기 :이 옵션을 사용하면 하나 이상의 타임 라인 순간을 잘라내거나 복사하거나 붙여 넣을 수 있습니다. 여기에서도 표준 키보드 단축키를 사용할 수 있습니다. / + " X ", / + " C " / + " V ".
  • 모두 선택 :이 옵션을 사용하면 타임 라인에서 모든 순간을 선택할 수 있습니다. 그런 다음 선택한 순간을 제거, 병합, 복제, 잘라 내기 또는 복사 할 수 있습니다.
  • 드래그 앤 드롭으로 변환 / 코드 : 타임 라인 모멘트는 코드를 사용하거나 드래그 앤 드롭 작업을 사용하여 프로젝트를 만들 때 선택한 방법을 사용하여 생성됩니다. 이 옵션을 사용하면 프로젝트 유형에 관계없이 하나에서 다른 것으로 변환 할 수 있습니다.
