회의실 편집기 환경 설정

회의실 편집기 환경 설정 은 회의실 편집기의 모양과 동작을 정의하는 데 사용됩니다. 일반적인 설정은 다음과 같습니다.

  • 체인 페이드 아웃 시작 거리 (px) : 체인 링크가 페이드 아웃되는 부모 윈도우로부터의 거리를 설정합니다. 기본값은 10px입니다.
  • 체인 페이드 아웃 중지 거리 (px) : 체인 링크의 페이드 아웃 종료까지 부모 창에서 거리를 설정합니다. 기본값은 100px입니다.
  • Chain Minimum Alpha : 체인 알파 의 페이드 아웃 폴 오프 지점 이후의 최종 알파 값을 설정합니다. 기본값은 0.3입니다.
  • 기본 도면층 깊이 간격 : 룸 편집기에서 여러 도면층을 작성하면 도면층리스트 내 위치에 따라 깊이가 자동으로 지정됩니다. 이 깊이는 여기에 주어진 양만큼 증가합니다. 기본값은 100이므로 각 레이어는 이전 레이어 + 깊이 100입니다.
  • 기본 회의실 너비 : 새 회의실의 기본값으로 선택 될 초기 너비를 설정합니다. 기본값은 1024px입니다.
  • 기본 방 높이 : 새 방의 기본값으로 선택 될 초기 높이를 설정합니다. 기본값은 768px입니다.
  • 기본 룸 레이어 추가 : 새로운 룸 리소스를 만들면 GameMaker Studio 2는 배경 레이어와 인스턴스 레이어라는 몇 가지 "기본"레이어를 생성합니다. 새 방에 미리 만들어진 레이어가 없도록 해당 동작을 해제 할 수 있습니다. 이것은 기본적으로 켜져 있습니다.
  • 룸 뷰 표시 : 룸 내에서 뷰가 활성화 된 경우 뷰 경계가 룸 내에서 윤곽이 표시되며 편집기 자체의 전용 버튼을 사용하여이 영역의 가시성을 전환 할 수 있습니다. 그러나이 옵션을 선택 취소하여 룸을 열 때 뷰 경계가 보이지 않도록 선택할 수 있습니다 (단추를 사용하여 다시 표시 할 수 있음). 기본값은 켜져 있으며, 방을 열 때 뷰 경계가 표시됩니다.
  • 일반 색상 옵션 : 이들은 룸 편집기 인터페이스 전체에서 사용되는 다양한 색상과 관련이 있으며 견본을 두 번 클릭하면 해당 설정의 색상을 편집 할 수 있습니다.
  • 마스터 볼륨 설정 : Room Editor에서 미리보기 재생을위한 마스터 볼륨 설정으로, 0과 1 사이의 값입니다. 기본값은 1입니다.

대화 상자 확인 대화 상자 확인

대화 상자 확인 환경 설정은 특정 작업을 수행 할 때 회의실 편집기에 표시되는 메시지를 변경하는 데 사용됩니다. 변경할 수있는 메시지는 다음과 같습니다.

  • 레이어 심도 업데이트에 대한 자동 응답 : 레이어 심도 를 변경하면이 작업을 수행하면 다른 모든 레이어 심도도 자동으로 업데이트된다는 경고 메시지가 표시됩니다. 이 옵션을 사용하면 메시지가 표시되도록하거나 "확인"을 클릭 한 것처럼 레이어를 업데이트하거나 "아니요"를 클릭 한 것처럼 아무것도 업데이트하지 않도록 선택할 수 있습니다.
  • 레이어 잠김 경고에 대한 자동 응답 : 회의실에서 레이어를 삭제하려고 시도하고 잠긴 경우 기본적으로 메시지가 표시됩니다. 그러나이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 레이어를 삭제할 수 없습니다.
  • 항목 잠금 경고에 대한 자동 응답 : 항목을 삭제하려고하면 잠겨있는 경우 기본적으로 메시지가 표시됩니다. 그러나이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 항목을 삭제할 수 없습니다.
  • 항목 소유자 레이어 잠김 경고에 대한 자동 응답 : 레이어에서 항목 을 삭제하려고 시도하고 레이어가 잠긴 경우 (항목이 아닌 경우에도) 기본적으로 메시지가 표시되어 하지마 그러나이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 항목을 삭제할 수 없습니다.
  • 시퀀스 / 스프라이트를 부적합한 레이어로 드래그 할 때 새 자산 레이어 생성 : 자산 레이어가 아닌 레이어에 스프라이트 (또는 시퀀스)를 추가하려고하면 기본적으로 메시지가 표시되어 끌지 말고 드래그 할 항목에 대해 새 자산 레이어를 만들지 묻습니다. 그러나이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 스프라이트 (및 시퀀스)에 대한 새 레이어가 자동으로 생성됩니다.
  • 부모 방 항목 삭제에 대한 자동 응답 : 여기에서 부모 방의 일부인 항목을 삭제할 때 수행되는 작업을 선택할 수 있습니다 (모든 자식 방에 있음). 기본값은 경고 메시지를 표시하고 수행 할 작업을 결정하는 것입니다. 그러나 "모두 삭제"로 설정하여 메시지를 표시하지 않고 상위 방과 모든 하위 항목에서 항목 삭제를 진행할 수 있습니다. 메시지가 삭제를 중단하지 않도록하려면 "모두 유지"를 선택하십시오.
  • 부모 방 레이어 삭제에 대한 자동 응답 : 부모 방의 일부인 레이어를 삭제할 때 발생하는 모든 작업을 선택할 수 있습니다 (따라서 모든 자식 방에 있음). 기본값은 경고 메시지를 표시하고 수행 할 작업을 결정하는 것입니다. 그러나 "모두 삭제"로 설정하여 메시지를 억제하고 부모 방과 모든 자식에서 계층 삭제를 진행할 수 있습니다. 메시지가 삭제를 중단하지 않도록하려면 "모두 유지"를 선택하십시오.
  • 유효하지 않은 자산 드래그 오류에 대한 자동 응답 : 인스턴스 유형에 대한 경로 추가와 같이 자산 유형에 대해 지정되지 않은 레이어에 자산을 추가하려고하면 기본적으로 메시지가 표시됩니다 당신은 그것을 할 수 없다는 것을. 그러나이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 자산을 추가 할 수 없습니다. 이 옵션은 인스턴스, 스프라이트 또는 시퀀스에는 적용되지 않습니다.
  • 객체를 부적합한 레이어로 드래그 할 때 새 인스턴스 레이어 생성 : 인스턴스 레이어가 아닌 레이어에 객체 인스턴스를 추가하려고하면 기본적으로 메시지를 표시하여 수행 할 수 없다는 메시지가 표시됩니다 드래그 할 항목에 대한 새 인스턴스 레이어를 만들려는 경우 그러나이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 인스턴스에 대한 새 레이어가 자동으로 생성됩니다.
  • 유효하지 않은 도면층 선택 끌기 오류에 대한 자동 응답 : 여기에서 도면층 편집기에서 여러 도면층을 선택했거나 도면층을 전혀 선택하지 않은 경우 자원을 회의실 편집기로 끌 때 발생하는 상황을 선택할 수 있습니다. 기본값은 경고 메시지를 표시하는 것이지만이 옵션을 "확인"으로 설정하면 메시지를 표시하지 않고 리소스를 추가하지 않고 계속할 수 있습니다.
  • 잠겨 있거나 보이지 않는 레이어 드래그 오류에 대한 자동 응답 : 리소스가 잠겨 있거나 보이지 않는 것으로 표시된 레이어로 리소스를 드래그 할 때 발생하는 작업을 선택할 수 있습니다. 기본값은 경고 메시지를 표시하는 것이지만이 옵션을 "확인"으로 설정하면 메시지를 표시하지 않고 리소스를 추가하지 않고 계속할 수 있습니다.

 

그리드 설정 그리드 설정

그리드 설정 환경 설정은 다음 옵션을 사용하여 그리드 오버레이 작동 방식을 변경하는 데 사용됩니다.

  • 그리드 수평 간격 (px) : 룸 그리드의 거더 셀에 대한 수평 공간을 픽셀 단위로 설정합니다. 기본값은 32px입니다.
  • 그리드 수직 간격 (px) : 룸 그리드의 거더 셀에 대한 수직 공간을 픽셀 단위로 설정합니다. 기본값은 32px입니다.
  • 그리드 표시 :이 옵션은 룸을 처음 열 때 룸 그리드를 켜거나 끕니다. 기본적으로 켜져 있지만 꺼져 있어도 룸 편집기 자체에서 해당 버튼을 사용하여 다시 활성화 할 수 있습니다.
  • 그리드에 스냅 :이 옵션을 사용하면 룸을 처음 열 때 룸에 자산이 룸 그리드에 스냅되는지 여부를 설정할 수 있습니다. 기본적으로 켜져 있지만 꺼져 있어도 룸 편집기 자체에서 해당 버튼을 사용하여 다시 활성화 할 수 있습니다.
  • 격자 선 색상 :이 옵션을 사용하면 룸 격자 선의 색상 (알파 구성 요소 포함)을 설정할 수 있습니다. 기본값은 ( ARGB ) # 30FFFFFF입니다.

스마트 가이드 스마트 가이드

이 섹션에서는 회의실 편집기의 스마트 안내서 기본 설정에 대해 자세히 설명합니다. 사용 가능한 옵션은 다음과 같습니다.

  • 스마트 가이드 활성화 : 룸 편집기에서 스마트 가이드 사용을 활성화 / 비활성화하는 데 사용됩니다. 기본값은 활성화입니다.
  • 눈금자 활성화 :이 옵션은 룸 편집기에서 사이드 눈금자를 활성화 / 비활성화합니다. 기본값은 활성화입니다.
  • 눈금자를 모서리에 스냅 :이 옵션을 선택하면 눈금자에서 룸에 배치 된 자산으로 룸 편집기로 드래그 한 안내선이 스냅됩니다. 기본값은 활성화입니다.
  • Snap Distance (스냅 거리) :이 설정은 룸 에디터에게 에셋의 측면으로 "스냅"되기 전에 가이드가 에셋에서 얼마나 떨어져 있어야하는지 알려줍니다. 기본값은 6px입니다.
