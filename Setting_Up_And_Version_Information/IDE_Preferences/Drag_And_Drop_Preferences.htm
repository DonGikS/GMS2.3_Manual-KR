끌어서 놓기 환경 설정

드래그 앤 드롭 환경 설정은 드래그 앤 드롭 스크립트 편집기의 모양과 느낌을 정의하는 데 사용됩니다. 사용 가능한 다른 옵션은 각각의 기능에 대한 설명과 함께 아래에 나열되어 있습니다.

  • DnD ™ 도구 상자 기본 위치 : 기본적으로 DnD ™ 도구 상자 (모든 다른 작업 라이브러리가있는)는 편집기 오른쪽에 있지만이 옵션을 사용하여 기본값을 왼쪽으로 지정할 수 있습니다.
  • DnD • 노드 개요 가시성 모드 : "개요"노드는 작업 공간의 특정 작업 옆에 나타나는 간단한 설명 텍스트입니다 (예 : 다음 인스턴스 삭제 작업). 항상 사용하도록 설정하거나 작업이 축소 된 경우에만 사용하거나 (기본 설정) 사용하지 않도록 선택할 수 있습니다.
  • DnD ™ GameMaker 언어로의 자동 변환에 대한 자동 응답 : 드래그 앤 드롭 이벤트 또는 스크립트를 GML 코드로 변환하면 기본적으로 수행 할 작업에 대해 경고하고 계속할 것인지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 메시지를 표시하지 않고 변경 내용을 적용해도됩니다.
  • DnD ™ 노드 삭제에 대한 자동 응답 : 끌어서 놓기 이벤트 노드를 삭제하면 기본적으로 수행 할 작업에 대해 경고하고 계속할 것인지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 메시지를 표시하지 않고 삭제 만하면됩니다.
  • DnD ™ 툴박스 액션 썸네일의 최대 크기 : Action Toolbox의 맨 아래에는 드래그 앤 드롭 아이콘의 크기를 변경하기위한 슬라이더가 있습니다. 이 설정은 최대 허용 크기 (기본값 64px)를 정의하는 데 사용됩니다.
  • DnD ™ 도구 상자 작업 축소판의 최소 크기 : Action Toolbox의 맨 아래에는 끌어서 놓기 아이콘의 크기를 변경할 수있는 슬라이더가 있습니다. 이 설정은 허용되는 최소 크기를 정의하는 데 사용됩니다 (기본값 16px).
  • DnD ™ 도구 상자 작업 축소판 의 기본 크기 :이 설정은 도구 상자의 모든 작업 아이콘의 기본 크기를 정의하는 데 사용됩니다 (기본값 32px).
  • 표시 할 최근에 사용한 최대 동작 수 : 동작 도구 상자에는 가장 최근에 사용한 동작이있는 라이브러리가 맨 위에 있습니다. 이 옵션을 사용하여 표시 할 최근 작업의 최대 수를 변경할 수 있습니다 (기본값 5).
  • DnDâ "¢ 작업을 연결하는 체인의 두께 :이 정의 체인 그래픽의 두께있는 드래그 & 드롭 코드 블록에있는 링크 다른 조치 (기본 3px의).
  • 노드 끌기 시작 시간 (ms) : 노드 를 클릭하고 끌면 작업 코드 내에서 노드의 위치가 변경됩니다. 그러나 마우스를 잠시 누르고있다가 이동하면 코드 순서가 아닌 작업 공간 내에서 노드 위치가 이동합니다. 이 설정은 마우스 버튼을 누른 상태에서 노드를 이동하기 전에 기다려야 할 시간 (밀리 초)을 설정합니다 (위치를 바꾸지 않음). 기본값은 350입니다.
  • DnD ™ 노드 개요 패널 숨기기 : 작업 코드 편집기의 개요 패널을 숨기도록 선택할 수 있습니다. 기본적으로이 기능은 비활성화되어 있으며 활성화하면 편집기에서 일반 텍스트 체인 목록이 제거됩니다.

DnD ™ 주석에 대한 하위 메뉴도 있습니다.

주석 환경 설정은 끌어서 놓기 동작 스크립트에 추가 된 주석에 대해 다른 색상 및 서식 옵션을 제어합니다. 이 옵톤은 다음과 같습니다.

  • 색상 : 창에 쓰여지는 텍스트의 기본 색상 (기본 ARGB # FF53E270).
  • 글꼴 이름 : 사용할 글꼴 이름입니다 (사용 가능한 글꼴은 시스템에 설치된 글꼴에 따라 다름).
  • 스타일 : 굵은 체 또는 이탤릭체 와 같이 사용할 글꼴 스타일 (일부 창에서 동일한 스타일을 사용할 수있는 것은 아니며 사용 된 글꼴에 따라 다름).
  • 크기 : 주어진 출력 창에 글꼴을 설정할 크기입니다.
  • 앤티 앨리어싱 : 여기에서 출력 창 글꼴 렌더링에 앤티 앨리어싱 모드를 설정할 수 있습니다. 기본값 사용 을 선택하여 IDE가 결정하도록하거나 해당 옵션 중 하나를 선택하여 강제로 켜 거나 끌 수 있습니다.
