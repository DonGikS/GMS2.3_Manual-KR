자산 브라우저 환경 설정

자산 브라우저 기본 설정 은 자산 브라우저 및 해당 요소의 작동 방식에 대한 특정 속성을 정의하는 데 사용됩니다. 주요 옵션은 다음과 같습니다.

  • 빠른 액세스 표시 :이 옵션을 활성화하면 자산 브라우저 상단에 "빠른 액세스"섹션이 표시됩니다. 이 옵션은 기본적으로 활성화되어 있습니다.
  • 새 프로젝트에서 기본 폴더 생성 :이 옵션을 활성화하면 모든 새 프로젝트가 이미 생성 된 기본 자산 폴더로 시작하고 비활성화되면 모든 프로젝트에 필요한 기본 룸으로 룸 폴더 만 생성됩니다. 이 옵션은 기본적으로 활성화되어 있습니다.
  • 경고없이 동시에 열 수있는 최대 자산 수 : 한 번에 여러 자산을 선택 하여 열 수 있습니다. / + 그들을 선택하고 마우스 오른쪽 버튼을 클릭 모두 열기를 선택합니다. 이 옵션에 지정된 수보다 많은 수를 선택한 경우 한 번에 너무 많은 자산을 열면 하한 시스템의 성능에 문제가 발생할 수 있으므로 경고가 표시됩니다. 기본값은 10입니다.
  • 최근에 편집 한 최대 항목 수 :이 설정은 자산 브라우저의 "최근"섹션에 나열된 자산 수를 제어합니다. 기본값은 10입니다.
  • 색상 팔레트 크기 : 자산 그룹에 대한 자산 브라우저 팔레트에서 사용 가능한 색상 수를 설정할 수 있습니다. 기본값은 16이며 최소값은 16이고 최대 값은 100입니다.
  • 빠른 액세스에 자산 색상 적용 :이 옵션을 선택하면 주어진 자산 유형에 설정된 색상을 사용하여 브라우저의 빠른 액세스 섹션에서 자산에 색상을 지정합니다. 이 옵션을 선택 취소하면 자산 색상이 사용되지 않습니다. 기본적으로 활성화되어 있습니다.
  • 그룹 색상은 하위 색상을 재정의합니다 .이 옵션을 선택하면 최상위 레벨 그룹 색상을 사용하여 그룹 내의 모든 자산에 색상을 지정하고 개별 자산에 설정된 색상은 무시합니다. 기본적으로 비활성화되어 있습니다.
  • 자산 대화 스타일 생성 : 자산 브라우저에서 "자산 추가"버튼을 클릭하면 프로젝트에 추가 할 수있는 다양한 자산을 보여주는 창이 나타납니다. 이 창은 목록보기 또는 격자보기로 표시 될 수 있으며이 설정은 기본적으로 표시되는 것을 제어합니다. 기본값은 그리드입니다.
  • 기본 정렬 순서 : 브라우저에서 자산의 기본 정렬 순서를 알파벳 순서로 (A에서 Z 또는 Z에서 A로) 설정할 수 있습니다. 기본값은 A에서 Z입니다.
  • 기본 정렬 폴더 우선 :이 옵션을 활성화하면 자산 브라우저가 먼저 폴더를 정렬 한 다음 "느슨한"자산을 두 번째로 정렬합니다.이 옵션을 사용하지 않으면 모든 자산과 폴더가 함께 정렬됩니다. 기본적으로 활성화되어 있습니다.
  • 스프라이트 더블 클릭 하면 이미지 편집기가 열립니다 : 기본적으로 더블 클릭 모든 자산에서 편집 할 자산이 열리지 만 스프라이트를 다룰 때 새 자산을 만들 때 초기 설정 후 스프라이트 속성 대신 이미지를 편집하기 위해 자산을 엽니 다. 이 옵션을 선택하면 스프라이트 자산 자체 대신 스프라이트 자산의 이미지 편집기가 열립니다. 마우스 오른쪽 버튼을 클릭하여 스프라이트 편집기를 계속 열 수 있습니다 편집을 선택합니다. 이 옵션은 기본적으로 해제되어 있습니다.
  • 객체 더블 클릭은 모든 객체 이벤트를 엽니 다 . 기본적으로 더블 클릭 모든 자산에서 편집을 위해 해당 자산을 열지 만, 종종 오브젝트를 처리 할 때 새 자산을 만들 때 초기 설정 후 자산 속성이 아닌 이벤트 만 편집하기 위해 자산을 엽니 다. 이 옵션을 선택하면 새 작업 공간에서 다른 오브젝트 이벤트가 열리고 각 이벤트에는 새 작업 공간 내에서 고유 한 탭이 ​​제공됩니다. 마우스 오른쪽 버튼을 클릭하여 개체 편집기를 열 수 있습니다 편집을 선택합니다. 이 옵션은 기본적으로 해제되어 있습니다.
  • 기본 확대 / 축소 (%) : 새 프로젝트를 시작할 때 자산 브라우저의 기본 확대 / 축소 비율을 설정합니다. 기본값은 100 %입니다.
  • 최소 줌 (%) : 자산 브라우저에 허용되는 최소 줌 값을 설정합니다. 기본값은 50 %입니다.
  • 최대 줌 (%) : 자산 브라우저에 허용되는 최대 줌 값을 설정합니다. 기본값은 250 %입니다.
  • 트리 라인 표시 : 자산 브라우저에서 자산 측면에 표시되는 트리 라인을 활성화 / 비활성화 할 수 있습니다. 트리 선을 표시하기 위해 기본적으로 선택되어 있습니다.

 

대화 상자 확인 대화 상자 확인

여기에서 자산 브라우저에서 자산으로 작업 할 때 확인 대화 상자의 동작을 선택할 수 있습니다. 사용 가능한 옵션은 다음과 같습니다.

  • 자산 삭제에 대한 자동 응답 : 자산 브라우저에서 자산을 삭제하면 일반적으로 진행할 것인지 묻는 경고가 표시됩니다. 이 옵션을 "확인"으로 설정하면 "확인"버튼을 클릭 한 것처럼 메시지가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 여러 자산 열기에 대한 자동 응답 : 일반 자산 브라우저 환경 설정에서 위에서 언급 한 것처럼 자산 브라우저에서 여러 자산을 한 번에 열면 숫자가 지정된 수를 초과하면 경고가 표시 될 수 있습니다. 이 옵션을 "확인"으로 설정하면 "확인"버튼을 클릭 한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • DnD ™ GameMaker 언어로의 변환에 대한 자동 응답 : DnD ™ 노드를 GML로 변환하는 옵션을 선택하면 취소 할 수 없으므로 변경을 확인하라는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 "확인"버튼을 클릭 한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 중복 그룹 이름 오류에 대한 자동 응답 : 자산을 구성하기 위해 그룹 폴더를 생성하고 브라우저의 다른 폴더와 동일하게 이름을 지정할 때 계속하고 이름이 같은 두 개의 폴더가 있는지 묻는 메시지가 표시됩니다 . 이 옵션을 "확인"으로 설정하면 "확인"버튼을 클릭 한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
  • 노드 끌기를 통해 회의실 상위 설정에 대한 자동 응답 : 회의실 관리자에서 다른 회의실 위로 회의실을 끌어다 놓으면 회의실을 회의실의 "자식"으로 만들 것인지 묻는 메시지가 표시됩니다. 떨어졌다. 이 옵션을 "확인"으로 설정하면 "확인"버튼을 클릭 한 것처럼 경고가 표시되지 않습니다. 기본값은 "메시지 표시"입니다.
