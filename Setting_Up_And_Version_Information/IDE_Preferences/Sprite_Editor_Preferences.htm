스프라이트 에디터 환경 설정

Sprite Editor Preferences 는 Sprite Editor의 모양과 기능을 정의하는 데 사용됩니다. 다음과 같은 옵션이 있습니다.

  • 기본 스프라이트 원산지 :이 옵션을 사용하면 생성 한 새 스프라이트 리소스의 기본 원점 위치를 설정할 수 있습니다. 기본값은 왼쪽 위입니다.
  • 기본 스프라이트 충돌 마스크 종류 :이 옵션을 사용하면 새 스프라이트에 사용할 기본 충돌 마스크 종류를 설정할 수 있습니다. 직사각형 , 회전 직사각형 , 타원 , 다이아몬드 또는 프레임 당 정밀 / 정확도를 선택할 수 있습니다. 기본값은 rectangle 입니다.
  • 기본 스프라이트 애니메이션 모드 :이 옵션을 사용하면 스프라이트 편집기에서 스프라이트의 기본 애니메이션 모드를 설정할 수 있습니다. " 초당 프레임 "또는 " 게임 프레임 당 프레임 "을 선택할 수 있으며 기본 설정은 " 초당 프레임 "입니다.
  • 기본 스프라이트 애니메이션 속도 :이 옵션을 사용하면 새 스프라이트 자산을 만들 때의 기본 애니메이션 속도 값을 설정할 수 있습니다. 이 값은 애니메이션 모드에 연결되며, 예를 들어 초당 3 개의 프레임은 게임 프레임 당 30 개의 프레임과 동일하지 않으므로 선호하는 모드를 참조하여이 값을 올바르게 설정하십시오. 기본값은 30입니다.
  • 기본 스프라이트 너비 :이 옵션을 사용하면 생성 된 모든 새 스프라이트의 기본 너비를 설정할 수 있습니다. 기본값은 64입니다.
  • 기본 스프라이트 높이 :이 옵션을 사용하면 모든 새 스프라이트의 기본 높이를 설정할 수 있습니다. 기본값은 64입니다.
  • 스프라이트 원점에 크기 조정 및 크기 조정 적용 :이 옵션을 선택하면 스프라이트의 크기를 조정하거나 크기를 조정할 때 스프라이트의 새 값에 맞게 새 원점이 이동 / 크기 조정될 때 스프라이트의 원점이 변경됩니다. 이는 스프라이트 편집기에서 잠기지 않은 사용자 지정 스프라이트 원점에만 적용됩니다. 원점이 잠겨 있거나 스프라이트가 기본 위치 중 하나 (예 : "중앙"또는 "왼쪽 상단"등)를 사용하는 경우이 옵션은 회색으로 표시되며 변경할 수 없습니다. 기본값은 활성화입니다.

Sprite Editor에는 다음과 같은 하위 범주도 있습니다.

대화 상자 확인 대화 상자 확인

여기에서 사용 가능한 옵션은 작업을 확인해야 할 때 표시되는 메시지를 제어합니다. 이 설정을 변경하여 항상 메시지를 표시하거나 특정 조치 (예 또는 아니오)로 메시지를 닫을 수 있습니다.

  • 파일 가져 오기에 대한 자동 응답 : 새 이미지를 가져오고 이미 존재하는 경우 기본적으로 기존 이미지를 가져오고 덮어 쓰거나 작업을 취소하라는 메시지가 표시됩니다. 이 옵션을 사용하면이 메시지를 억제하고 항상 "아니오"를 선택하여 가져 오기를 취소하거나 "예"를 선택하여 현재 스프라이트를 덮어 씁니다. 기본값은 "메시지 표시"입니다.
  • 이미지로드 실패에 대한 자동 응답 : 새 이미지를 가져오고 가져 오기에 실패하면 기본적으로 오류가 발생했음을 알리는 경고 메시지가 표시됩니다. 이 옵션을 사용하면 "확인"을 선택하여이 메시지를 표시하지 않을 수 있습니다. 기본값은 "메시지 표시"입니다.
