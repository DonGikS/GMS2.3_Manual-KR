이미지 편집기 환경 설정

이미지 편집기 환경 설정 은 이미지 편집기 창의 모양과 느낌을 정의하는 데 사용됩니다. 다음은 일반적인 옵션입니다.

  • 스프라이트 컨트롤 오버레이 기본적으로 확장 :이 옵션을 선택하면 편집 할 스프라이트를 열 때마다 스프라이트 컨트롤 및 프레임 정보가 이미지 편집기에 표시됩니다 (축소 버튼을 사용하여 다시 닫을 수 있음) 스프라이트를 다시 편집하면 나중에 다시 열립니다). 이 옵션을 선택하지 않으면 스프라이트를 열 때 스프라이트 컨트롤과 프레임 정보가 숨겨집니다. 디폴트는 이것을 체크하지 않는 것입니다.
  • 기본 루프 모드 : 이미지 편집기의 루프 버튼은 두 가지 모드로 작동하도록 설정할 수 있습니다. 한 번 재생, 마지막 프레임에 도달 할 때 애니메이션이 중지되는 위치, 시작부터 반복, 애니메이션이 다시 시작되는 위치 애니메이션이 끝날 때의 첫 번째 프레임 또는 탁구, 애니메이션이 이미지를 통해 끝까지 올라간 다음 다시 처음으로 다시 내려갑니다. 여기서 기본값은 "loop"입니다.
  • 얇은 선택 윤곽선 그리기 :이 옵션을 선택하면 이미지의 배율에 관계없이 선택 윤곽선의 너비가 절대적으로 1px가되고, 선택하지 않으면 선택 윤곽선의 너비가 1px이며 영상. 기본값은이 옵션을 체크하는 것입니다.
  • 선택 윤곽선 기본 색상 : 섹션 윤곽선은 두 색상 사이에서 번갈아 나타나는 파선입니다. 이 옵션을 사용하면 첫 번째 색상 (알파 포함)을 설정할 수 있으며 기본값은 ARGB # FFFFFD0입니다.
  • 선택 윤곽선 보조 색상 : 섹션 윤곽선은 두 색상 사이에서 번갈아 나타나는 파선입니다. 이 옵션을 사용하면 두 번째 색상 (알파 포함)을 사용할 수 있으며 기본값은 ARGB # 0000D0입니다.
  • 편집기 배경색 : 스프라이트 프레임을 편집하는 동안 이미지가 확대 또는 축소되거나 전체 그림이 창에 맞지 ​​않을 수 있습니다.이 경우 여기에서 설정할 수있는 배경색이 표시됩니다. 기본값은 ARGB # 282828FF입니다.

이미지 편집기 환경 설정에는 다음과 같은 하위 범주도 있습니다.

배경 배경

여기에서 스프라이트 이미지를 편집하는 배경에 대한 옵션을 설정할 수 있습니다. 기본적으로 이것은 회색 사각형 격자이지만 다음 설정을 사용하여 변경할 수 있습니다.

  • 캔버스 스타일 : 여기에서 " 블록 패턴 "또는 " 단색 "중 하나 인 배경의 "스타일"을 설정합니다. 단일 색상은 단순히 배경을 선택한 색상으로 설정하고 (아래 설명 된 색상 1 설정 사용) 블록 패턴은 기본 설정입니다 (아래에 표시된 나머지 대화 옵션을 사용하여 편집 할 수도 있음). 기본값은 " 블록 패턴 "입니다.
  • 캔버스 블록 크기 : 배경에 대해 블록 패턴 을 선택한 경우 여기에서 개별 블록의 크기를 설정할 수 있습니다. 기본값은 16px이며 최소값은 2px이며 최대 256px입니다.
  • 캔버스 기본 색상 : 배경에 대해 단일 색상을 선택한 경우이 색상이 사용됩니다. 색상 견본을 클릭하여 색상 편집기를 열고 변경할 수 있습니다. 블록 패턴 옵션을 사용하면 대체 블록의 첫 번째 색상이 설정됩니다. 기본 색상은 ( ARGB ) # FFC3C3C3입니다.
  • 캔버스 보조 색상 (블록 패턴 스타일) : 배경에 대해 블록 패턴을 선택한 경우 색상 1 (위)과 번갈아 사용되는 색상이이 색상입니다. 색상 견본을 클릭하여 색상 편집기를 열고 변경할 수 있습니다. 기본 색상은 (ARGB) # FF808080입니다.

 

대화 상자 확인 대화 상자 확인

여기에서 현재 스프라이트에 대해 다른 작업을 수행 할 때 확인 대화 상자의 동작을 선택할 수 있습니다. 그들은:

  • 프레임 삭제에 대한 자동 응답 : 기본적으로 프레임을 삭제하면이를 수행 할 것인지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면이 메시지를 제거하고 추가 메시지가 표시되지 않고 프레임 삭제를 허용 할 수 있습니다. 기본값은 "메시지 표시"입니다.
  • 스트립 가져 오기 중 이미지 교체에 대한 자동 응답 : 이미지 드롭 다운 메뉴에서 스트립 가져 오기를 선택하면 스프라이트 리소스에 이미있는 이미지를 덮어 쓰거나 계속할지 여부와 함께 경고한다는 메시지가 표시됩니다. 이 메시지를 표시 할 것인지 또는 IDE에서 "예"또는 "아니오"옵션을 선택하도록할지 여기에서 설정할 수 있습니다. 기본값은 "메시지 표시"입니다.
  • 도구 변경시 변경 사항 커밋에 대한 자동 응답 : 프레임을 편집 한 후 그리기 도구를 변경하면 기본적으로 변경 사항을 커밋할지 묻는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면이 메시지를 제거하고 도구를 변경할 때 추가 메시지가 표시되지 않고 프레임 에디션을 커밋 할 수 있습니다. "아니오"를 선택하면 도구를 변경할 때 프레임 에디션이 자동으로 취소됩니다. 기본값은 "메시지 표시"입니다.
  • 잠긴 레이어 경고에 대한 자동 응답 : 잠긴 레이어 를 편집하려고하면 기본적으로이를 수행 할 수 없다는 메시지가 표시됩니다. 이 옵션을 "확인"으로 설정하면 메시지가 표시되지 않고 버전이 자동으로 실패합니다.
  • 그룹 선택 경고에 대한 자동 응답 : 도면층 편집기에서 여러 도면층을 선택한 다음 그리기 또는 편집을 시도하면 기본적으로 불가능하다는 메시지가 표시됩니다. 여기에서 "확인"을 선택하여이 메시지를 제거하고 단일 레이어 만 선택할 때까지 이미지를 자동으로 편집하지 못합니다. 기본값은 "메시지 표시"입니다.

그리드 설정 그리드 설정

격자 선 설정 환경 설정은 다음 옵션을 사용하여 이미지 격자 선 표시 방법을 설정하는 데 사용됩니다.

  • 격자 가로 간격 (px) :이 값은 이미지 격자의 가로 선 간격을 설정합니다. 기본값은 1px입니다.
  • 격자 세로 간격 (px) :이 값은 이미지 격자의 세로 선 간격을 설정합니다. 기본값은 1px입니다.
  • 격자 표시 :이 옵션을 선택하면 이미지 격자가 켜진 상태로 이미지 편집기가 열립니다. 기본값은 꺼져 있습니다.
  • 그리드에 스냅 :이 옵션을 선택하면 이미지 편집기가 특정 도구를 이미지 그리드에 자동으로 스냅합니다. 기본값은 꺼져 있습니다.
  • 그리드 라인 컬러 : 이미지 그리드의 색상 (알파 값 포함)을 설정할 수 있습니다. 기본값은 ( ARGB ) # FF000000입니다.

 

도구 도구

여기에서 일부 그리기 도구가 작동하고 표시되는 방식에 대해 다른 옵션을 설정할 수 있습니다.

아크 도구를 사용하는 경우 그런 다음 노드 속성을 그리는 데 사용되는 다른 색상을 설정할 수 있습니다.

  • 앵커 포인트 색상 : 스플라인 / 호에서 다른 앵커 포인트의 색상을 설정합니다. 기본값은 ( ARGB ) # FFFF0000입니다.
  • 제어점 색상 : 기준점 에서 각 제어점 "핸들"의 색상을 설정합니다. 기본값은 (ARGB) # FFFF0000입니다.
  • 탄젠트 선 색상 : 제어점을 앵커에 연결하는 선의 색상을 설정합니다. 기본값은 (ARGB) # FF00FFFF입니다.
  • 선택된 점 색 : 스플라인 / 호에서 선택한 기준점에 사용할 색을 설정합니다. 기본값은 (ARGB) # FF00FF00입니다.
  • 포인트 크기 : 사용할 앵커 포인트와 제어점의 크기를 설정합니다. 기본값은 10px입니다.
