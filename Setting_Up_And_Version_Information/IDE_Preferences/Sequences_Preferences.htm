시퀀스 환경 설정

시퀀스 환경 설정 은 시퀀스 편집기 및 해당 요소가 작동하는 방식에 대한 특정 속성을 정의하는 데 사용됩니다. 주요 옵션은 다음과 같습니다.

  • 새로 만든 시퀀스에 캔버스 프레임 표시 : 활성화하면 캔버스 프레임 스마트 가이드가 모든 새 시퀀스에 대해 시퀀스 편집기의 캔버스에 표시됩니다. 이 옵션은 편집기 자체의 캔버스 속성에서도 필요에 따라 활성화 / 비활성화 할 수 있습니다. 기본값이 활성화되어 있습니다.
  • 새로 만든 시퀀스의 캔버스 프레임 너비 : 여기에서 모든 새 시퀀스의 기본 프레임 너비를 설정할 수 있습니다. 기본값은 1366px입니다.
  • CA는 새로 생성 된 시퀀스에 프레임 높이를 nvas : 여기에 모든 새로운 시퀀스의 기본 프레임 높이를 설정할 수 있습니다. 기본값은 768px입니다.
  • 변경 사항 자동 기록 :이 옵션을 사용하면 추가 한 모든 새 시퀀스에 대해 시퀀스 편집기에서 "변경 내용을 자동으로 기록"단추를 사용할 수 있습니다. 시퀀스 편집기 자체에서도 활성화 / 비활성화 할 수 있습니다. 기본값은 활성화입니다.
  • 캔버스를 클릭 할 때 일시 중지 :이 옵션을 선택하면 시퀀스 편집기 캔버스의 아무 곳이나 클릭하면 시퀀스 재생이 일시 중지됩니다. 기본적으로 활성화되어 있습니다.
  • 밀리 초 단위의 겹침 선택 지연 : 시퀀스 편집기 캔버스에서 항목 도구 상자가 겹칠 때까지 항목 도구 상자가 표시되기까지 얼마나 기다려야하는지 제어합니다. 기본값은 1000ms입니다.
  • 캔버스 체인 두께 : 시퀀스 편집기 캔버스 내 모든 체인 요소의 두께 (픽셀)입니다. 기본값은 4px입니다.
  • 캔버스 체인 색상 : 시퀀스 편집기 캔버스에서 요소 사이의 모든 체인에 사용되는 색상입니다. 기본값은 $ A800FFFF ( ARGB )입니다.
  • 변환 경로 점 삽입 여백 : 시퀀스 편집기 캔버스에서 자산을 추가 한 다음 위치 매개 변수 트랙을 만들면 자산이 이동할 때 수행 할 경로를 보여주기 위해 점선이 그려집니다. 마우스를이 선에 가깝게 배치하면 새 매개 변수 추적 점을 추가 할 수 있으며이 옵션은 마우스가 점 마커가 표시되기 전에 변환 선으로부터의 거리를 설정합니다. 기본값은 6px입니다.
  • 기본 파라미터 트랙 자동 생성 :이 옵션을 사용하면 시퀀스의 트랙 패널에서 생성 된 새 트랙에 위치, 스케일, 회전 및 원점에 대한 파라미터 트랙이 자동으로 추가됩니다. 기본값은 비활성화입니다.

 

스마트 가이드 스마트 가이드

여기서 Sequences Editor에서 작업 할 때 스마트 가이드의 동작을 선택할 수 있습니다. 사용 가능한 옵션은 다음과 같습니다.

  • 스마트 안내선 사용 : 시퀀스 편집기 캔버스에서 스마트 안내선을 사용 또는 사용하지 않도록 설정할 수 있습니다. 기본적으로 활성화되어 있습니다.
  • 눈금자 사용 : 시퀀스 편집기 캔버스에서 스마트 안내선을 사용 또는 사용하지 않도록 설정할 수 있습니다. 기본적으로 활성화되어 있습니다.
  • 눈금자를 모서리에 맞추기 :이 옵션을 사용하면 눈금자가 시퀀스 캔버스에서 자산의 모서리에 "맞춤"(또는 그 반대) 할 수 있습니다. 기본값은 활성화입니다.
  • 스냅 거리 : 항목이 눈금자 (또는 항목에 대한 눈금자)와 위치가 "스냅"될 때까지의 거리 (픽셀)입니다. 기본값은 6px입니다.
