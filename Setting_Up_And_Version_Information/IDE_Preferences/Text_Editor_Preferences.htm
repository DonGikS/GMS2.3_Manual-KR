텍스트 편집기 편집기 환경 설정

텍스트 편집기 환경 설정 은 다른 코드 / 스크립트 편집기의 모양과 느낌을 정의하는 데 사용됩니다. 메인 페이지에는 다음과 같은 옵션이 있습니다.

  • 구문 검사 지연 (ms) : 구문 검사기가 실행되기까지의 지연 시간 (밀리 초)입니다. 이 기본값은 2000ms입니다.
  • 코드 완료 지연 (ms) : 지연 시간 (밀리 초)
  • 전체 화면 편집기에서 스크립트 열기 :이 옵션을 선택하면 새 작업 공간에서 스크립트 자원이 자동으로 열립니다. 이 확인란을 선택하지 않으면 현재 작업 공간 내의 창에서 열립니다. 이 옵션은 기본적으로 선택되어 있지 않습니다.
  • 전체 화면 편집기에서 셰이더 열기 :이 옵션을 선택하면 새 작업 공간에서 셰이더 리소스가 자동으로 열립니다. 이 확인란을 선택하지 않으면 현재 작업 공간 내의 창에서 열립니다. 이 옵션은 기본적으로 선택되어 있지 않습니다.
  • 전체 화면 편집기에서 오브젝트 이벤트 열기 :이 옵션을 선택하면 오브젝트 작업 코드 편집기가 새 작업 공간에서 자동으로 열립니다. 이 옵션을 선택하지 않으면 개체 이벤트 편집기에 연결된 창에서 열립니다. 이 옵션은 기본적으로 선택되어 있지 않습니다.
  • 동일한 창에서 이벤트 스크립트를 탭으로 열기 : 이벤트에 코드 또는 드래그 앤 드롭을 추가하면 코드 창이 이벤트 창에 연결되며이 옵션을 선택하면 편집을 위해 열린 추가 이벤트가 동일하게 추가됩니다. 상단에 탭으로 창. 이 옵션을 선택 취소하면 각 이벤트가 고유 한 체인 창으로 열립니다. 이 옵션은 기본적으로 선택되어 있습니다.
  • 링크 된 스크립트를 같은 창에서 탭으로 열기 : 링크 된 스크립트가 여러 개 있고 스크립트를 열면 기본적으로 탭과 동일한 창에서 그룹화됩니다. 이 옵션을 선택 취소하면 각 창 / 작업 공간에서 각각 열 수 있습니다. 이 옵션은 기본적으로 선택되어 있습니다.
  • 포커스가있는 텍스트 편집기로 초점 이동 : 개체 이벤트에서 텍스트 편집기를 열 때이 옵션을 선택하면 작업 영역이 자동으로 이동하여 편집기가 경계 내로 이동합니다. 이 옵션은 기본적으로 해제되어 있습니다.
  • 기본 코드 창 너비 : 코드 창이 열릴 때의 기본 너비 (픽셀)를 설정합니다. 기본값은 900입니다.
  • 기본 코드 창 높이 : 코드 창이 열릴 때의 기본 높이 (픽셀)를 설정합니다. 기본값은 600입니다.
  • 최소 코드 창 너비 : 코드 창 에서 허용하는 최소 너비 (픽셀)를 설정합니다. 기본값은 300입니다.
  • 최소 코드 창 높이 : 코드 창 에서 허용하는 최소 높이 (픽셀)를 설정합니다. 기본값은 200입니다.

위의 옵션 외에도 코드 편집기 환경 설정을위한 하위 범주가 있습니다.

코드 편집기 코드 편집기

색상 환경 설정 은 코드와 스크립트 편집기가 다른 텍스트를 표시하는 방법을 변경하는 데 사용됩니다. 각 항목에는 위의 예제 이미지에 표시된대로 각각의 개별 텍스트가 어떻게 표시되어야 하는지를 정확하게 정의 할 수있는 여러 하위 항목이 있습니다. 일반적으로 각기 다른 옵션에 대해 색상, 글꼴, 크기 및 스타일을 설정할 수 있습니다. 또한 코드 브래킷의 라인 선택 등과 같은 것들에 대해 더 일반적인 스타일을 설정할 수도 있습니다.

색상 기본 설정에는 다양한 스타일 옵션이 있습니다. 이러한 환경 설정은 다음 옵션을 사용하여 코드 / 스크립트 편집기의 동작을 설정하는 데 사용됩니다.

  • 탭 유지 :이 옵션을 선택하면 편집기에서 탭 키를 탭 문자 ' \\ t '로 유지합니다. 선택하지 않으면 탭이 공백으로 변환됩니다. 기본값은 꺼져 있습니다.
  • 탭 크기 : "Tab"을 눌러 코드를 들여 쓰는 문자 공간의 수입니다. 기본값은 4입니다.
  • 공백 표시 : 공백을 표시하려면 " . "를 표시하십시오 (아래 이미지 참조). 기본값은 꺼져 있습니다.
  • 줄 번호 표시 : 줄 번호를 왼쪽 아래에 표시할지 여부입니다. 기본값은 켜져 있습니다.
  • 탭 라인 표시 : 탭 공간을 표시하려면 " .... "를 표시하십시오. 기본값은 꺼져 있습니다.
  • 색상 코딩 사용 : 편집기의 색상 코딩을 사용하거나 사용하지 않습니다. 꺼져 있으면 코드의 스타일이 지정되지 않고 켜져 있으면 색상 섹션에서 위에 제공된 형식을 사용하여 스타일이 지정됩니다. 기본값은 켜져 있습니다.
  • 문서 줄 제한 : 주어진 편집기 문서의 최대 줄 수를 설정합니다. 기본값은 0 (0 이하는 기본적으로 무한대)이며 다른 값은 허용되는 행을 제한합니다.
  • 새 줄에 들여 쓰기 일치 : 코드의 자동 들여 쓰기를 활성화 또는 비활성화합니다. 기본값은 켜져 있으며 새 줄이 추가 될 때 줄에 설정된 탭을 유지합니다.
  • 중괄호 후 자동 들여 쓰기 : 중괄호 {}를 추가 할 때 코드에 대한 자동 들여 쓰기를 활성화 또는 비활성화합니다. 새 가새 { 및 새 행이 추가 될 때 다음 코드 행을 탭하는 기본값이 설정되어 있습니다.
  • 코드 완성 활성화 : 코드 완성 창을 활성화 또는 비활성화합니다. 코드를 작성하고 활성화하면 현재 입력 텍스트를 기반으로 사용할 수있는 가능한 기능을 보여주는 팝업 창이 나타납니다. 기본값은 켜져 있습니다.
  • 자동 대괄호 추가 : 자동 완성을 사용할 때 대괄호 자동 추가 ()를 활성화 또는 비활성화합니다. 코드를 작성할 때이 기능이 활성화되면 IDE는 함수에 필요한 두 개의 괄호 () 를 자동으로 추가하고, 함수가 인수를 사용하면 커서를 그 안에 넣고, 그렇지 않으면 커서를 그 뒤에 놓습니다. 이 기능은 자동 완성에서 기능을 선택한 경우에만 작동합니다. 기본값은 켜져 있습니다.
  • Undo / Redo Stack Limit : Undo / Redo 스택 제한 을 설정할 수 있습니다. 0 미만의 것은 본질적으로 무한하며 (사용 가능한 메모리에 의해 제한됨) 양수 값은 스택을 해당 수로 제한하여 한계에 도달하면 스택의 꼬리에서 모든 작업을 제거합니다. 기본값은 -1입니다.
  • 하이라이트 선택 일치 : 다른 곳에서 복제 된 코드 섹션을 가볍게 강조 표시합니다. 예를 들어 함수를 선택하면 현재 코드 창에서이 함수의 모든 인스턴스도 강조 표시됩니다. 기본값은 켜져 있습니다.
  • 더 이상 사용되지 않는 기능에 대한 색상 코딩 / 코드 완성 : 더 이상 사용되지 않는 코드 섹션이 강조 표시됩니다. 기본값은 켜져 있습니다.
